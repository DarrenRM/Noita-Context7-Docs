# Monolith Vegetation Sprite

This documentation describes the sprite data for the Monolith vegetation in Noita.

## Sprite

The `<Sprite>` element defines the visual representation of the Monolith.

### Key Attributes

*   **filename**: Specifies the path to the sprite image file.
    *   `data/vegetation/monolith.png`
*   **offset\_x**: The horizontal offset of the sprite's origin.
    *   `10`
*   **offset\_y**: The vertical offset of the sprite's origin.
    *   `47`
