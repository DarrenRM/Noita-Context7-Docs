# Polish Language Configuration

This file configures the Polish language settings for Noita, primarily focusing on font definitions and UI offsets.

## Language Attributes

| Attribute | Description |
|---|---|
| `name` | The display name of the language (e.g., "Polska"). |
| `font_default` | Path to the default font used for general text. |
| `font_inventory_title` | Path to the font used for inventory titles. |
| `font_important_message_title` | Path to the font used for important message titles. |
| `font_world_space_message` | Path to the font used for messages displayed in the game world. |
| `fonts_utf8` | Flag indicating if UTF-8 font support is enabled (1 for enabled). |
| `fonts_pixel_font` | Flag indicating if pixel font rendering is enabled (1 for enabled). |

## UI Offsets

These attributes control the positioning of certain UI elements.

| Attribute | Description |
|---|---|
| `ui_action_info_offset2` | Offset for action information display. |
| `ui_wand_info_offset1` | Primary offset for wand information. |
| `ui_wand_info_offset2` | Secondary offset for wand information. |
