# Italian Language Configuration

This file configures the Italian language settings for Noita, primarily focusing on font definitions and UI offsets.

## Language Attributes

*   **name**: Specifies the name of the language.
    *   `Italiano`
*   **font\_default**: Path to the default font used in-game.
    *   `data/fonts/font_pixel.xml`
*   **font\_inventory\_title**: Path to the font used for inventory titles.
    *   `data/fonts/font_pixel_big.xml`
*   **font\_important\_message\_title**: Path to the font used for important message titles.
    *   `data/fonts/font_pixel_huge.xml`
*   **font\_world\_space\_message**: Path to the font used for messages displayed in the game world.
    *   `data/fonts/font_pixel.xml`
*   **fonts\_utf8**: Enables UTF-8 font support.
    *   `1` (Enabled)
*   **fonts\_pixel\_font**: Enables pixel font support.
    *   `1` (Enabled)

## UI Offset Attributes

These attributes control the positioning of various UI elements.

*   **ui\_action\_info\_offset2**: Offset for action information.
    *   `10`
*   **ui\_wand\_info\_offset1**: First offset for wand information.
    *   `0`
*   **ui\_wand\_info\_offset2**: Second offset for wand information.
    *   `5`
*   **ui\_configurecontrols\_offset2**: Offset for the configure controls UI.
    *   `190`
