# Pixel Runes Font Data

This document describes the configuration for the "Pixel Runes" font used in Noita. It defines the texture containing the glyphs and the spacing parameters for rendering text.

## Core Font Properties

These are the main attributes that control how the font is rendered.

*   **`<Texture>`**: Specifies the image file that contains the font's glyphs.
    *   `data/fonts/font_pixel_runes.png`
*   **`<LineHeight>`**: The vertical space between lines of text.
    *   `7`
*   **`<CharSpace>`**: The horizontal space between individual characters.
    *   `0`
*   **`<WordSpace>`**: The horizontal space between words.
    *   `6`

## Character Definitions (`<QuadChar>`)

Each `<QuadChar>` element defines a single character's properties within the font texture. The most important attributes are:

*   **`id`**: The Unicode (or ASCII) value of the character.
*   **`width`**: The horizontal width of the character's bounding box.
*   **`rect_x`**: The x-coordinate of the top-left corner of the character's bounding box within the texture.
*   **`rect_y`**: The y-coordinate of the top-left corner of the character's bounding box within the texture.
*   **`rect_w`**: The width of the character's bounding box within the texture.
*   **`rect_h`**: The height of the character's bounding box within the texture.
*   **`offset_x`**: Horizontal offset for the character.
*   **`offset_y`**: Vertical offset for the character.
