# Portuguese (Brazil) Language Settings

This file defines the language settings for the Portuguese (Brazil) localization in Noita. It primarily focuses on specifying the fonts used for different text elements within the game.

## Key Attributes

*   **`name`**: The display name of the language.
    *   `Portugues do Brasil`
*   **`font_default`**: The default font used for most in-game text.
    *   `data/fonts/font_pixel.xml`
*   **`font_inventory_title`**: The font used for titles within the inventory screen.
    *   `data/fonts/font_pixel_big.xml`
*   **`font_important_message_title`**: The font used for titles of important messages.
    *   `data/fonts/font_pixel_huge.xml`
*   **`font_world_space_message`**: The font used for messages displayed in the game world.
    *   `data/fonts/font_pixel.xml`
*   **`fonts_utf8`**: Indicates if UTF-8 font support is enabled.
    *   `1` (Enabled)
*   **`fonts_pixel_font`**: Indicates if pixel font rendering is enabled.
    *   `1` (Enabled)

## UI Offsets

These attributes control the positioning of certain UI elements, ensuring proper alignment for the Portuguese (Brazil) localization.

*   **`ui_action_info_offset2`**: Offset for action information.
    *   `40`
*   **`ui_wand_info_offset1`**: Offset for wand information (first part).
    *   `20`
*   **`ui_wand_info_offset2`**: Offset for wand information (second part).
    *   `30`
*   **`ui_configurecontrols_offset2`**: Offset for the "Configure Controls" screen.
    *   `160`
