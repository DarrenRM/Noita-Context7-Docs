# Chinese (Simplified) Language Configuration

This file configures the Simplified Chinese language for Noita. It primarily specifies the font files used for different text elements within the game.

## Key Attributes

*   **`name`**: The display name of the language.
    *   Value: `"简体中文"` (Simplified Chinese)
*   **`font_default`**: The font file used for general in-game text.
    *   Value: `"data/fonts/generated/notosans_zhcn_48.bin"`
*   **`font_inventory_title`**: The font file used for inventory titles.
    *   Value: `"data/fonts/generated/notosans_zhcn_48.bin"`
*   **`font_important_message_title`**: The font file used for important message titles.
    *   Value: `"data/fonts/generated/notosans_zhcn_48.bin"`
*   **`font_world_space_message`**: The font file used for messages displayed in the game world.
    *   Value: `"data/fonts/generated/notosans_zhcn_48.bin"`
*   **`fonts_utf8`**: A flag indicating whether UTF-8 encoding is used for fonts.
    *   Value: `"1"` (Enabled)

This configuration ensures that the game correctly displays Simplified Chinese characters using the specified Noto Sans font.
