## Font Definitions

This section specifies which font files are used for different UI elements.

### Default Fonts

*   **font\_default**: `data/fonts/font_pixel.xml` - The primary font for general text.
*   **font\_inventory\_title**: `data/fonts/font_pixel_big.xml` - Font used for inventory titles.
*   **font\_important\_message\_title**: `data/fonts/font_pixel_huge.xml` - Font for prominent message titles.
*   **font\_world\_space\_message**: `data/fonts/font_pixel.xml` - Font for messages displayed in the game world.
