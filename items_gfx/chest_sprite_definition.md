# Chest Sprite Definition

This document describes the sprite definition for the chest item in Noita.

## Sprite

The `<Sprite>` element defines the visual representation of the chest.

### Key Attributes

*   **filename**: Specifies the path to the image file used for the sprite.
    *   `data/items_gfx/chest.png`
*   **offset\_x**: The horizontal offset of the sprite from its origin.
    *   `8`
*   **offset\_y**: The vertical offset of the sprite from its origin.
    *   `6`
