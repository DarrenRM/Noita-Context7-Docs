# Pressure Bottle Sprite

This document describes the sprite definition for the Pressure Bottle item in Noita.

## Sprite Definition

The `Sprite` element defines the visual representation of the Pressure Bottle.

### Key Attributes

*   **filename**: Specifies the path to the sprite image file.
    *   `data/items_gfx/pressure_bottle.png`
*   **offset\_x**: The horizontal offset of the sprite from its origin.
    *   `5`
*   **offset\_y**: The vertical offset of the sprite from its origin.
    *   `6`
