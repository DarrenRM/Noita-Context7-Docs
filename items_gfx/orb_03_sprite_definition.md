# Orb 03 Sprite Definition

This document describes the sprite definition for "Orb 03" in Noita, focusing on its visual representation and animation.

## Sprite Attributes

The primary sprite for Orb 03 is defined by the following attributes:

*   **filename**: `data/items_gfx/orbs/orb_03.png` - The path to the image file containing the orb's graphics.
*   **offset\_x**: `20` - Horizontal offset for the sprite's origin.
*   **offset\_y**: `48` - Vertical offset for the sprite's origin.
*   **default\_animation**: `default` - Specifies the name of the default animation to be used.

## Default Animation Details

The `default` animation for Orb 03 is configured as follows:

*   **name**: `default` - The identifier for this animation.
*   **pos\_x**: `0` - X-coordinate of the animation's starting position within the sprite sheet.
*   **pos\_y**: `0` - Y-coordinate of the animation's starting position within the sprite sheet.
*   **frame\_count**: `7` - The total number of frames in this animation.
*   **frame\_width**: `40` - The width of each individual frame.
*   **frame\_height**: `50` - The height of each individual frame.
*   **frame\_wait**: `0.12` - The duration (in seconds) each frame is displayed before transitioning to the next.
*   **frames\_per\_row**: `7` - The number of frames arranged horizontally in the sprite sheet.
*   **loop**: `1` - Indicates that the animation will loop continuously (1 for true, 0 for false).
