# Canister Sprite

This documentation describes the sprite definition for the "canister" item in Noita.

## Sprite Definition

The `<Sprite>` element defines the visual representation of the canister.

### Key Attributes

*   **filename**: Specifies the path to the image file used for the sprite.
    *   `data/items_gfx/canister.png`
*   **offset\_x**: The horizontal offset of the sprite from its origin.
    *   `5`
*   **offset\_y**: The vertical offset of the sprite from its origin.
    *   `6`
