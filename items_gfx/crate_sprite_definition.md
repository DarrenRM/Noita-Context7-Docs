# Crate Sprite Definition

This document describes the sprite definition for the "crate" item in Noita, used for its visual representation.

## Sprite Attributes

The `<Sprite>` tag defines the visual appearance of the crate.

### Key Attributes:

*   **filename**: Specifies the path to the image file used for the sprite.
    *   `data/items_gfx/crate.png`
*   **offset\_x**: The horizontal offset of the sprite from its origin.
    *   `5`
*   **offset\_y**: The vertical offset of the sprite from its origin.
    *   `6`

```xml
<Sprite 
	filename="data/items_gfx/crate.png" 
	offset_x="5" 
	offset_y="6" 
	>
</Sprite>
```
