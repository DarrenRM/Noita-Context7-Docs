# Orbroom Image Emitter 07_02

This configuration defines a particle emitter for the "orbroom" visual effect, specifically using the `07_02.png` image animation.

## Entity

The root element for this particle effect.

## Base

Inherits properties from a base orbroom particle emitter configuration.

### ParticleEmitterComponent

This component controls the particle emission.

*   **image_animation_file**: Specifies the image animation file to be used for the particles.
    *   Value: `data/particles/image_emitters/orbrooms/07_02.png`
