# Dummy Target Sprite

This document describes the sprite definition for the "dummy target" prop in Noita.

## Sprite Definition

The `Sprite` element defines the visual appearance of the prop.

### Key Attributes

*   **filename**: Specifies the path to the sprite image file.
    *   `data/props_gfx/dummy_target.png`
*   **offset\_x**: The horizontal offset of the sprite's origin.
    *   `10.5`
*   **offset\_y**: The vertical offset of the sprite's origin.
    *   `23`
