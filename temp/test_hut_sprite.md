# Test Hut Sprite

This documentation describes the sprite definition for the "test_hut" asset in Noita.

## Sprite Definition

The `<Sprite>` element defines the visual representation of an in-game object.

### Key Attributes

*   **filename**: Specifies the path to the sprite image file.
    *   `data/temp/test_hut.png`
*   **offset\_x**: The horizontal offset of the sprite's origin.
    *   `28`
*   **offset\_y**: The vertical offset of the sprite's origin.
    *   `30`
