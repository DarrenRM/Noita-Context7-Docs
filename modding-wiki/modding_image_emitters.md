# Modding: Image Emitters

Image emitters are a powerful tool in Noita modding, primarily used for creating dynamic particle animations. They are integral to many visual effects, from item pickups to spell casting, and understanding them is key to adding polish and unique visual flair to your mods. This documentation covers the fundamentals of image emitters, their properties, and how to create custom effects.

## Coding an Image Emitter

An image emitter is implemented using the `ParticleEmitterComponent`. Detailed information about this component can be found in the game's `component_documentation.txt` file, located within the Noita installation directory under `tools_modding`.
