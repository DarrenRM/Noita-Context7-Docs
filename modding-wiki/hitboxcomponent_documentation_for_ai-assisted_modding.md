# HitboxComponent Documentation for AI-Assisted Modding

This documentation covers the `HitboxComponent` in Noita, a crucial element for defining the physical boundaries and interactions of entities. Understanding and manipulating this component is essential for modders looking to create custom entities, alter existing behaviors, or implement unique gameplay mechanics involving collisions and damage.

## Understanding the HitboxComponent

The `HitboxComponent` is responsible for defining the shape and properties of an entity's hitbox. This hitbox determines where an entity can be hit, what areas can damage other entities, and how collisions are registered. It's a fundamental component for any entity that needs to interact physically with the game world or other entities.
