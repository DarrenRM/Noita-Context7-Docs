# DamageNearbyEntitiesComponent

This documentation page details the `DamageNearbyEntitiesComponent` in Noita, a crucial component for modders looking to implement damage-dealing mechanics that affect entities within a certain radius. Understanding this component allows for the creation of spells, environmental hazards, or entity behaviors that inflict damage on their surroundings.

## Overview

The `DamageNearbyEntitiesComponent` is a fundamental component used in Noita's entity system to apply damage to other entities within a specified range. This is achieved by defining various parameters that control the damage type, amount, radius, and other effects.

## Component Properties

The `DamageNearbyEntitiesComponent` has several properties that can be configured within an entity's XML definition. These properties allow for fine-grained control over the damage-dealing behavior.
