# Configuring Damage Types

This documentation explains how to configure damage types in Noita, a crucial aspect for modding that allows for fine-grained control over how entities inflict and receive damage. Understanding these configurations is essential for creating balanced and interesting combat mechanics in your mods.

## Damage Type Configuration

Noita uses a system to define and manage various damage types, influencing their interactions with resistances, immunities, and other damage-related effects. This is primarily handled within the `data/config/ConfigDamagesByType.xml` file.

### `ConfigDamagesByType.xml` Structure

The main configuration file for damage types is `data/config/ConfigDamagesByType.xml`. This XML file contains a root `<ConfigDamagesByType>` element, which in turn holds multiple `<DamageType>` elements, each defining a specific damage type.
