# PotionComponent Documentation

This documentation covers the `PotionComponent` in Noita, a crucial element for understanding and modifying how potions behave within the game. By detailing its properties and associated data, modders can gain fine-grained control over potion effects, creation, and interactions, enabling more complex and personalized gameplay experiences.

## PotionComponent

The `PotionComponent` is a fundamental component in Noita's entity system that defines the properties and behaviors of potions. It dictates what effects a potion has, how it's created, and how it interacts with the game world.

### Properties

The `PotionComponent` has several properties that can be modified by modders. These properties control various aspects of a potion's functionality.
