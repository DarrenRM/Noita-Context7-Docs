# GunComponent Documentation

This document provides a comprehensive guide to the `GunComponent` in Noita, a crucial element for defining projectile behavior and weapon mechanics. Understanding `GunComponent` is essential for modders looking to create custom weapons, modify existing ones, or implement unique firing patterns and projectile effects.

## Overview of GunComponent

The `GunComponent` is an XML component that dictates how a projectile is fired from a wand or other entity. It controls aspects such as projectile speed, spread, firing rate, and the types of projectiles that can be spawned. This component is fundamental to weapon modding in Noita.

## GunComponent Properties

The `GunComponent` accepts various properties that fine-tune its behavior. These properties are defined within the `<GunComponent>` tag in your `.xml` files.
