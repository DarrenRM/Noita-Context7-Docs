# Noita Modding: List of All Tags

This document provides a comprehensive list of all tags used within Noita's modding system. Understanding these tags is crucial for AI-assisted modding, as they define the behavior, properties, and interactions of components, entities, and materials. By referencing these tags, you can accurately instruct AI tools to modify existing game elements or create new ones with specific functionalities.
