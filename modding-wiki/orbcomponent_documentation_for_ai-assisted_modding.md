# OrbComponent Documentation

This document details the `OrbComponent` in Noita, a crucial component for defining the behavior and properties of orbs. Understanding `OrbComponent` is essential for modders looking to create custom orbs, modify existing ones, or implement orb-related game mechanics. This guide breaks down its structure, properties, and common use cases.

## OrbComponent Structure and Properties

The `OrbComponent` is defined within Noita's entity system and controls various aspects of an orb's functionality. It is typically found in `.xml` files within the game's data.
