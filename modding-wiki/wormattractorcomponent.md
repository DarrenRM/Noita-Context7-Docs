# WormAttractorComponent

This documentation page details the `WormAttractorComponent` in Noita, a crucial component for understanding and modifying how certain entities attract worms. Modders can leverage this information to create custom behaviors for enemies or environmental elements that interact with worms in unique ways.

## WormAttractorComponent

The `WormAttractorComponent` is a component that can be added to entities to make them attract worms. This component is primarily used for entities that are meant to lure worms towards them, such as certain enemy types or environmental hazards.
