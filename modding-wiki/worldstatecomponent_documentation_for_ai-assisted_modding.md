# WorldStateComponent Documentation

This document details the `WorldStateComponent` in Noita, a crucial component for managing the game's world state and its various elements. Understanding this component is vital for modders looking to influence or alter how the game world behaves, including biome transitions, entity spawning, and environmental effects.

## Overview of WorldStateComponent

The `WorldStateComponent` is a fundamental entity component in Noita that governs various aspects of the game's world. It plays a significant role in defining the current state of the game world, influencing factors such as biome progression, environmental conditions, and the overall atmosphere. For modders, manipulating or understanding this component allows for deep control over world generation, event triggers, and the dynamic behavior of the game environment.

## Core Functionality and Parameters

The `WorldStateComponent` contains several key parameters that dictate its behavior. These parameters are often defined within the `data/entities/world_state.xml` file.
