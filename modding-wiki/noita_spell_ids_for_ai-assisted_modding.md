# Noita Spell IDs for AI-Assisted Modding

This documentation provides a comprehensive list of Spell IDs in Noita, crucial for AI-assisted modding. Understanding these IDs allows for precise manipulation and creation of spells within the game, enabling more complex and customized gameplay experiences. This resource is essential for modders looking to interact with or modify spell mechanics.

## Spell ID Categories

Spell IDs are primarily used within the `gun_actions.lua` file and are categorized by their action type. This organization helps in understanding the fundamental behavior of each spell.
