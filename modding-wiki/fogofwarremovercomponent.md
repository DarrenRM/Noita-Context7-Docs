# FogOfWarRemoverComponent

This documentation page explains the `FogOfWarRemoverComponent` in Noita, a crucial component for modders looking to implement custom fog of war mechanics or remove it entirely. Understanding this component allows for greater control over player visibility and exploration within modded game worlds.

## Overview

The `FogOfWarRemoverComponent` is a game entity component that, when attached to an entity, causes it to reveal a portion of the fog of war around it. This is commonly used for player characters, light sources, or any entity that should naturally uncover the map.
