# Noita Modding: Material IDs

This documentation provides a comprehensive list of material IDs used within Noita. Understanding these IDs is crucial for modders, as they are fundamental to defining and manipulating the game's various substances, from liquids and gasses to solids and fires, within custom mods.

## Material ID Categories

Noita's materials are categorized for easier reference. Each category lists the material's common name and its corresponding internal ID, which is used in game files and scripting.
