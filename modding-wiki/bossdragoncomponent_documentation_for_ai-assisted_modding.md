# BossDragonComponent Documentation

This document details the `BossDragonComponent` in Noita, a crucial component for defining the behavior and characteristics of boss dragons. Understanding this component is essential for modders looking to create custom boss encounters, modify existing ones, or integrate dragon-like entities into their mods.

## Overview of BossDragonComponent

The `BossDragonComponent` is an XML component used within Noita's entity system to define the properties and behaviors of boss dragons. It governs aspects such as their health, damage, movement patterns, and interactions with the game world. Modders can leverage this component to customize these elements or create entirely new dragon entities.

## Component Properties

The `BossDragonComponent` has several configurable properties that dictate the dragon's behavior. These are typically defined within an entity's XML definition.
