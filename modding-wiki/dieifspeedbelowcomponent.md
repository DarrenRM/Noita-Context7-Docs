# DieIfSpeedBelowComponent

This documentation page explains the `DieIfSpeedBelowComponent` in Noita, a crucial component for modders looking to implement death conditions based on player speed. Understanding this component allows for the creation of unique gameplay mechanics, challenges, and hazards that react to how fast a player is moving.

## Overview

The `DieIfSpeedBelowComponent` is a game component that causes an entity to die if its speed drops below a specified threshold. This is primarily used for player-controlled entities or objects that have a speed attribute.
