# IKLimbAttackerComponent

This documentation page details the `IKLimbAttackerComponent` in Noita, a crucial component for defining how entities attack using their limbs. Understanding this component is essential for modders looking to customize enemy attack behaviors, create new attack patterns, or modify existing ones.

## Overview

The `IKLimbAttackerComponent` is responsible for managing the logic behind an entity's limb-based attacks. It defines the properties and behaviors associated with these attacks, allowing for fine-grained control over their timing, damage, and visual effects.

## Component Properties

The `IKLimbAttackerComponent` has several properties that can be configured within an entity's XML definition. These properties allow modders to tailor the attack mechanics to their specific needs.
