# Laser Emitter Component

This documentation page details the `LaserEmitterComponent` in Noita, a crucial component for creating and modifying laser-based projectiles and effects. Understanding its properties and how to configure them is essential for modders looking to implement custom laser behaviors, from simple beam adjustments to complex, interactive energy weapons.

## Overview

The `LaserEmitterComponent` is responsible for defining the behavior and appearance of laser beams in Noita. It controls aspects such as the laser's damage, visual effects, firing patterns, and interactions with the environment and entities. This component is fundamental for any mod that introduces new laser weapons, modifies existing ones, or creates unique laser-based environmental hazards.

## Component Properties

The `LaserEmitterComponent` has a variety of properties that can be configured within the game's XML files. These properties allow for fine-grained control over every aspect of the laser's functionality.
