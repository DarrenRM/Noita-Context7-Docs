# TextLogComponent Documentation

This documentation covers the `TextLogComponent` in Noita, a crucial component for displaying text messages within the game. Understanding and utilizing this component is essential for modders who wish to implement custom UI elements, in-game notifications, or dialogue systems.

## Overview

The `TextLogComponent` is responsible for managing and displaying text messages in a log-like format. This can be used for various purposes, such as displaying game events, tutorial messages, or character dialogue.

## Component Properties

The `TextLogComponent` has several properties that can be configured in your mod's XML files.
