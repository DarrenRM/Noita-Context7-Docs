# TorchComponent

This documentation page details the `TorchComponent` in Noita, a crucial component for creating and managing light sources within the game. Understanding this component is essential for modders looking to implement custom torches, lanterns, or any entity that emits light, allowing for greater control over visual effects and gameplay mechanics related to illumination.

## TorchComponent Properties

The `TorchComponent` is attached to entities that produce light. Its properties can be modified in the entity's XML definition to alter the behavior and appearance of the light source.
