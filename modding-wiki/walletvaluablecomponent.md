# WalletValuableComponent

This documentation page explains the `WalletValuableComponent` in Noita, a crucial component for defining items that can be collected by the player's wallet. Understanding this component is essential for modders who wish to create custom collectibles, currency items, or modify existing ones to interact with the game's wallet system.

## Overview

The `WalletValuableComponent` is attached to game entities to designate them as items that can be picked up and stored within the player's wallet. This component dictates how the item is valued, whether it's a currency, and how it behaves when collected.
