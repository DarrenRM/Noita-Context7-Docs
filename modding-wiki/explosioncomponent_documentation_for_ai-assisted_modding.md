# ExplosionComponent Documentation

This document details the `ExplosionComponent` in Noita, a crucial element for understanding and modifying how explosions function within the game. By leveraging this information, modders can precisely control explosion properties, create custom explosive effects, and integrate them seamlessly into their mods.

## Understanding the ExplosionComponent

The `ExplosionComponent` is an entity component that defines the behavior and characteristics of an explosion. It is attached to entities that can cause or are involved in an explosion. Modifying this component allows for deep customization of explosive events.
