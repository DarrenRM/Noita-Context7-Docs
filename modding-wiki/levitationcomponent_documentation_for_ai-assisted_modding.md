# LevitationComponent Documentation

This document details the `LevitationComponent` in Noita, a crucial component for implementing levitation effects on entities. Understanding its properties and how to configure it is essential for modders looking to create custom levitation mechanics, spells, or environmental effects.

## Overview of LevitationComponent

The `LevitationComponent` is responsible for controlling how an entity behaves when affected by levitation. This includes its upward force, how it interacts with gravity, and its overall stability in the air. Modding this component allows for fine-tuning the physics and feel of levitation within the game.

## Component Properties

The `LevitationComponent` has several properties that can be adjusted to customize its behavior. These are typically defined within an entity's XML definition.
