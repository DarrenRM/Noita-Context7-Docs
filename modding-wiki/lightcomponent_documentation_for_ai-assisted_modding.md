# LightComponent Documentation

This documentation covers the `LightComponent` in Noita, a crucial element for controlling visual lighting effects within the game. Understanding and modifying this component is essential for modders aiming to create custom visual experiences, dynamic lighting, or unique spell effects.

## LightComponent Properties

The `LightComponent` is attached to entities and defines their light-emitting properties. These properties can be adjusted to change the color, intensity, radius, and behavior of the light.
