# GasBubbleComponent

This documentation page details the `GasBubbleComponent` in Noita, a crucial component for creating and managing gas bubbles. Understanding this component is essential for modders looking to implement custom gas effects, environmental hazards, or unique gameplay mechanics involving airborne substances.

## Overview

The `GasBubbleComponent` is responsible for defining the properties and behavior of gas bubbles within the game. This includes their visual appearance, how they interact with the environment and entities, and their underlying physical characteristics. Modders can leverage this component to create a wide variety of gas-related effects, from simple visual flourishes to complex gameplay systems.

## Component Properties

The `GasBubbleComponent` has several configurable properties that determine its behavior. These properties are typically defined within an entity's XML definition.

### Core Properties

| Property Name | Type | Description | Default Value |
|---|---|---|---|
| `bubble_material` | String | The material ID of the gas bubble. This affects its visual appearance and interaction properties. | `gas_bubble` |
| `bubble_spawn_probability` | Float | The probability (0.0 to 1.0) that a bubble will spawn when the component is active. | `1.0` |
| `bubble_lifetime` | Float | The duration (in seconds) a bubble exists before dissipating. | `5.0` |
| `bubble_radius` | Float | The initial radius of the spawned bubble. | `0.5` |
| `bubble_force` | Float | The force applied by the bubble to entities within its radius. | `100.0` |
| `bubble_force_radius` | Float | The radius within which the `bubble_force` is applied. | `2.0` |
| `bubble_damage` | Float | The amount of damage dealt by the bubble per second to entities within its radius. | `0.0` |
| `bubble_damage_radius` | Float | The radius within which `bubble_damage` is applied. | `1.0` |
| `bubble_color` | Color | The color tint of the gas bubble. | `(255, 255, 255, 255)` |
| `bubble_particle_effect` | String | The name of a particle effect to spawn when a bubble is created. | `None` |
| `bubble_sound_effect` | String | The name of a sound effect to play when a bubble is created. | `None` |
| `bubble_is_player_only` | Boolean | If true, only affects players. | `False` |
| `bubble_is_enemy_only` | Boolean | If true, only affects enemies. | `False` |
| `bubble_is_static` | Boolean | If true, the bubble does not move. | `False` |
| `bubble_is_toxic` | Boolean | If true, the bubble applies a "toxic" status effect. | `False` |
| `bubble_toxic_duration` | Float | The duration (in seconds) of the toxic status effect. | `5.0` |
| `bubble_toxic_damage` | Float | The damage dealt by the toxic status effect per second. | `1.0` |

### Spawn Conditions

| Property Name | Type | Description | Default Value |
|---|---|---|---|
| `spawn_on_damage` | Boolean | If true, a bubble spawns when the entity takes damage. | `False` |
| `spawn_on_death` | Boolean | If true, a bubble spawns when the entity dies. | `False` |
| `spawn_on_hit` | Boolean | If true, a bubble spawns when the entity is hit by something. | `False` |
| `spawn_on_use` | Boolean | If true, a bubble spawns when the entity is used (e.g., a potion). | `False` |
| `spawn_on_pickup` | Boolean | If true, a bubble spawns when the entity is picked up. | `False` |

### Visual Properties

| Property Name | Type | Description | Default Value |
|---|---|---|---|
| `bubble_texture` | String | The texture path for the gas bubble. | `data/textures/gas_bubble.png` |
| `bubble_animation_speed` | Float | The speed of the bubble's animation. | `1.0` |
| `bubble_animation_frames` | Integer | The number of frames in the bubble's animation. | `1` |
