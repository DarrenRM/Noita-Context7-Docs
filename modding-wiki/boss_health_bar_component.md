# Boss Health Bar Component

This documentation covers the `BossHealthBarComponent` in Noita, a crucial component for displaying health bars for bosses and other significant entities. Understanding and utilizing this component is essential for modders who wish to create custom boss encounters or modify existing ones with unique health bar behaviors.

## Overview

The `BossHealthBarComponent` is responsible for managing and displaying a health bar for an entity. This is typically used for bosses, but can be applied to any entity that requires a visual representation of its health. It allows for customization of the bar's appearance, behavior, and the conditions under which it is displayed.

## Component Properties

The `BossHealthBarComponent` has several properties that can be configured within an entity's XML definition. These properties control various aspects of the health bar's functionality.
