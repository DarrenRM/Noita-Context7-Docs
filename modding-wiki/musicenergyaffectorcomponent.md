# MusicEnergyAffectorComponent

This documentation page details the `MusicEnergyAffectorComponent` in Noita, a crucial component for modders looking to influence gameplay mechanics related to music and energy. Understanding this component allows for the creation of custom effects that react to or alter the game's musical state, impacting player experience and game dynamics.

## Overview

The `MusicEnergyAffectorComponent` is an XML component used within Noita's entity system. It allows for the creation of entities that can modify the player's energy based on the current music track playing in the game. This provides a powerful tool for modders to implement unique gameplay mechanics tied to the game's audio.
