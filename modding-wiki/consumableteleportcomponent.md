# ConsumableTeleportComponent

This documentation covers the `ConsumableTeleportComponent` in Noita, a crucial component for creating consumable items that trigger teleportation effects. Understanding this component is essential for modders looking to implement custom teleportation mechanics, such as potions that warp the player to specific locations or items that create temporary portals.

## Overview

The `ConsumableTeleportComponent` is attached to entities that, when consumed, initiate a teleportation event. This component defines the destination of the teleport and any associated effects or conditions.
