# Noita Modding: Perk IDs

This documentation provides a comprehensive list of Perk IDs used within Noita. Understanding these IDs is crucial for modders who wish to modify existing perks, create new ones, or interact with perk mechanics through Lua scripting. This list serves as a direct reference for implementing perk-related changes in your mods.
