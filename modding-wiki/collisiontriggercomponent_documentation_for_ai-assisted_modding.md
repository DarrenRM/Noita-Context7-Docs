# CollisionTriggerComponent Documentation

This documentation explains the `CollisionTriggerComponent` in Noita, a crucial component for defining how entities interact with each other and the environment. Understanding this component is essential for modders looking to create custom behaviors, trigger events, or modify existing entity interactions within the game.

## Understanding CollisionTriggerComponent

The `CollisionTriggerComponent` is used to define specific collision behaviors for entities. It allows you to specify what kind of entities an object should trigger upon collision, and what actions should be taken. This is fundamental for creating custom spells, enemies, environmental hazards, and more.
