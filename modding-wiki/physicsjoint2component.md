# PhysicsJoint2Component

This documentation page details the `PhysicsJoint2Component` in Noita, a crucial component for creating and managing physics-based connections between entities. Understanding this component is essential for modders looking to implement complex interactions, create articulated objects, or simulate realistic physical behaviors within their mods.

## Overview

The `PhysicsJoint2Component` is used to create a physics joint between two entities. This allows for the simulation of various physical connections, such as hinges, springs, or fixed connections, enabling more dynamic and interactive gameplay elements.
