# ConfigParallaxBackground Documentation

This documentation explains how to configure parallax backgrounds in Noita using the `ConfigParallaxBackground` component. Understanding these settings is crucial for modders who wish to customize or add new visual elements to the game's environments. By manipulating these parameters, you can control the depth, speed, and appearance of background layers, significantly impacting the game's atmosphere.

## Understanding Parallax Backgrounds

Parallax scrolling is a visual technique where background elements move slower than foreground elements, creating an illusion of depth. In Noita, this is achieved through the `ConfigParallaxBackground` component, which allows for fine-grained control over multiple background layers.

## `ConfigParallaxBackground` Component

The `ConfigParallaxBackground` component is attached to entities that should have parallax background effects. It contains several properties that define the behavior of these backgrounds.
