# CharacterDataComponent

This documentation page details the `CharacterDataComponent` in Noita, a crucial component for defining and modifying character properties. Understanding this component is essential for modders looking to alter player stats, enemy behaviors, and the fundamental attributes of any entity that possesses character-like data.

## Overview

The `CharacterDataComponent` is a fundamental XML component in Noita that governs a wide array of character-specific attributes. It controls aspects such as health, mana, stamina, movement speed, damage modifiers, and various other properties that define how an entity behaves and interacts within the game world. Modifying this component allows for deep customization of characters, from creating powerful heroes to introducing unique and challenging enemies.

## Component Structure and Properties

The `CharacterDataComponent` is defined within an entity's XML file and contains numerous properties that can be adjusted.
