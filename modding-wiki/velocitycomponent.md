# VelocityComponent

This documentation page explains the `VelocityComponent` in Noita, a crucial component for controlling the movement and physics of entities. Understanding this component is essential for modders looking to create custom behaviors, modify existing entity movement, or implement unique physics interactions within their mods.

## VelocityComponent Properties

The `VelocityComponent` governs an entity's velocity, which is a vector representing both speed and direction. It's a fundamental part of how entities interact with the game world, including gravity, collisions, and player input.
