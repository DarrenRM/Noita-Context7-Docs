# UIIconComponent Documentation

This documentation covers the `UIIconComponent` in Noita, a crucial component for displaying icons within the game's user interface. Understanding and utilizing this component is essential for modders looking to add custom icons, modify existing ones, or integrate new visual elements into the game's UI.

## Overview of UIIconComponent

The `UIIconComponent` is responsible for rendering icons in Noita's UI. It allows for the specification of an icon's visual representation, its position, and its associated data. This component is fundamental for creating visually informative and interactive UI elements in your mods.

## Component Properties

The `UIIconComponent` has several properties that can be configured to customize the appearance and behavior of an icon. These properties are typically set within an entity's XML definition.
