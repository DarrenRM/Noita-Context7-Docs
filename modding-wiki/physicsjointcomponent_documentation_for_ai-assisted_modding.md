# PhysicsJointComponent Documentation

This documentation covers the `PhysicsJointComponent` in Noita, a crucial component for creating interactive and dynamic elements within your mods. Understanding how to configure and utilize physics joints allows you to implement features like swinging platforms, connected objects, and more complex mechanical systems, significantly enhancing the gameplay experience.

## Understanding PhysicsJointComponent

The `PhysicsJointComponent` is used to define physical connections between two entities in Noita. These joints dictate how entities interact with each other under physics simulation, enabling a wide range of mechanical behaviors.

### Component Structure

The `PhysicsJointComponent` is defined within an entity's XML file. It has several key properties that control the type and behavior of the joint.
