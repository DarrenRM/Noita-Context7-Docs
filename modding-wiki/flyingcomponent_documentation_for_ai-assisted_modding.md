# FlyingComponent Documentation for AI-Assisted Modding

This document details the `FlyingComponent` in Noita, a crucial component for enabling flight mechanics in entities. Understanding its properties and how to modify them is essential for creating mods that introduce new flying creatures, alter existing flight behaviors, or implement custom movement systems.

## Understanding the FlyingComponent

The `FlyingComponent` is responsible for defining how an entity interacts with and utilizes flight. It governs aspects such as the entity's ability to fly, its flight speed, and how it handles aerial movement.

The `FlyingComponent` has several properties that can be adjusted to modify flight behavior. These properties are typically found within the entity's XML definition.
