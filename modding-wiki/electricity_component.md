# Electricity Component

This documentation covers the `ElectricityComponent` in Noita, a crucial element for creating and manipulating electrical effects within the game. Understanding this component is essential for modders looking to implement custom electrical interactions, hazards, or unique spell behaviors.

## Overview

The `ElectricityComponent` is responsible for defining how entities interact with and generate electricity. It dictates properties such as the voltage, the type of electrical effect, and how it propagates. This component is fundamental for any mod that involves electrical phenomena.

## Component Properties

The `ElectricityComponent` has several properties that can be configured in your mod's XML files. These properties allow for fine-grained control over electrical behavior.
