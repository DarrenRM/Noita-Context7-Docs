# DebugFollowMouseComponent

This documentation page explains the `DebugFollowMouseComponent` in Noita, a component primarily used for debugging purposes. Understanding this component is crucial for modders who want to implement or debug entity behavior that should react to or follow the player's mouse cursor.

## Overview

The `DebugFollowMouseComponent` is a built-in component in Noita that allows an entity to follow the mouse cursor. While its name suggests a debugging tool, it can be leveraged in modding for specific effects or to understand how entities can interact with mouse input. This section details its properties and how it functions.
