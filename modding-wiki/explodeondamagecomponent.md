# ExplodeOnDamageComponent

This documentation covers the `ExplodeOnDamageComponent` in Noita, a crucial component for creating entities that react to damage by exploding. Understanding this component is essential for modders who wish to implement custom explosion effects, trigger events upon taking damage, or modify existing entity behaviors.

## Overview

The `ExplodeOnDamageComponent` allows entities to trigger an explosion when they receive a certain amount of damage. This component is highly configurable, enabling control over the type of explosion, its radius, damage, and visual effects.
