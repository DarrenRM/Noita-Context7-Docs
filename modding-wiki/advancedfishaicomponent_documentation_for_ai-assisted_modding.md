# AdvancedFishAIComponent

This documentation covers the `AdvancedFishAIComponent` in Noita, a crucial component for creating and modifying enemy AI behaviors, particularly for fish-like entities. Understanding this component is essential for modders looking to implement custom AI patterns, reactions, and interactions for their creatures, enhancing the depth and challenge of their mods.

## Overview of AdvancedFishAIComponent

The `AdvancedFishAIComponent` is a powerful tool within Noita's modding framework that allows for the creation of complex and dynamic AI behaviors for entities, primarily designed for fish-like creatures. It provides a structured way to define how these entities perceive their environment, react to stimuli, and navigate through the game world.
