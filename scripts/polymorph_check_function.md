# Polymorph Check Function

This Lua script defines a function `polymorphing_to` which was intended for checking polymorphing behavior in Noita.

## Function Definition

### `polymorphing_to( entity_id, polymorph_target )`

This function is currently **unused** and marked for potential deletion. Its original purpose was to handle logic related to an entity being polymorphed into a `polymorph_target`.

**Parameters:**

*   `entity_id`: The unique identifier of the entity undergoing polymorphing.
*   `polymorph_target`: The target entity or state the entity is being polymorphed into.

**Note:** As of version 10.5.2023, this function is not actively used in the game's code.
