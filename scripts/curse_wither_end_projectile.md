# Curse Wither End Projectile

This script defines the behavior of a projectile that applies a "wither" curse effect, reducing damage multipliers for a specific damage type.

## Key Components and Logic

### Variable Storage and Damage Modification

The script primarily interacts with `VariableStorageComponent` and `DamageModelComponent` to achieve its effect.

*   **`effect_curse_wither_type`**: A `VariableStorageComponent` is used to store the `name` of the damage type that will be affected by the curse.
*   **`DamageModelComponent`**: The script retrieves the `DamageModelComponent` from the root entity.
*   **`damage_multipliers`**: It then accesses the `damage_multipliers` table within the `DamageModelComponent`.
*   **Damage Reduction**: The script retrieves the current multiplier for the specified `name` and reduces it by `0.25`. This effectively weakens the damage of that type.

### Core Functionality

1.  **Get Entity IDs**: Retrieves the current entity ID and its root entity ID.
2.  **Read Curse Type**: Reads the `value_string` from the `VariableStorageComponent` to determine which damage type is being cursed.
3.  **Access Damage Model**: Locates the `DamageModelComponent` on the root entity.
4.  **Modify Damage Multiplier**: Updates the `damage_multipliers` table for the specified damage type, decreasing its effectiveness.

## Example Usage (Conceptual)

While this script defines the *effect* of the curse, its application would typically be triggered by another entity or projectile that spawns this "curse\_wither\_end" projectile. For instance, a spell that applies a "poison" curse might spawn this projectile, and this script would then reduce the effectiveness of "poison" damage.
