# Gun Action Configuration

This document outlines the structure and key parameters for defining gun actions in Noita, primarily used for AI-assisted modding. It details the attributes that control a gun's behavior, effects, and spawning properties.

## Core Gun Action Properties

These attributes define the fundamental identity and behavior of a gun action.

| Attribute                       | Type    | Description                                                                 | Default Value                               |
| :------------------------------ | :------ | :-------------------------------------------------------------------------- | :------------------------------------------ |
| `action_id`                     | String  | Unique identifier for the gun action.                                       | `""`                                        |
| `action_name`                   | String  | Display name of the gun action.                                             | `""`                                        |
| `action_description`            | String  | Description of the gun action.                                              | `""`                                        |
| `action_sprite_filename`        | String  | Path to the sprite used for the action in the UI.                           | `""`                                        |
| `action_unidentified_sprite_filename` | String  | Path to the sprite used when the action is unidentified.                    | `"data/ui_gfx/gun_actions/unidentified.png"` |
| `action_type`                   | Enum    | Type of action (e.g., `ACTION_TYPE_PROJECTILE`).                            | `ACTION_TYPE_PROJECTILE`                    |
| `action_mana_drain`             | Number  | Mana cost to use the action.                                                | `10`                                        |
| `action_is_dangerous_blast`     | Boolean | Whether the action is considered a dangerous blast (affects AI behavior).   | `false`                                     |
| `action_ai_never_uses`          | Boolean | If true, AI will never use this action.                                     | `false`                                     |
| `action_never_unlimited`        | Boolean | If true, the action cannot be made unlimited.                               | `false`                                     |

## Spawning and Unlock Properties

These parameters control how and where a gun action can be found in the game.

| Attribute                   | Type    | Description                                                                 | Default Value |
| :-------------------------- | :------ | :-------------------------------------------------------------------------- | :------------ |
| `action_spawn_level`        | String  | Level tag where the action can spawn.                                       | `""`          |
| `action_spawn_probability`  | String  | Probability of spawning (e.g., "0.1").                                      | `""`          |
| `action_spawn_requires_flag`| String  | A game flag that must be active for the action to spawn.                    | `""`          |
| `action_spawn_manual_unlock`| Boolean | If true, the action must be manually unlocked by the player.                | `false`       |
| `action_max_uses`           | Number  | Maximum number of times the action can be used before disappearing (-1 for infinite). | `-1`          |

## Projectile and Effect Modifiers

These attributes modify the behavior and impact of projectiles or other effects generated by the gun action.

| Attribute                   | Type    | Description                                                                 | Default Value |
| :-------------------------- | :------ | :-------------------------------------------------------------------------- | :------------ |
| `speed_multiplier`          | Number  | Multiplier for the projectile's speed.                                      | `1.0`         |
| `explosion_radius`          | Number  | Radius of any explosion caused by the projectile.                           | `0`           |
| `spread_degrees`            | Number  | Angular spread of projectiles.                                              | `0`           |
| `screenshake`               | Number  | Intensity of screen shake when the action is used.                          | `0`           |
| `recoil`                    | Number  | Recoil force applied to the player.                                         | `0`           |
| `damage_projectile_add`     | Number  | Additional damage applied to projectiles.                                   | `0.0`         |
| `damage_fire_add`           | Number  | Additional fire damage.                                                     | `0.0`         |
| `damage_explosion_add`      | Number  | Additional explosion damage.                                                | `0.0`         |
| `knockback_force`           | Number  | Force applied to knock back entities.                                       | `0`           |
| `reload_time`               | Number  | Time in seconds before the gun can fire again.                              | `0`           |
| `material`                  | String  | The material the projectile is made of (e.g., "wood").                      | `""`          |
| `material_amount`           | Number  | Amount of the specified material to use.                                    | `0`           |
| `bounces`                   | Number  | Number of times the projectile can bounce.                                  | `0`           |
| `gravity`                   | Number  | Gravity applied to the projectile (0 for no gravity).                       | `0`           |
| `lifetime_add`              | Number  | Additional lifetime for the projectile in seconds.                          | `0`           |

## Visual and Audio Properties

These control the visual and auditory feedback associated with the gun action.

| Attribute                   | Type    | Description                                                                 | Default Value |
| :-------------------------- | :------ | :-------------------------------------------------------------------------- | :------------ |
| `sprite`                    | String  | Path to a custom sprite for the projectile itself.                          | `""`          |
| `trail_material`            | String  | Material used for the projectile's trail.                                   | `""`          |
| `trail_material_amount`     | Number  | Amount of trail material.                                                   | `0`           |
| `light`                     | Number  | Intensity of light emitted by the projectile.                               | `0`           |
| `gore_particles`            | Number  | Number of gore particles to spawn on impact.                                | `0`           |
| `ragdoll_fx`                | Number  | Controls the ragdoll effect on death.                                       | `0`           |
| `sound_loop_tag`            | String  | Tag for a looping sound effect.                                             | `""`          |

## Advanced and Special Properties

These cover more specialized functionalities and interactions.

| Attribute                   | Type    | Description                                                                 | Default Value |
| :-------------------------- | :------ | :-------------------------------------------------------------------------- | :------------ |
| `custom_xml_file`           | String  | Path to a custom XML file for more complex behavior.                        | `""`          |
| `damage_null_all`           | Number  | Sets all damage types to this value.                                        | `0.0`         |
| `damage_critical_chance`    | Number  | Chance for a critical hit.                                                  | `0`           |
| `damage_critical_multiplier`| Number  | Multiplier for critical hit damage.                                         | `0.0`         |
| `explosion_damage_to_materials` | Number  | How much damage explosions deal to environmental materials.                 | `0`           |
| `friendly_fire`             | Boolean | If true, the action can damage the player or allies.                        | `false`       |
| `extra_entities`            | String  | Comma-separated list of entity IDs to spawn alongside the projectile.       | `""`          |
| `game_effect_entities`      | String  | Comma-separated list of game effect entity IDs to apply.                    | `""`          |

---

**Note:** The `ConfigGunActionInfo_Init`, `ConfigGunActionInfo_PassToGame`, `ConfigGunActionInfo_ReadToLua`, and `ConfigGunActionInfo_Copy` functions are internal helper functions used for initializing, passing to the game, reading from, and copying gun action data respectively. They are not directly modifiable parameters but demonstrate how these properties are managed.
