# French Language Configuration

This file configures the French language settings for Noita, primarily focusing on font usage for different UI elements.

## Key Attributes

*   **`name`**: The display name of the language.
*   **`font_default`**: The default font used for most in-game text.
*   **`font_inventory_title`**: The font used for inventory titles.
*   **`font_important_message_title`**: The font used for important message titles.
*   **`font_world_space_message`**: The font used for messages displayed in the game world.
*   **`fonts_utf8`**: Indicates if UTF-8 font support is enabled.
*   **`fonts_pixel_font`**: Indicates if pixel font support is enabled.

## UI Offsets

These attributes control the positioning of certain UI elements.

*   **`ui_action_info_offset2`**: Offset for action information.
*   **`ui_wand_info_offset1`**: Offset for wand information.
*   **`ui_wand_info_offset2`**: Another offset for wand information.
*   **`ui_configurecontrols_offset2`**: Offset for the configure controls screen.
