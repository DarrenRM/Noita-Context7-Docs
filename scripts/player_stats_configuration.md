# Player Stats Configuration

This document details the configuration of player statistics within Noita, primarily focusing on the `PlayerStatsComponent`. This component governs fundamental player attributes.

## PlayerStatsComponent

The `PlayerStatsComponent` defines the core statistical properties of the player character.

### Key Attributes

*   **`max_hp`**: The maximum health points the player can possess. This is a crucial value for player survivability.

    *   **Example:** `max_hp="4.0"` sets the player's maximum health to 4.0.
