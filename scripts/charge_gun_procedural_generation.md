# Charge Gun Procedural Generation

This script defines the procedural generation logic for a "Charge Gun" in Noita. It dynamically creates variations of the gun based on random chance and rarity, influencing its core attributes and projectile types.

## Key Attributes and Their Generation

The script primarily modifies attributes of the `AbilityComponent` attached to the gun entity.
