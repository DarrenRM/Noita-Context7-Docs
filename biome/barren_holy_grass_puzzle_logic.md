# Barren Holy Grass Puzzle Logic

This entity defines the logic for a puzzle involving a specific area of "holy grass" within a barren biome. When a certain amount of this material is present within a defined area, a Lua script is triggered.

## Key Components

### MaterialAreaCheckerComponent

This component checks for the presence of a specific material within a defined rectangular area.
