# Boss Arena Biome Splice

This document describes the configuration for the Boss Arena biome, specifically how it's generated and its key properties.

## Biome Generation

The Boss Arena biome is generated by splicing a pixel scene.

### Key Attributes

*   **Pixel Scene:** `data/biome_impl/spliced/boss_arena.png`
*   **X Offset:** `1536`
*   **Y Offset:** `12288`
*   **Debug Mode:** `1` (Indicates debug output is enabled during generation)

## Generation Command

The following command is used to generate this biome:

```bash
pushd ..\..\..\
wizard_physics.exe -splice_pixel_scene data/biome_impl/spliced/boss_arena.png -x 1536 -y 12288 -debug 1
popd
```
