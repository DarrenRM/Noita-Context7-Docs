# Pyramid Hallway Biome Configuration

This document details the configuration for the Pyramid Hallway biome in Noita, focusing on its generation parameters, visual elements, and material distribution.

## Topology

The `Topology` element defines the fundamental structure and generation logic for the biome.

### Key Attributes

| Attribute Name        | Description                                                                                             | Example Value                               |
| :-------------------- | :------------------------------------------------------------------------------------------------------ | :------------------------------------------ |
| `name`                | Internal identifier for the biome.                                                                      | `$biome_pyramid`                            |
| `background_image`    | Path to the main background image for the biome.                                                        | `data/weather_gfx/background_pyramid.png`   |
| `lua_script`          | Path to the Lua script that controls biome-specific behaviors and generation.                           | `data/scripts/biomes/pyramid_hallway.lua`   |
| `noise_type`          | The primary noise algorithm used for terrain generation.                                                | `SIN_CAPPED_SIMPLEX`                        |
| `mGradientEndY`       | The Y-coordinate where the gradient effect ends.                                                        | `1024`                                      |
| `mGradientStartY`     | The Y-coordinate where the gradient effect starts.                                                      | `-1024`                                     |
| `mInsideNoiseType`    | The noise algorithm used for internal structure generation.                                             | `SimplexNoise1234`                          |
| `mInsidePerlinScaleX` | Scaling factor for Perlin noise on the X-axis for internal structures.                                | `0.96`                                      |
| `mInsidePerlinScaleY` | Scaling factor for Perlin noise on the Y-axis for internal structures.                                | `0.96`                                      |
| `audio_ambience`      | The primary ambient sound theme for the biome.                                                          | `temple`                                    |
| `audio_ambience_surface` | The ambient sound theme for the surface layer of the biome.                                           | `desert`                                    |

### Bitmap Caves

The `BitmapCaves` element defines parameters for generating cave-like structures within the biome.

| Attribute Name              | Description                                                              | Example Value |
| :-------------------------- | :----------------------------------------------------------------------- | :------------ |
| `size_x`                    | Width of the bitmap used for cave generation.                            | `512`         |
| `size_y`                    | Height of the bitmap used for cave generation.                           | `256`         |
| `blob_caves_count_min`      | Minimum number of "blob" cave formations.                                | `5`           |
| `blob_caves_radius_max`     | Maximum radius of "blob" cave formations.                                | `10`          |
| `cave_count_min`            | Minimum number of general cave structures.                               | `10`          |
| `cave_strength_max`         | Maximum strength/depth of cave structures.                               | `1`           |
| `surface_caves_count_max`   | Maximum number of surface cave structures.                               | `3`           |

## Other Elements

*   **`audio_ambience`**: Specifies the general soundscape of the biome.
*   **`audio_ambience_surface`**: Specifies the soundscape for the surface layer.
*   **`has_rain`**: A flag indicating if rain occurs in this biome (0 for no rain).

This configuration provides a foundation for understanding how the Pyramid Hallway biome is constructed and how its visual and physical characteristics are determined. Modders can leverage this information to make targeted adjustments to terrain generation, material placement, and overall biome feel.
