# Trailer Logo Entity

This entity defines the visual elements and particle effects used in the Noita trailer's logo sequence. It primarily consists of a background sprite and two particle emitters that generate animated sparks.

## Key Components

### ParticleEmitterComponent

This component is responsible for emitting particles. The entity uses two instances of this component, each with slightly different parameters to create a dynamic visual effect.
