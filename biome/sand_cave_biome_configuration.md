# Sand Cave Biome Configuration

This document details the configuration for the Sand Cave biome in Noita, focusing on its visual and material properties.

## Topology

The `<Topology>` tag defines the fundamental characteristics of the biome's appearance and generation.

### Key Attributes:

| Attribute             | Description                                                                 | Example Value                               |
| :-------------------- | :-------------------------------------------------------------------------- | :------------------------------------------ |
| `name`                | Internal identifier for the biome.                                          | `$biome_sandcave`                           |
| `type`                | Type of biome generation (Wang tiles are used for procedural generation).   | `BIOME_WANG_TILE`                           |
| `background_image`    | Path to the main background image for the biome.                            | `data/weather_gfx/background_cave_09.png`   |
| `background_edge_*`   | Paths to images for different edges of the background.                      | `data/weather_gfx/edges/background_cave_09_*.png` |
| `background_edge_priority` | Determines which background edge is drawn when overlapping.                 | `11`                                        |
| `wang_template_file`  | Path to the Wang tile image used for generating the biome's structure.      | `data/wang_tiles/sandcave.png`              |
| `lua_script`          | Path to the Lua script that governs biome-specific behaviors and events.    | `data/scripts/biomes/sandcave.lua`          |
| `wang_map_width`      | Width of the Wang tile map.                                                 | `256`                                       |
| `wang_map_height`     | Height of the Wang tile map.                                                | `256`                                       |
| `audio_music_2`       | Music track to play in this biome.                                          | `side_biome_action`                         |
| `audio_ambience`      | Ambient sound effect for the biome.                                         | `excavationsite`                            |
| `audio_ambience_surface` | Ambient sound effect for the surface layer of this biome.                   | `desert`                                    |
