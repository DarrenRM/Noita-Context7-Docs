# Snowcave Secret Chamber Biome

This document describes the configuration for the "Snowcave Secret Chamber" biome in Noita, focusing on its visual and structural properties.

## Topology

The `Topology` element defines the fundamental generation and visual characteristics of the biome.

### Key Attributes:

*   **`name`**: `_EMPTY_` - Indicates this is a base biome definition.
*   **`background_image`**: `data/weather_gfx/background_snowcave.png` - The primary background image for this biome.
*   **`background_edge_left`, `background_edge_right`, `background_edge_top`, `background_edge_bottom`**: Paths to specific edge graphics for the background, providing visual continuity.
*   **`background_edge_priority`**: `10` - Determines the rendering order of background edges. Higher values are drawn on top.
*   **`lua_script`**: `data/scripts/biomes/snowcave_secret_chamber.lua` - The associated Lua script that controls dynamic elements and behaviors within this biome.
*   **`noise_type`**: `SIN_CAPPED_SIMPLEX` - The type of noise algorithm used for terrain generation.
*   **Gradient Attributes (`mGradientAddNoise`, `mGradientEndY`, `mGradientHighNoise`, `mGradientLowNoise`, `mGradientNoiseScale`, `mGradientStartY`)**: These control the vertical gradient of the biome, influencing color and potentially other properties based on height.
*   **Inside Perlin Attributes (`mInsidePerlinClamped`, `mInsidePerlinScaleMax`, `mInsidePerlinScaleMin`, `mInsidePerlinScaleX`, `mInsidePerlinScaleY`, `mInsidePerlinScaled`, `mInsidePerlinSquared`)**: These attributes govern the use and scaling of Perlin noise for internal biome structure and detail.
*   **`mMultiplierGradient`**: `0.871429` - A multiplier applied to the gradient, affecting its overall intensity.

**Notes on Material Components:**

*   **`_enabled`**: `1` indicates the material component is active.
*   **`is_rare`**: `1` marks a material as rare.
*   **`limit_y`**: `0` means no specific vertical limit for this component.
*   **`material_index`**: Refers to the internal index of the material type.
*   **`material_max`, `material_min`**: Define the range of influence for the material.
*   **`rare_use_polka`**: `1` enables a "polka dot" distribution pattern for rare materials.
*   **`rare_polka_probability`, `rare_polka_radius_low`, `rare_polka_radius_high`**: Control the density and spread of the rare material's distribution.
*   **`rare_scale_x`, `rare_scale_y`**: Define the scaling of the noise used for the rare material's distribution.
