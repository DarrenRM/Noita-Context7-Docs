# Arrow Trap Right Sprite

This document describes the sprite definition for the right-facing arrow trap in Noita.

## Sprite Definition

The `Sprite` element defines the visual appearance of the arrow trap.

### Key Attributes

*   **filename**: Specifies the path to the sprite image file.
    *   `data/buildings_gfx/arrowtrap_right.png`
*   **offset_x**: The horizontal offset of the sprite from its origin.
    *   `5`
*   **offset_y**: The vertical offset of the sprite from its origin.
    *   `5`
