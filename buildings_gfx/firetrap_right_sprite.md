# Firetrap Right Sprite

This document describes the sprite definition for the "firetrap_right" building graphic in Noita.

## Sprite Definition

The `Sprite` element defines the visual appearance of the firetrap_right.

### Key Attributes

*   **filename**: Specifies the path to the sprite image file.
    *   `data/buildings_gfx/firetrap_right.png`
*   **offset_x**: The horizontal offset of the sprite from its origin.
    *   `5`
*   **offset_y**: The vertical offset of the sprite from its origin.
    *   `5`
