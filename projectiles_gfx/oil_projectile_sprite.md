# Oil Projectile Sprite

This documentation describes the sprite definition for the "oil" projectile in Noita.

## Sprite Definition

The `Sprite` element defines the visual appearance of the projectile.

### Key Attributes

*   **filename**: Specifies the path to the image file used for the sprite.
    *   `data/projectiles_gfx/oil.png`
*   **offset\_x**: The horizontal offset of the sprite's origin.
    *   `0.5` (Centered horizontally)
*   **offset\_y**: The vertical offset of the sprite's origin.
    *   `0.5` (Centered vertically)
