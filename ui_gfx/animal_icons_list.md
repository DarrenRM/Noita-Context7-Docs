# Animal Icons List

This document lists the internal names for various animal and enemy sprites used in the Noita UI. These names are often referenced in game files to display specific icons for creatures.

## Key Attributes

The primary function of these entries is to serve as identifiers. They are typically used in conjunction with other game data to:

*   **Display icons in the UI:** For example, showing a specific creature's portrait in a bestiary or when it appears on screen.
*   **Reference enemy types:** In scripts or data files, these names can be used to target specific enemies for effects, spawning, or AI behavior.

## Notable Entries

The following is a selection of key entries from the list, categorized for clarity:

### Player and Common Creatures

*   `player`: The main player character.
*   `sheep`: A common passive creature.
*   `fish`: Aquatic creature.
*   `duck`: Another common passive creature.
*   `coward`: A creature that flees from the player.
*   `ant`: Small, common enemy.
*   `rat`: Small, common enemy.
*   `bat`: Flying creature.
*   `fly`: Small flying creature.
*   `frog`: Amphibious creature.

### Undead and Golems
