# Black Screen Sprite

This document describes the `black_screen.xml` file, which defines a simple black sprite used in the Noita UI.

## Sprite Definition

The core of this file is a `<Sprite>` element.

### Key Attributes

*   **filename**: Specifies the image file used for the sprite. In this case, it's `data/ui_gfx/empty_black.png`.
*   **offset\_x**: The horizontal offset of the sprite from its origin.
*   **offset\_y**: The vertical offset of the sprite from its origin.
*   **position\_x**: The horizontal position of the sprite.
*   **position\_y**: The vertical position of the sprite.

```xml
<Sprite
 filename="data/ui_gfx/empty_black.png"
 offset_x="0"
 offset_y="0"
 position_x="0"
 position_y="0"
 >
</Sprite>
```
