# Dark Blue Vine Part B1

This entity represents a single segment of a dark blue vine, designed for use inverlet chains.

## Sprite Component

The `SpriteComponent` defines the visual appearance and positioning of the vine segment.

### Key Attributes:

*   **`image_file`**: `data/entities/verlet_chains/vines/vine_parts_blue/vine_dark_b_1.xml` - Specifies the sprite image to be used.
*   **`offset_x`**: `0` - Horizontal offset of the sprite.
*   **`offset_y`**: `1.5` - Vertical offset of the sprite.
*   **`update_transform`**: `0` - Disables automatic transform updates.
*   **`update_transform_rotation`**: `0` - Disables automatic rotation updates.
