# Shop Wand Level 01

This document describes the `shop_wand_level_01.xml` entity, which represents a basic wand available for purchase in the shop.

## Inheritance

This entity inherits from:
*   `data/entities/items/wand_level_02.xml`: Provides the core wand functionality.
*   `data/entities/base_shop_item.xml`: Grants shop-related properties.

## Key Components

### ItemCostComponent

*   **cost**: `350` - The price of this wand in the shop.

This is a minimal entity, primarily serving as a placeholder for a shop-purchasable wand with a specific cost. Its behavior and appearance are largely defined by its inherited base files.
