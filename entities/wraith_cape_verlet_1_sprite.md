# Wraith Cape Verlet 1 Sprite

This document describes the sprite component for a specific part of the Wraith Cape's storm effect in Noita.

## SpriteComponent

This component defines the visual appearance and positioning of the sprite.

### Key Attributes:

*   **image\_file**: `data/entities/verlet_chains/wraith_cape/wraith_storm/parts/cape_1.xml`
    *   Specifies the XML file containing the sprite's visual data.
*   **offset\_x**: `1`
    *   Horizontal offset of the sprite.
*   **offset\_y**: `12`
    *   Vertical offset of the sprite.
*   **z\_index**: `1.2`
    *   Determines the rendering order of the sprite. Higher values are rendered on top.
*   **update\_transform**: `0`
    *   Disables automatic sprite transformation based on entity movement.
*   **update\_transform\_rotation**: `0`
    *   Disables automatic sprite rotation based on entity rotation.
