# Glitter Bomb Explosion

This entity defines the effects that occur when a "Glitter Bomb" projectile explodes. It primarily spawns other entities to create visual and gameplay effects.

## Key Components

### LoadEntitiesComponent

This component is responsible for spawning other entities when the Glitter Bomb explodes.
