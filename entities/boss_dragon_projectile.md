# Boss Dragon Projectile

This document details the configuration of the Boss Dragon's projectile entity in Noita, focusing on its behavior, appearance, and effects.

## Entity Definition

The core of the projectile is defined by the `<Entity>` tag, which inherits base projectile properties from `data/entities/base_projectile.xml`.

```xml
<Entity
  name="$projectile_default"
>
  <Base file="data/entities/base_projectile.xml">
    <VelocityComponent
      mass="0.09"
    >
    </VelocityComponent>
  </Base>
  </Entity>
```

## Projectile Component

This component governs the projectile's movement, damage, and interaction with the environment.

### Key Attributes:

| Attribute                 | Description                                                                 | Value        |
| :------------------------ | :-------------------------------------------------------------------------- | :----------- |
| `lob_min`, `lob_max`      | Minimum and maximum lob angle (in radians) for projectile trajectory.       | `0.8`, `1.0` |
| `speed_min`, `speed_max`  | Minimum and maximum projectile speed.                                       | `200`, `400` |
| `friction`                | Friction applied to the projectile, affecting its deceleration.             | `1.2`        |
| `direction_random_rad`    | Random deviation in projectile direction (in radians).                      | `0.1`        |
| `die_on_low_velocity`     | If true, the projectile explodes when its velocity drops below a threshold. | `1`          |
| `on_death_explode`        | If true, the projectile explodes upon death.                                | `1`          |
| `on_lifetime_out_explode` | If true, the projectile explodes when its lifetime expires.                 | `1`          |
| `on_collision_die`        | If true, the projectile is destroyed upon collision.                        | `1`          |
| `lifetime`                | Base lifetime of the projectile in seconds.                                 | `20`         |
| `lifetime_randomness`     | Randomness added to the projectile's lifetime.                              | `7`          |
| `damage`                  | Base damage dealt by the projectile.                                        | `0.08`       |
| `knockback_force`         | Force applied to entities when hit by the projectile.                       | `1.2`        |
| `camera_shake_when_shot`  | Magnitude of camera shake when the projectile is fired.                     | `2.0`        |
| `shoot_light_flash_radius`| Radius of the light flash effect when the projectile is fired.              | `100`        |

### `config_explosion` Sub-element:

Defines the properties of the explosion that occurs when the projectile dies or its lifetime expires.

| Attribute                 | Description                                                                 | Value        |
| :------------------------ | :-------------------------------------------------------------------------- | :----------- |
| `camera_shake`            | Magnitude of camera shake during the explosion.                             | `5`          |
| `explosion_radius`        | Radius of the explosion effect.                                             | `5`          |
| `explosion_sprite`        | Path to the sprite used for the explosion effect.                           | `data/particles/explosion_016.xml` |
| `explosion_sprite_emissive`| If true, the explosion sprite is emissive.                                  | `1`          |
| `explosion_sprite_additive`| If true, the explosion sprite uses additive blending.                       | `1`          |
| `hole_enabled`            | If true, the explosion creates a hole in the environment.                   | `1`          |
| `ray_energy`              | Energy of the explosion's damaging rays.                                    | `50000`      |
| `damage`                  | Damage dealt by the explosion itself (distinct from projectile damage).     | `0.0`        |
| `damage_mortals`          | If true, the explosion damages mortal entities.                             | `1`          |
| `physics_explosion_power.min`, `physics_explosion_power.max` | Minimum and maximum power of the physics-based explosion force. | `0.1`, `0.3` |
| `physics_throw_enabled`   | If true, the explosion can throw physics objects.                           | `1`          |
| `shake_vegetation`        | If true, the explosion shakes vegetation.                                   | `1`          |
| `sparks_enabled`          | If true, sparks are generated by the explosion.                             | `1`          |
| `light_enabled`           | If true, the explosion generates light.                                     | `1`          |
| `stains_enabled`          | If true, the explosion leaves stains on surfaces.                           | `1`          |
| `stains_radius`           | Radius of the stains left by the explosion.                                 | `1`          |

## Sprite Component

Defines the visual representation of the projectile.

```xml
<SpriteComponent
  _enabled="1"
  alpha="1"
  image_file="data/projectiles_gfx/fireball_alt.xml"
  offset_x="16"
  offset_y="12"
  additive="1"
  emissive="1"
>
</SpriteComponent>
```

*   `image_file`: Specifies the graphical asset for the projectile.
*   `offset_x`, `offset_y`: Adjusts the sprite's position relative to the projectile's origin.
*   `additive`, `emissive`: Enable additive blending and emissive properties for a glowing effect.

## Particle Emitter Component

Responsible for generating cosmetic particles, such as sparks, around the projectile.

```xml
<ParticleEmitterComponent
  emitted_material_name="spark"
  x_pos_offset_min="-1"
  y_pos_offset_min="-1"
  x_pos_offset_max="1"
  y_pos_offset_max="1"
  x_vel_min="-10"
  x_vel_max="10"
  y_vel_min="0"
  y_vel_max="0"
  count_min="1"
  count_max="3"
  gravity.y="0"
  lifetime_min="0.3"
  lifetime_max="0.5"
  airflow_force="1.5"
  airflow_time="0.401"
  airflow_scale="1.05"
  fade_based_on_lifetime="1"
  create_real_particles="0"
  emit_cosmetic_particles="1"
  emission_interval_min_frames="1"
  emission_interval_max_frames="1"
  is_emitting="1"
>
</ParticleEmitterComponent>
```

*   `emitted_material_name`: The type of particle to emit (e.g., "spark").
*   `count_min`, `count_max`: The number of particles emitted per interval.
*   `lifetime_min`, `lifetime_max`: The duration of each emitted particle.
*   `emit_cosmetic_particles`: Ensures only cosmetic particles are generated.

## Audio Component

Defines the sound effects associated with the projectile.

```xml
<AudioComponent
  file="data/audio/Desktop/projectiles.bank"
  event_root="projectiles/flamethrower">
</AudioComponent>
```

*   `file`: The audio bank containing the sound events.
*   `event_root`: The specific sound event to trigger.

## Variable Storage Component

Stores variables associated with the entity.

```xml
<VariableStorageComponent
  name="projectile_file"
  value_string="data/entities/projectiles/bossdragon.xml"
>
</VariableStorageComponent>
```

*   `name`: The name of the variable.
*   `value_string`: The string value of the variable, in this case, referencing its own file path.
