# Destruction Particle Entity

This entity defines the visual and behavioral components for a particle effect that occurs during destruction events in Noita. It primarily uses `ProjectileComponent` and `ParticleEmitterComponent` to manage its behavior and visual output.

## Key Components

### ProjectileComponent

This component dictates how the particle entity behaves as a projectile, particularly its lifespan and how it interacts with the environment upon death or collision.
