# Infinite Lifetime Projectile Component

This entity defines a projectile that has an infinite lifetime, meaning it will not despawn on its own. This is achieved by attaching a Lua component that executes a script responsible for managing its behavior.

## Key Components

### LuaComponent

This component is crucial for defining the custom behavior of the entity.
