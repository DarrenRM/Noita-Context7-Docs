# Temple Door Cursor Physics

This entity defines the visual representation and basic properties of the "temple door" cursor in Noita. It's primarily used for visual feedback when interacting with certain game elements.

## Sprite Component

This component handles the visual aspects of the entity.

### Key Attributes:

*   **`image_file`**: Specifies the texture file used for the sprite.
    *   `data/props_gfx/temple_door.png`
*   **`offset_x`**: Horizontal offset for the sprite's position.
    *   `7`
*   **`offset_y`**: Vertical offset for the sprite's position.
    *   `30`
