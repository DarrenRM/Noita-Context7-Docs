# Effect Apply Wet

This entity applies the "WET" status effect to entities.

## Key Components

### GameEffectComponent

This component defines the effect to be applied and its duration.

| Attribute | Value | Description |
|---|---|---|
| `effect` | `WET` | Specifies the status effect to apply. |
| `frames` | `720` | The duration of the effect in frames (720 frames = 12 seconds at 60 FPS). |
