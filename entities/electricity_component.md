# Electricity Component

This component defines the electrical properties of an entity.

## Key Attributes

*   **`probability_to_heat`**: (float) The probability (0.0 to 1.0) that this entity will generate heat when electricity passes through it.

## Usage

This component is typically attached to entities that can conduct or be affected by electricity. It allows for dynamic interactions where electrical phenomena can cause heating effects.
