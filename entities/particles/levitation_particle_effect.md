# Levitation Particle Effect

This document describes the Noita entity responsible for the visual particle effect associated with levitation. It utilizes a `SpriteParticleEmitterComponent` to generate particles and a `LightComponent` to add a subtle glow.

## SpriteParticleEmitterComponent

This component controls the emission and behavior of the particles.
