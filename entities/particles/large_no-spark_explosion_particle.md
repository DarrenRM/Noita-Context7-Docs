# Large No-Spark Explosion Particle

This entity defines a large explosion effect that does not produce sparks. It primarily functions by loading other particle entities to create the visual and environmental effects of an explosion.

## Key Components

### LoadEntitiesComponent

This component is used to spawn other entities. In this case, it's used to instantiate various explosion-related particle effects.
