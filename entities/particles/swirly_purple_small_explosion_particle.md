# Swirly Purple Small Explosion Particle

This entity defines a small, swirly purple explosion effect. It primarily functions by loading other pre-defined particle entities to create the visual and functional aspects of the explosion.

## Key Components

### LoadEntitiesComponent

This component is used to spawn other entities. In this case, it's used to instantiate various explosion-related particle effects.
