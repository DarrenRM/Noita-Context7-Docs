# Gamepad Inventory Image Emitter

This entity defines a particle emitter that displays a gamepad image animation. It's designed to be used for UI elements related to inventory controls, specifically when using a gamepad.

## Key Components

### ParticleEmitterComponent

This component handles the emission and rendering of particles.
