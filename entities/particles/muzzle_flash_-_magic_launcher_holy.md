# Muzzle Flash - Magic Launcher Holy

This entity defines the visual and particle effects for the "Magic Launcher Holy" muzzle flash in Noita. It's designed to create a bright, energetic burst of sparks and light when the weapon is fired.

## Key Components

### SpriteComponent

This component handles the primary visual representation of the muzzle flash.
