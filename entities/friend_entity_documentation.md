# Friend Entity Documentation

This document details the configuration of the "Friend" entity in Noita, primarily focusing on its AI, combat, and physical attributes for modding purposes.

## Core Attributes

The Friend entity is a flying creature with aggressive tendencies, capable of both melee and ranged attacks. It exhibits creature detection and can flee when damaged.
