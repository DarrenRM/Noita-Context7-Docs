# Firebomb Projectile

This document details the configuration of the Firebomb projectile entity in Noita, designed for AI-assisted modding.

## Entity Definition

The core entity definition for the firebomb projectile.

```xml
<Entity name="$projectile_default" tags="projectile_player" >
    <!-- ... other components ... -->
</Entity>
```

## Key Components

### Base Projectile (`BaseComponent`)

Defines fundamental physics properties inherited by the projectile.

| Attribute      | Value   | Description                                     |
|----------------|---------|-------------------------------------------------|
| `gravity_y`    | `0`     | No vertical gravity applied to the projectile.  |
| `air_friction` | `-0.1`  | Slight resistance to air movement.              |
| `mass`         | `0.03`  | The mass of the projectile.                     |

### Projectile (`ProjectileComponent`)

Governs the behavior and effects of the projectile.

| Attribute                     | Value   | Description                                                                 |
|-------------------------------|---------|-----------------------------------------------------------------------------|
| `lob_min`, `lob_max`          | `0.9`, `1.0` | Controls the arc of the projectile.                                         |
| `speed_min`, `speed_max`      | `120`, `140` | The initial speed range of the projectile.                                  |
| `friction`                    | `0.6`   | How much the projectile slows down over time.                               |
| `direction_random_rad`        | `0.05`  | Adds a small amount of randomness to the projectile's initial direction.    |
| `on_death_explode`            | `1`     | The projectile explodes upon death.                                         |
| `on_lifetime_out_explode`     | `1`     | The projectile explodes when its lifetime expires.                          |
| `on_collision_die`            | `1`     | The projectile dies upon colliding with something.                          |
| `die_on_liquid_collision`     | `1`     | The projectile dies upon colliding with liquids.                            |
| `lifetime`                    | `70`    | The base lifetime of the projectile in frames.                              |
| `lifetime_randomness`         | `7`     | Adds randomness to the projectile's lifetime.                               |
| `damage`                      | `0`     | Base damage of the projectile itself (explosion damage is configured below).|
| `knockback_force`             | `0.6`   | The force applied to knock back entities upon collision.                    |
| `camera_shake_when_shot`      | `1.0`   | Triggers camera shake when the projectile is fired.                         |
| `shoot_light_flash_radius`    | `60`    | Radius of the light flash when the projectile is shot.                      |

#### Damage by Type (`damage_by_type`)

Specifies how much damage this projectile deals to specific damage types.

| Type  | Value |
|-------|-------|
| `fire`| `0.4` |

#### Explosion Configuration (`config_explosion`)

Details the properties of the explosion that occurs when the projectile dies.

| Attribute                 | Value   | Description                                                                 |
|---------------------------|---------|-----------------------------------------------------------------------------|
| `camera_shake`            | `4`     | Intensity of camera shake during the explosion.                             |
| `explosion_radius`        | `7`     | The radius of the explosion effect.                                         |
| `explosion_sprite`        | `data/particles/explosion_016.xml` | The visual sprite used for the explosion.                               |
| `create_cell_probability` | `20`    | Probability (in %) of creating a material cell upon explosion.              |
| `create_cell_material`    | `fire`  | The material of the cell created by the explosion.                          |
| `hole_enabled`            | `1`     | Enables the creation of holes in terrain by the explosion.                  |
| `ray_energy`              | `100000`| The energy of the explosion's damaging rays.                                |
| `damage`                  | `0`     | Base damage of the explosion itself (this is overridden by `damage_mortals`).|
| `damage_mortals`          | `1`     | Enables damage to mortal entities from the explosion.                       |
| `physics_explosion_power` | `0.05` to `0.1` | The force of the physics-based explosion.                                   |
| `sparks_enabled`          | `1`     | Enables sparks to be generated by the explosion.                            |
| `stains_enabled`          | `1`     | Enables stains to be left on surfaces by the explosion.                     |
| `stains_radius`           | `12`    | The radius of the stains left by the explosion.                             |

### Sprite (`SpriteComponent`)

Defines the visual appearance of the projectile.

| Attribute     | Value                               | Description                                     |
|---------------|-------------------------------------|-------------------------------------------------|
| `image_file`  | `data/projectiles_gfx/grenade_scavenger_small.xml` | The sprite asset used for the projectile.       |
| `additive`    | `1`                                 | Enables additive blending for the sprite.       |
| `emissive`    | `1`                                 | Makes the sprite emissive (glows).              |

### Particle Emitter (`ParticleEmitterComponent`)

Manages the emission of particles from the projectile.

| Attribute               | Value   | Description                                                                 |
|-------------------------|---------|-----------------------------------------------------------------------------|
| `emitted_material_name` | `fire`  | The material of the particles being emitted.                                |
| `count_min`, `count_max`| `1`, `1`| The number of particles emitted per emission.                               |
| `lifetime_min`, `lifetime_max` | `0.1`, `0.3` | The lifetime range of emitted particles.                                    |
| `emission_interval_min_frames`, `emission_interval_max_frames` | `1`, `2` | The frame interval between particle emissions.                              |
| `is_emitting`           | `1`     | Ensures the particle emitter is active.                                     |

### Light (`LightComponent`)

Adds a light source to the projectile.

| Attribute | Value | Description                                     |
|-----------|-------|-------------------------------------------------|
| `radius`  | `40`  | The radius of the light emitted by the projectile. |

### Audio (`AudioComponent`)

Handles sound effects associated with the projectile.

| `event_root`| `player_projectiles/bullet_fire_small`| The specific sound event to play.               |
