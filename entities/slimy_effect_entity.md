# Slimy Effect Entity

This entity defines the "SLIMY" game effect.

## Entity Structure

The entity is a simple XML structure containing a single component.

### GameEffectComponent

This component is responsible for defining the game effect.

| Attribute | Description                                     |
| :-------- | :---------------------------------------------- |
| `effect`  | The identifier for the game effect. Set to `SLIMY`. |
| `frames`  | The duration of the effect in frames. Set to `3600`. |
