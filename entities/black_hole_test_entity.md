# Black Hole Test Entity

This entity is a simple test case for the `BlackHoleComponent` and `DebugFollowMouseComponent` in Noita. It's primarily used for debugging and development purposes.

## Entity Structure

The entity is composed of two main components:

*   **`BlackHoleComponent`**: This component defines the behavior of a black hole.
*   **`DebugFollowMouseComponent`**: This component makes the entity follow the player's mouse cursor.

## Key Components and Attributes

### `BlackHoleComponent`

This component is responsible for the black hole's gravitational pull and its visual representation.
