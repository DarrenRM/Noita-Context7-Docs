# Main Swirly Red Particle Explosion

This entity defines a specific type of particle explosion, characterized by swirly red visual effects. It primarily functions by loading other pre-defined particle entities to construct its visual and behavioral components.

## Key Components

### LoadEntitiesComponent

This component is responsible for spawning other entities. In this case, it's used to instantiate various particle effects that contribute to the overall explosion.
