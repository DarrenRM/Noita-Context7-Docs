# Large Radius Explosion Particle Entity

This entity defines the components that constitute a large radius explosion effect in Noita. It primarily uses `LoadEntitiesComponent` to spawn other particle entities, creating a layered and visually complex explosion.

## Key Components

### LoadEntitiesComponent

This component is responsible for loading and spawning other entities. In this context, it's used to instantiate various particle effects that combine to form the overall explosion.
