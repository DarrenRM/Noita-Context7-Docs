# Circular Muzzle Flash Particle

This entity defines a circular muzzle flash effect, primarily used for visual feedback when a projectile is fired. It utilizes a sprite animation and a particle emitter to create a dynamic visual.

## Key Components

### SpriteComponent

This component handles the visual representation of the muzzle flash.
