# Wand Trap - Electricity

This entity defines a "wand trap" that triggers an electrical effect.

## Key Components

### Entity

*   **tags**: `wand_trap` - Identifies this entity as a wand trap.

### VariableStorageComponent

*   **name**: `entity_file`
*   **value\_string**: `data/entities/misc/electricity.xml` - Specifies the entity file to be spawned when the trap is triggered. This points to the definition of the electrical effect.
