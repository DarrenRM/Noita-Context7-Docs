# Scavenger Glue Entity Documentation

This document details the `scavenger_glue.xml` entity, a flying enemy creature in Noita.

## Core Attributes

The Scavenger Glue is a flying enemy with both melee and ranged attack capabilities. It can detect creatures within a significant range and will attempt to escape if damaged.
