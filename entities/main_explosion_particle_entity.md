# Main Explosion Particle Entity

This entity defines the core components and particle effects that constitute a generic explosion in Noita. It acts as a container, loading various other particle entities to create a visually complex explosion.

## Key Components

### LoadEntitiesComponent

This component is responsible for loading other entity files. In this context, it's used to spawn different types of particles that make up the explosion.
