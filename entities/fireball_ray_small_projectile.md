# Fireball Ray Small Projectile

This document details the `fireball_ray_small.xml` entity, representing a small, fast-moving fireball projectile in Noita. It's designed for AI-assisted modding, highlighting key attributes for customization.

## Core Attributes

The `Entity` tag defines the projectile with a base type and player projectile tag.

```xml
<Entity
  name="$projectile_default"
  tags="projectile_player"
>
  <!-- ... other components ... -->
</Entity>
```

## Base Projectile Configuration

Inherits from `base_projectile.xml`, defining fundamental physics properties.

### VelocityComponent

Controls the projectile's movement and physical interactions.

| Attribute   | Value   | Description                                     |
| :---------- | :------ | :---------------------------------------------- |
| `gravity_y` | `100`   | Downward acceleration due to gravity.           |
| `mass`      | `0.09`  | The projectile's mass, affecting its momentum. |

## Projectile Component

This is the primary component defining the projectile's behavior and effects.

### Key Attributes

| Attribute                 | Value   | Description                                                                 |
| :------------------------ | :------ | :-------------------------------------------------------------------------- |
| `lob_min`, `lob_max`      | `0.8`, `1.0` | Controls the arc of the projectile's trajectory.                            |
| `speed_min`, `speed_max`  | `160`, `170` | The range of initial speeds for the projectile.                             |
| `die_on_low_velocity`     | `0`     | Projectile does not disappear if its velocity drops too low.                |
| `on_death_explode`        | `1`     | The projectile explodes upon death (e.g., hitting a surface).               |
| `on_lifetime_out_explode` | `1`     | The projectile explodes when its lifetime expires.                          |
| `die_on_liquid_collision` | `1`     | The projectile is destroyed upon colliding with liquids.                    |
| `damage`                  | `0`     | Base damage of the projectile itself (before type-specific modifiers).      |
| `on_collision_die`        | `1`     | The projectile is destroyed upon collision with any entity.                 |
| `lifetime`                | `60`    | The maximum duration (in frames) the projectile exists before expiring.     |
| `shoot_light_flash_radius`| `150`   | The radius of the light flash emitted when the projectile is fired.         |
| `knockback_force`         | `1.5`   | The force applied to entities that are hit by the projectile.               |
| `physics_impulse_coeff`   | `2800`  | Coefficient for physics impulse, affecting how it interacts with physics. |

### Explosion Configuration (`config_explosion`)

Defines the parameters of the explosion that occurs upon the projectile's death.

| Attribute                 | Value   | Description                                                                   |
| :------------------------ | :------ | :---------------------------------------------------------------------------- |
| `camera_shake`            | `4.5`   | Intensity of camera shake during the explosion.                               |
| `explosion_radius`        | `16`    | The radius of the explosion's effect.                                         |
| `create_cell_probability` | `50`    | Probability (in percent) of creating new material cells (e.g., fire).         |
| `create_cell_material`    | `fire`  | The material to create when cells are generated by the explosion.             |
| `hole_enabled`            | `1`     | Enables the creation of holes in terrain by the explosion.                    |
| `damage`                  | `0.8`   | Damage dealt by the explosion itself.                                         |
| `ray_energy`              | `50000` | Energy value for ray-based effects of the explosion.                          |
| `particle_effect`         | `1`     | Enables particle effects associated with the explosion.                       |
| `damage_mortals`          | `1`     | The explosion deals damage to living entities.                                |
| `physics_explosion_power` | `0.6` - `0.9` | The minimum and maximum power of the physics-based explosion force.         |
| `sparks_enabled`          | `1`     | Enables the generation of sparks during the explosion.                        |
| `stains_enabled`          | `1`     | Enables the creation of impact stains on surfaces.                            |
| `audio_event_name`        | `explosions/fireball` | The audio event triggered by the explosion.                               |

## Particle Emitters

This projectile utilizes multiple `ParticleEmitterComponent`s to generate visual effects.

### Smoke Emitter

Generates smoke particles behind the projectile.

| Attribute             | Value   | Description                                                              |
| :-------------------- | :------ | :----------------------------------------------------------------------- |
| `emitted_material_name` | `smoke` | The material of the particles being emitted.                             |
| `count_min`, `count_max`| `5`, `5`  | The number of particles emitted per emission cycle.                      |
| `lifetime_min`, `lifetime_max` | `0.1`, `0.3` | The duration (in seconds) each particle exists.                          |
| `create_real_particles` | `1`     | Emits actual game particles that interact with the world.                |

### Spark Emitter

Generates sparks, likely for visual flair and trail effects.

| Attribute             | Value   | Description                                                              |
| :-------------------- | :------ | :----------------------------------------------------------------------- |
| `emitted_material_name` | `spark` | The material of the particles being emitted.                             |
| `is_trail`            | `1`     | Indicates that these particles form a trail behind the projectile.       |
| `render_on_grid`      | `1`     | Particles are rendered on the game grid.                                 |
| `fade_based_on_lifetime`| `1`     | Particles fade out as their lifetime decreases.                          |
| `create_real_particles` | `0`     | Emits cosmetic particles that do not interact with the game world.       |
