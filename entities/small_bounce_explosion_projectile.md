# Small Bounce Explosion Projectile

This document details the configuration for a small explosion projectile in Noita, designed for AI-assisted modding.

## Entity Definition

The core of this projectile is defined by the `<Entity>` tag.

```xml
<Entity
  name="$projectile_default" tags="projectile_player"
>
  <!-- ... components ... -->
</Entity>
```

*   **`name="$projectile_default"`**: Inherits default projectile naming conventions.
*   **`tags="projectile_player"`**: Identifies this projectile as originating from the player.

## Base Projectile Configuration

The `<Base>` tag inherits common projectile properties.

```xml
<Base file="data/entities/base_projectile.xml">
  <VelocityComponent>
    <!-- Velocity properties are typically defined here, but are empty for this projectile -->
  </VelocityComponent>
</Base>
```

*   **`file="data/entities/base_projectile.xml"`**: Links to the base projectile template.

## Projectile Component

The `<ProjectileComponent>` defines the specific behavior and effects of this projectile.

```xml
<ProjectileComponent
  _enabled="1"
  lob_min="0.8"
  lob_max="1.0"
  speed_min="0"
  speed_max="0"
  die_on_low_velocity="1"
  on_death_explode="1"
  on_death_gfx_leave_sprite="0"
  on_lifetime_out_explode="1"
  explosion_dont_damage_shooter="0"
  on_collision_die="1"
  shoot_light_flash_radius="15"
  shoot_light_flash_r="255"
  shoot_light_flash_g="10"
  shoot_light_flash_b="255"
  damage="0"
  lifetime="0"
>
  <!-- ... config_explosion ... -->
</ProjectileComponent>
```

**Key Attributes:**

*   **`lob_min`, `lob_max`**: Controls the projectile's lobbing behavior (0.8 to 1.0 suggests a slight arc).
*   **`speed_min`, `speed_max`**: Set to 0, indicating the projectile's speed is not directly controlled here and likely determined by its origin or other factors.
*   **`die_on_low_velocity="1"`**: The projectile will be destroyed if its velocity drops too low.
*   **`on_death_explode="1"`**: Triggers an explosion upon death.
*   **`on_lifetime_out_explode="1"`**: Triggers an explosion when its lifetime expires.
*   **`on_collision_die="1"`**: The projectile is destroyed upon collision.
*   **`shoot_light_flash_radius`, `shoot_light_flash_r`, `shoot_light_flash_g`, `shoot_light_flash_b`**: Defines a light flash effect when the projectile is fired, with a specific radius and color (purple/pink).
*   **`damage="0"`**: The projectile itself deals no direct damage. Damage is handled by the explosion.
*   **`lifetime="0"`**: The projectile has no inherent lifetime; it's likely destroyed by collision or low velocity.

### Explosion Configuration (`config_explosion`)

This nested tag defines the parameters of the explosion that occurs when the projectile dies.

```xml
<config_explosion
  never_cache="1"
  camera_shake="5.5"
  explosion_radius="15"
  explosion_sprite="data/particles/explosion_016.xml"
  explosion_sprite_lifetime="0"
  create_cell_probability="0"
  hole_destroy_liquid="0"
  ray_energy="6000"
  damage="0.7"
  hole_enabled="1"
  hole_image="data/temp/explosion_hole.png"
  particle_effect="1"
  damage_mortals="1"
  physics_explosion_power.min="0.3"
  physics_explosion_power.max="0.6"
  physics_throw_enabled="1"
  shake_vegetation="1"
  sparks_enabled="1"
  sparks_count_max="12"
  sparks_count_min="8"
  spark_material="fire"
  light_fade_time="0.8"
  stains_enabled="1"
  stains_image="data/temp/explosion_stain.png"
  audio_event_name="explosions/tnt"
>
</config_explosion>
```

**Key Attributes:**

*   **`never_cache="1"`**: Prevents caching of this explosion effect.
*   **`camera_shake="5.5"`**: The intensity of camera shake caused by the explosion.
*   **`explosion_radius="15"`**: The radius of the explosion's effect.
*   **`explosion_sprite="data/particles/explosion_016.xml"`**: Specifies the visual sprite for the explosion.
*   **`ray_energy="6000"`**: The energy of the explosion's damaging rays.
*   **`damage="0.7"`**: The damage dealt by the explosion.
*   **`hole_enabled="1"`**: Enables the creation of holes in the environment.
*   **`particle_effect="1"`**: Enables particle effects for the explosion.
*   **`damage_mortals="1"`**: The explosion will damage living entities.
*   **`physics_explosion_power.min`, `physics_explosion_power.max`**: Controls the force of the physics-based explosion.
*   **`physics_throw_enabled="1"`**: Objects affected by the explosion will be thrown.
*   **`shake_vegetation="1"`**: Vegetation will be shaken by the explosion.
*   **`sparks_enabled="1"`**: Sparks will be generated by the explosion.
*   **`sparks_count_min`, `sparks_count_max`**: The range for the number of sparks.
*   **`spark_material="fire"`**: The material type of the sparks.
*   **`stains_enabled="1"`**: Enables the creation of explosion stains on surfaces.
*   **`audio_event_name="explosions/tnt"`**: The sound effect played for the explosion.

## Variable Storage Component

This component stores a reference to the projectile's own XML file.

```xml
<VariableStorageComponent
  name="projectile_file"
  value_string="data/entities/projectiles/deck/bounce_small_explosion.xml"
>
</VariableStorageComponent>
```

*   **`name="projectile_file"`**: The name of the variable.
*   **`value_string="..."`**: The value, which is the path to this entity's XML file.
