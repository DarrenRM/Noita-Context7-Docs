# Cord Part A2 Verlet Chain

This document describes the `cord_verlet_a_2.xml` entity, which represents a segment of a verlet chain cord in Noita.

## Sprite Component

The `SpriteComponent` defines the visual representation of this cord segment.

### Key Attributes:

*   **`image_file`**: Specifies the sprite image used for this cord part. In this case, it points to `data/entities/verlet_chains/cords/cord_a_2.xml`.
*   **`offset_x`**: Horizontal offset for the sprite.
*   **`offset_y`**: Vertical offset for the sprite.
*   **`update_transform`**: Controls whether the sprite's transform is updated. Set to `0` (false) here.
*   **`update_transform_rotation`**: Controls whether the sprite's rotation is updated. Set to `0` (false) here.
