# Green Explosion Particle Entity

This entity defines the visual components of a green explosion particle effect in Noita. It primarily uses `LoadEntitiesComponent` to spawn other pre-defined particle entities, creating a layered and dynamic explosion.

## Key Components

### LoadEntitiesComponent

This component is responsible for loading and spawning other entities. In this case, it's used to instantiate various explosion-related particle effects.
