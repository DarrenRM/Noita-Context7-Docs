# Verlet Worm Entity

This document describes the `verlet_worm.xml` entity, which defines a segmented worm-like creature using Noita's Verlet physics system.

## Entity Structure

The `verlet_worm.xml` entity is composed of a `VerletPhysicsComponent` and several `Base` components that define its constituent parts.

## VerletPhysicsComponent

This component governs the physics simulation of the worm's segmented body.
