# Pink Big Slow Orb Projectile

This document details the configuration of the "Pink Big Slow Orb" projectile entity in Noita, designed for AI-assisted modding.

## Core Entity Configuration

The projectile is based on the `base_projectile.xml` entity, inheriting its fundamental properties.

```xml
<Entity
  name="$projectile_default">
  <Base file="data/entities/base_projectile.xml">
    <VelocityComponent
      gravity_y="0"
      air_friction="0"
      mass="0.8">
    </VelocityComponent>
  </Base>
  <!-- ... other components ... -->
</Entity>
```

### Key Attributes:

*   **`gravity_y="0"`**: The projectile is not affected by gravity.
*   **`air_friction="0"`**: No air resistance acts upon the projectile.
*   **`mass="0.8"`**: Defines the projectile's mass.

## Projectile Behavior

The `ProjectileComponent` governs the projectile's movement, damage, and death effects.

```xml
<ProjectileComponent
  _enabled="1"
  lob_min="0.8"
  lob_max="1.0"
  speed_min="100"
  speed_max="120"
  die_on_low_velocity="0"
  on_death_explode="1"
  on_death_gfx_leave_sprite="0"
  on_lifetime_out_explode="1"
  explosion_dont_damage_shooter="1"
  damage="0.8"
  on_collision_die="1"
  lifetime="100"
  knockback_force="2.0">
  <!-- ... explosion configuration ... -->
</ProjectileComponent>
```

### Key Attributes:

*   **`lob_min="0.8"`, `lob_max="1.0"`**: Controls the projectile's lobbing behavior, influencing its trajectory.
*   **`speed_min="100"`, `speed_max="120"`**: Sets the minimum and maximum speed of the projectile.
*   **`on_death_explode="1"`**: The projectile explodes upon death.
*   **`on_lifetime_out_explode="1"`**: The projectile explodes when its lifetime expires.
*   **`damage="0.8"`**: The base damage dealt by the projectile.
*   **`on_collision_die="1"`**: The projectile is destroyed upon collision.
*   **`lifetime="100"`**: The duration in frames before the projectile expires.
*   **`knockback_force="2.0"`**: The force applied to entities upon collision.

### Explosion Configuration:

The `config_explosion` sub-element defines the properties of the explosion that occurs upon the projectile's death.

```xml
<config_explosion
  never_cache="1"
  camera_shake="4.0"
  explosion_radius="20"
  explosion_sprite="data/particles/explosion_032_pink.xml"
  create_cell_probability="1"
  create_cell_material="plasma_fading_pink"
  ray_energy="10000"
  hole_destroy_liquid="1"
  hole_enabled="1"
  damage="0.5"
  physics_explosion_power.min="0.4"
  physics_explosion_power.max="0.6"
  physics_throw_enabled="1"
  sparks_enabled="1"
  spark_material="plasma_fading_pink"
  sparks_count_min="12"
  sparks_count_max="16"
  light_r="155"
  light_g="15"
  light_b="140">
</config_explosion>
```

#### Key Explosion Attributes:

*   **`camera_shake="4.0"`**: The intensity of camera shake during the explosion.
*   **`explosion_radius="20"`**: The radius of the explosion effect.
*   **`explosion_sprite="data/particles/explosion_032_pink.xml"`**: The visual sprite used for the explosion.
*   **`create_cell_material="plasma_fading_pink"`**: The material of cells created by the explosion.
*   **`ray_energy="10000"`**: The energy of any rays generated by the explosion.
*   **`hole_destroy_liquid="1"`**: Enables the creation of holes that destroy liquids.
*   **`damage="0.5"`**: The damage dealt by the explosion itself.
*   **`physics_explosion_power.min="0.4"`, `physics_explosion_power.max="0.6"`**: The minimum and maximum physics force applied by the explosion.
*   **`sparks_enabled="1"`**: Enables the spawning of sparks.
*   **`spark_material="plasma_fading_pink"`**: The material of the sparks.
*   **`light_r="155"`, `light_g="15"`, `light_b="140"`**: The RGB values for the light emitted by the explosion.

## Visual Representation

The projectile's appearance is defined by `SpriteComponent` elements.

```xml
<SpriteComponent
  _enabled="1"
  alpha="1"
  image_file="data/projectiles_gfx/orb_pink_big.xml"
  offset_x="8"
  offset_y="8"
  rect_animation="fireball">
</SpriteComponent>
```

### Key Attributes:

*   **`image_file="data/projectiles_gfx/orb_pink_big.xml"`**: Specifies the sprite image file.
*   **`offset_x="8"`, `offset_y="8"`**: Adjusts the sprite's position.
*   **`rect_animation="fireball"`**: Defines the sprite's animation.

There are two identical `SpriteComponent` entries, suggesting a potential redundancy or specific layering for visual effect.

### Key Attributes:

*   **`r="150"`, `g="35"`, `b="120"`**: The RGB color values of the light.

### Key Attributes:

*   **`event_root="projectiles/orb_b"`**: The root event for projectile sounds.

## Variable Storage

The `VariableStorageComponent` stores the projectile's own file path.

```xml
<VariableStorageComponent
  name="projectile_file"
  value_string="data/entities/projectiles/orb_pink_big_slow.xml">
</VariableStorageComponent>
```

### Key Attributes:

*   **`name="projectile_file"`**: The name of the variable.
*   **`value_string="data/entities/projectiles/orb_pink_big_slow.xml"`**: The string value, which is the path to this entity's XML file.
