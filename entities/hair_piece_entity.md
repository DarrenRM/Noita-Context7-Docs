# Hair Piece Entity

This document describes the `hair_piece.xml` entity, which appears to be a visual component for a boss limb in Noita.

## SpriteComponent

This component defines the visual representation of the entity.

### Key Attributes:

*   **image\_file**: `data/entities/animals/boss_limbs/hair_piece.png` - Specifies the texture file used for the sprite.
*   **offset\_x**: `0` - Horizontal offset of the sprite.
*   **offset\_y**: `1` - Vertical offset of the sprite.
*   **update\_transform**: `0` - Indicates whether the sprite's transform should be updated.
*   **update\_transform\_rotation**: `0` - Indicates whether the sprite's rotation should be updated.
