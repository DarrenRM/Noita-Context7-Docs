# Green Firework Projectile

This document details the configuration for a green firework projectile in Noita, designed for AI-assisted modding.

## Core Projectile Properties

The `ProjectileComponent` defines the fundamental behavior of the projectile.

| Attribute                 | Value      | Description                                                                 |
| :------------------------ | :--------- | :-------------------------------------------------------------------------- |
| `lob_min`, `lob_max`      | `0.8`, `1.0` | Controls the arc of the projectile. Higher values mean a flatter trajectory. |
| `speed_min`, `speed_max`  | `70`, `80` | The initial speed range of the projectile.                                  |
| `on_death_explode`        | `1`        | The projectile will explode upon death (e.g., hitting a surface).           |
| `on_lifetime_out_explode` | `1`        | The projectile will explode when its lifetime expires.                      |
| `on_collision_die`        | `1`        | The projectile will be destroyed upon collision.                            |
| `lifetime`                | `30`       | The base lifetime of the projectile in frames.                              |
| `lifetime_randomness`     | `7`        | Adds random variation to the projectile's lifetime.                         |
| `damage`                  | `1`        | Base damage dealt by the projectile.                                        |
| `knockback_force`         | `3.0`      | The force applied to entities upon impact.                                  |
| `camera_shake_when_shot`  | `15.0`     | The intensity of camera shake when this projectile is fired.                |

## Explosion Configuration

The `config_explosion` element within `ProjectileComponent` dictates the effects of the projectile's explosion.

| Attribute                 | Value      | Description                                                                                             |
| :------------------------ | :--------- | :------------------------------------------------------------------------------------------------------ |
| `never_cache`             | `1`        | Ensures the explosion entity is always loaded fresh.                                                    |
| `camera_shake`            | `20`       | The intensity of camera shake caused by the explosion.                                                  |
| `explosion_radius`        | `15`       | The radius of the explosion's effect.                                                                   |
| `load_this_entity`        | `data/entities/particles/fireworks/firework_green.xml` | The entity to load and spawn as the explosion effect.                                   |
| `create_cell_probability` | `10`       | Probability (out of 100) of creating liquid cells from the explosion.                                   |
| `hole_enabled`            | `1`        | Enables the creation of holes in terrain by the explosion.                                              |
| `ray_energy`              | `2000000`  | The energy of the explosion's damaging rays.                                                            |
| `damage`                  | `1.2`      | Damage dealt by the explosion itself.                                                                   |
| `physics_explosion_power.min`, `physics_explosion_power.max` | `1.0`, `1.4` | The range of force applied to physics objects by the explosion.                                         |
| `spark_material`          | `spark_green` | The material used for sparks generated by the explosion.                                                |
| `sparks_count_min`, `sparks_count_max` | `7`, `20` | The range for the number of sparks generated.                                                           |
| `audio_event_name`        | `explosions/glitter` | The audio event triggered by the explosion.                                                             |

## Visual and Light Effects

### Light Component

The `LightComponent` adds a persistent light source while the projectile is active.

| Attribute | Value | Description                               |
| :-------- | :---- | :---------------------------------------- |
| `r`, `g`, `b` | `45`, `255`, `10` | The RGB color of the light. |
| `radius`  | `150` | The radius of the light's illumination.   |

### Muzzle Flash

The `muzzle_flash_file` attribute in `ProjectileComponent` specifies the particle effect used for the muzzle flash when the projectile is fired.

```xml
<ProjectileComponent
    ...
    muzzle_flash_file="data/entities/particles/muzzle_flashes/muzzle_flash_launcher.xml"
    ...
>
```

## Particle Emitters

Multiple `ParticleEmitterComponent` elements are used to create various visual effects associated with the firework.

### Smoke Emitter

This emitter generates basic smoke particles.

| Attribute               | Value   | Description                                     |
| :---------------------- | :------ | :---------------------------------------------- |
| `emitted_material_name` | `smoke` | The material of the particles to emit.          |
| `count_min`, `count_max`| `5`, `5`  | Number of particles emitted per burst.          |
| `lifetime_min`, `lifetime_max` | `0.1`, `0.3` | Duration of each particle.                      |
| `is_emitting`           | `1`     | Enables the particle emitter.                   |

### Green Spark Emitters (Multiple)

These emitters generate green sparks, contributing to the firework's visual flair. They differ primarily in their `gravity.y` value, creating upward, downward, and neutral trajectories for the sparks.

| Attribute               | Value   | Description                                     |
| :---------------------- | :------ | :---------------------------------------------- |
| `emitted_material_name` | `spark_green` | The material of the particles to emit.          |
| `count_min`, `count_max`| `1`, `8`  | Number of particles emitted per burst.          |
| `lifetime_min`, `lifetime_max` | `0.3`, `0.5` | Duration of each particle.                      |
| `is_trail`              | `1`     | Particles leave a trail effect.                 |
| `render_on_grid`        | `1`     | Particles are rendered on the game grid.        |
| `emit_cosmetic_particles` | `1` | Emits particles that are purely visual.         |
| `is_emitting`           | `1`     | Enables the particle emitter.                   |

### Sparse Spark Emitter

This emitter creates fewer, longer-lasting sparks with a specific velocity and emission interval.

| Attribute               | Value   | Description                                     |
| :---------------------- | :------ | :---------------------------------------------- |
| `emitted_material_name` | `spark_green` | The material of the particles to emit.          |
| `count_min`, `count_max`| `1`, `1`  | Number of particles emitted per burst.          |
| `x_vel_min`, `x_vel_max`| `10`, `20` | Horizontal velocity range of sparks.            |
| `lifetime_min`, `lifetime_max` | `1.4`, `1.5` | Duration of each particle.                      |
| `emission_interval_min_frames`, `emission_interval_max_frames` | `8`, `12` | Delay between particle bursts.                  |
| `is_emitting`           | `1`     | Enables the particle emitter.                   |

## Audio Component

The `AudioComponent` links the projectile to specific sound effects.

| Attribute    | Value                               | Description                               |
| :----------- | :---------------------------------- | :---------------------------------------- |
| `file`       | `data/audio/Desktop/projectiles.bank` | The audio bank containing the sound event. |
| `event_root` | `player_projectiles/bullet_rocket`  | The root event name for projectile sounds. |

## Variable Storage

The `VariableStorageComponent` stores the projectile's own file path, which can be useful for other game logic.

| Attribute    | Value                                       | Description                               |
| :----------- | :------------------------------------------ | :---------------------------------------- |
| `name`       | `projectile_file`                           | The name of the variable.                 |
| `value_string` | `data/entities/projectiles/deck/fireworks/firework_green.xml` | The string value stored.                  |
