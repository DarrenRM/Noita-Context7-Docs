# Lance Projectile

This document details the configuration of the "Lance" projectile entity in Noita, designed for AI-assisted modding.

## Core Components

The Lance projectile is defined by several key components that govern its behavior, appearance, and effects.

### ProjectileComponent

This is the primary component defining the projectile's physics, damage, and interaction properties.

| Attribute                 | Description                                                                                             |
| :------------------------ | :------------------------------------------------------------------------------------------------------ |
| `_enabled`                | Enables or disables the component (1 for enabled).                                                      |
| `lob_min`, `lob_max`      | Minimum and maximum lob angle in radians, affecting projectile trajectory.                               |
| `speed_min`, `speed_max`  | Minimum and maximum projectile speed.                                                                   |
| `direction_random_rad`    | Random deviation in projectile direction in radians.                                                    |
| `on_death_explode`        | If the projectile explodes upon death (1 for true).                                                     |
| `on_lifetime_out_explode` | If the projectile explodes when its lifetime expires (1 for true).                                      |
| `die_on_low_velocity`     | If the projectile dies when its velocity drops below a certain threshold (1 for true).                  |
| `on_collision_die`        | If the projectile dies upon collision with an entity (1 for true).                                      |
| `lifetime`                | Duration in frames before the projectile expires.                                                       |
| `ground_penetration_coeff`| Coefficient determining how much the projectile penetrates the ground.                                  |
| `damage`                  | Base damage dealt by the projectile.                                                                    |
| `knockback_force`         | Force applied to the target upon impact.                                                                |
| `friendly_fire`           | Allows the projectile to damage the shooter (1 for true).                                               |
| `collide_with_shooter_frames` | Number of frames the projectile will ignore collisions with the shooter.                                |
| `muzzle_flash_file`       | Path to the muzzle flash particle effect.                                                               |
| `shoot_light_flash_radius`| Radius of the light flash emitted when the projectile is shot.                                          |
| `shoot_light_flash_r`, `g`, `b` | RGB values for the shoot light flash color.                                                             |

#### `config_explosion`

Nested within `ProjectileComponent`, this defines the properties of the explosion when `on_death_explode` or `on_lifetime_out_explode` is true.

| Attribute                 | Description                                                                                             |
| :------------------------ | :------------------------------------------------------------------------------------------------------ |
| `damage`                  | Damage dealt by the explosion (set to 0 here, likely relying on projectile damage).                     |
| `camera_shake`            | Intensity of camera shake caused by the explosion.                                                      |
| `explosion_radius`        | Radius of the explosion effect.                                                                         |
| `explosion_sprite`        | Path to the sprite used for the explosion visual.                                                       |
| `hole_enabled`            | If the explosion creates a hole in the environment (1 for true).                                        |
| `hole_image`              | Path to the image used for the explosion hole.                                                          |
| `ray_energy`              | Energy value for raycasting effects from the explosion.                                                 |
| `physics_explosion_power` | Minimum and maximum power of the physics-based explosion force.                                         |
| `sparks_enabled`          | If sparks are generated by the explosion (1 for true).                                                  |
| `spark_material`          | The material type of the sparks.                                                                        |
| `light_enabled`           | If the explosion emits light (1 for true).                                                              |
| `light_r`, `g`, `b`       | RGB values for the explosion light color.                                                               |
| `stains_enabled`          | If the explosion leaves stains on surfaces (1 for true).                                                |
| `stains_radius`           | Radius of the stains left by the explosion.                                                             |

### SpriteComponent

Defines the visual representation of the projectile.

| Attribute      | Description                                                                                             |
| :------------- | :------------------------------------------------------------------------------------------------------ |
| `_enabled`     | Enables or disables the component (1 for enabled).                                                      |
| `image_file`   | Path to the sprite image file.                                                                          |
| `additive`     | If the sprite uses additive blending (1 for true).                                                      |

### ParticleEmitterComponent

Responsible for generating particle effects, such as trails.

| Attribute                 | Description                                                                                             |
| :------------------------ | :------------------------------------------------------------------------------------------------------ |
| `emitted_material_name`   | The material name of the particles to be emitted.                                                       |
| `gravity.y`               | Vertical gravity applied to emitted particles.                                                          |
| `x_vel_min`, `x_vel_max`  | Minimum and maximum horizontal velocity of emitted particles.                                           |
| `y_vel_min`, `y_vel_max`  | Minimum and maximum vertical velocity of emitted particles.                                             |
| `count_min`, `count_max`  | Minimum and maximum number of particles emitted per emission.                                           |
| `lifetime_min`, `lifetime_max` | Minimum and maximum lifetime of emitted particles.                                                      |
| `is_trail`                | If the particle emitter functions as a trail (1 for true).                                              |
| `trail_gap`               | Spacing between particles in a trail.                                                                   |
| `fade_based_on_lifetime`  | If particles fade out as their lifetime decreases (1 for true).                                         |
| `emit_cosmetic_particles` | If the emitter should create cosmetic particles (1 for true).                                           |
| `is_emitting`             | If the emitter is currently active (1 for true).                                                        |

### LightComponent

Adds a light source to the projectile.

| Attribute | Description                                                                                             |
| :-------- | :------------------------------------------------------------------------------------------------------ |
| `_enabled`| Enables or disables the component (1 for enabled).                                                      |
| `radius`  | The radius of the light emitted.                                                                        |
| `r`, `g`, `b` | RGB values for the light color.                                                                         |

### AudioComponent

Handles sound effects associated with the projectile.

| Attribute   | Description                                                                                             |
| :---------- | :------------------------------------------------------------------------------------------------------ |
| `file`      | Path to the audio bank file.                                                                            |
| `event_root`| The root event name for the projectile's sounds.                                                        |

### VariableStorageComponent

Used to store custom variables, such as the projectile's own file path.

| Attribute   | Description                                                                                             |
| :---------- | :------------------------------------------------------------------------------------------------------ |
| `name`      | The name of the variable.                                                                               |
| `value_string`| The string value of the variable.                                                                       |
