# Failed Alchemist Entity

This document details the configuration of the "Failed Alchemist" entity in Noita, focusing on attributes relevant to AI-driven modding.

## Core Attributes

The Failed Alchemist is a basic enemy with melee attack capabilities and some defensive traits.

### `Base` Component

This component inherits from `base_enemy_basic.xml`, providing fundamental enemy behaviors.

#### `AnimalAIComponent`

Controls the AI and combat behavior of the Failed Alchemist.

| Attribute                      | Value   | Description                                                                 |
| :----------------------------- | :------ | :-------------------------------------------------------------------------- |
| `escape_if_damaged_probability`| `0`     | Cannot escape when damaged.                                                 |
| `hide_from_prey`               | `1`     | Will attempt to hide from prey.                                             |
| `hide_from_prey_target_distance`| `200`   | Distance at which it will attempt to hide.                                  |
| `hide_from_prey_time`          | `480`   | Duration (frames) it will attempt to hide.                                  |
| `attack_melee_damage_min`      | `2.0`   | Minimum damage dealt by melee attacks.                                      |
| `attack_melee_damage_max`      | `3.6`   | Maximum damage dealt by melee attacks.                                      |
| `attack_melee_max_distance`    | `20`    | Maximum distance for melee attacks.                                         |
| `needs_food`                   | `0`     | Does not require food.                                                      |
| `can_fly`                      | `0`     | Cannot fly.                                                                 |
| `aggressiveness_min`           | `1`     | Minimum aggressiveness level.                                               |
| `aggressiveness_max`           | `80`    | Maximum aggressiveness level.                                               |
| `attack_melee_action_frame`    | `6`     | Frame at which the melee attack action is initiated.                        |
| `attack_melee_finish_enabled`  | `1`     | Enables a special effect upon finishing a melee attack.                     |
| `attack_melee_finish_action_frame`| `6`     | Frame at which the melee attack finish effect is initiated.                 |

Details the explosion effect triggered at the end of a melee attack.

| Attribute                 | Value                                     | Description                                                              |
| :------------------------ | :---------------------------------------- | :----------------------------------------------------------------------- |

| `explosion_radius`        | `4`                                       | Radius of the explosion.                                                 |

| `explosion_sprite_lifetime`| `1`                                       | Lifetime of the explosion sprite.                                        |
| `hole_enabled`            | `1`                                       | Enables the creation of holes in terrain.                                |
