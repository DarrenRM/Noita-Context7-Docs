# Constellation Noita Logo Entity

This entity defines the visual representation of the Noita logo during the game's introduction sequence, utilizing a particle emitter to create a dynamic, glowing effect.

## Entity Structure

The entity inherits its base properties from `data/entities/intro/constellation_base.xml`.

## Key Components

### ParticleEmitterComponent

This component is responsible for generating the visual effects for the logo.
