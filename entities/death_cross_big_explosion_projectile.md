# Death Cross Big Explosion Projectile

This entity defines a projectile that triggers a large, non-damaging explosion upon its lifetime expiring. It's designed to be a visual effect or a trigger for other game events rather than a direct damage source.

## Key Components and Attributes

### ProjectileComponent

This component governs the projectile's behavior.
