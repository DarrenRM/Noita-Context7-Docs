# Large Explosion Particle Entity

This entity defines the components that make up a large explosion effect in Noita. It primarily functions by loading other pre-defined particle entities to create a layered and dynamic visual.

## Key Components

### LoadEntitiesComponent

This component is responsible for spawning other entities. In this case, it's used to instantiate various explosion-related particle effects.
