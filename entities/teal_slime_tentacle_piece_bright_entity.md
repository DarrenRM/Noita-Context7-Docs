# Teal Slime Tentacle Piece (Bright) Entity

This entity defines a single bright piece of a teal slime tentacle, used in Noita's physics and visual systems.

## Sprite Component

The `SpriteComponent` defines the visual representation of the tentacle piece.

### Key Attributes:

*   **`image_file`**: Specifies the texture file for the sprite.
    *   `data/entities/verlet_chains/tealslime_tentacle/tealslime_tentacle_piece_bright.png`
*   **`offset_x`**: Horizontal offset of the sprite.
    *   `0`
*   **`offset_y`**: Vertical offset of the sprite.
    *   `1`
*   **`update_transform`**: Controls whether the sprite's transform is updated by physics.
    *   `0` (Disabled)
*   **`update_transform_rotation`**: Controls whether the sprite's rotation is updated by physics.
    *   `0` (Disabled)
