# Radioactive Liquid to Acid Conversion

This entity defines a magical effect that converts various forms of radioactive liquid and radioactive clouds into acid. It's designed to be a persistent, area-of-effect conversion.

## Key Components

### `MagicConvertMaterialComponent`

This component handles the core conversion logic. The entity utilizes multiple instances of this component to target different types of radioactive materials.
