# Pink Big Explosive Orb Projectile

This document details the configuration of the "Pink Big Explosive Orb" projectile entity in Noita, focusing on attributes relevant for AI-assisted modding.

## Core Projectile Properties

The `ProjectileComponent` defines the fundamental behavior of the orb.

| Attribute             | Value   | Description                                                                 |
| :-------------------- | :------ | :-------------------------------------------------------------------------- |
| `lob_min`             | `0.8`   | Minimum lob angle for projectile trajectory.                                |
| `lob_max`             | `1.0`   | Maximum lob angle for projectile trajectory.                                |
| `speed_min`           | `100`   | Minimum projectile speed.                                                   |
| `speed_max`           | `120`   | Maximum projectile speed.                                                   |
| `die_on_low_velocity` | `0`     | Projectile does not die when its velocity drops too low.                    |
| `on_death_explode`    | `1`     | The projectile explodes upon death.                                         |
| `on_lifetime_out_explode` | `1` | The projectile explodes when its lifetime expires.                          |
| `explosion_dont_damage_shooter` | `1` | The explosion does not damage the entity that fired the projectile.       |
| `damage`              | `0.8`   | Base damage dealt by the projectile itself (before explosion).              |
| `on_collision_die`    | `1`     | The projectile dies upon collision with another entity.                     |
| `lifetime`            | `100`   | Duration in frames before the projectile explodes if `on_lifetime_out_explode` is set. |
| `knockback_force`     | `2.0`   | The force applied as knockback upon collision or explosion.                 |

| Attribute                 | Value        | Description                                                                 |
| :------------------------ | :----------- | :-------------------------------------------------------------------------- |

| `explosion_radius`        | `38`         | The radius of the explosion's effect.                                       |
| `explosion_sprite`        | `data/particles/explosion_032_pink.xml` | Path to the sprite used for the explosion visual.                       |
| `load_this_entity`        | `data/entities/particles/particle_explosion/main_swirly_pink.xml` | Entity to load for more complex explosion effects.                        |
| `hole_destroy_liquid`     | `1`          | The explosion can destroy liquids.                                          |
| `hole_enabled`            | `1`          | The explosion creates holes in terrain.                                     |
| `damage`                  | `1.0`        | Damage dealt by the explosion itself.                                       |
| `ray_energy`              | `100000`     | Energy value for raycasting effects from the explosion.                     |
| `physics_explosion_power.min` | `0.4`      | Minimum power of the physics-based explosion force.                         |
| `physics_explosion_power.max` | `0.6`      | Maximum power of the physics-based explosion force.                         |
| `sparks_enabled`          | `1`          | Sparks are generated by the explosion.                                      |
| `sparks_count_min`        | `60`         | Minimum number of sparks to emit.                                           |
| `sparks_count_max`        | `80`         | Maximum number of sparks to emit.                                           |
| `spark_material`          | `plasma_fading_pink` | The material used for the emitted sparks.                                   |
| `light_r`, `light_g`, `light_b` | `150`, `35`, `120` | RGB values for the light emitted by the explosion.                        |
| `audio_event_name`        | `explosions/magic_rocket_mid` | The audio event triggered by the explosion.                               |

    *   `gravity.y`: `10`
    *   `is_emitting`: `1`
    *   `randomize_rotation.min/max`: `-0.3415` to `0.3415`
    *   `randomize_angular_velocity.min/max`: `-1.3415` to `1.3415`

*   **`SpriteParticleEmitterComponent` (Secondary Fading):**
    *   `sprite_file`: `data/projectiles_gfx/orb_pink_big.xml`

    *   `scale_velocity.x/y`: `0.2` (particles grow)
    *   `entity_velocity_multiplier`: `-0.5` (particles move against projectile direction)

*   **`ParticleEmitterComponent` (Core Particles):**
    *   `emitted_material_name`: `plasma_fading_pink`

## Homing Behavior

The `HomingComponent` enables the projectile to track targets.

| Attribute             | Value   | Description                                     |
| :-------------------- | :------ | :---------------------------------------------- |
| `target_tag`          | `prey`  | The tag of entities to target.                  |
| `homing_targeting_coeff`| `8`     | Strength of the homing behavior.                |
| `detect_distance`     | `350`   | Maximum distance at which targets can be detected. |
| `homing_velocity_multiplier` | `1.0` | Multiplier for the projectile's velocity when homing. |

## Other Components

*   **`Base Component`**: Inherits base projectile properties, including `gravity_y="0"` and `air_friction="0"`, indicating no gravity or air resistance for the base projectile.
*   **`LightComponent`**: Defines a light source with `radius="150"` and RGB values `r="150"`, `g="35"`, `b="120"`.
*   **`AudioComponent`**: Plays a sound event upon projectile creation.
*   **`AudioLoopComponent`**: Plays a looping sound event while the projectile is active.
*   **`VariableStorageComponent`**: Stores the projectile's own file path in the `projectile_file` variable.
