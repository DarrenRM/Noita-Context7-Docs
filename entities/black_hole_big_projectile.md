# Black Hole Big Projectile

This document details the configuration of the "Black Hole Big" projectile entity in Noita, focusing on its key attributes for AI-assisted modding.

## Core Functionality

The `black_hole_big.xml` entity defines a powerful projectile that creates a localized gravitational effect, pulling in nearby entities and particles. It's designed to be a significant area-of-effect weapon.

## Key Components and Attributes

### ProjectileComponent

This is the primary component defining the projectile's behavior.

| Attribute                 | Value        | Description                                                                 |
| :------------------------ | :----------- | :-------------------------------------------------------------------------- |
| `lob_min`, `lob_max`      | `0.8`, `1.0` | Controls the lobbing behavior (how much it arcs).                           |
| `speed_min`, `speed_max`  | `0`, `0`     | Projectile starts with no initial velocity, relying on other forces.        |
| `on_death_explode`        | `1`          | The projectile explodes when its lifetime ends or it collides.              |
| `on_lifetime_out_explode` | `1`          | Explicitly states it explodes when lifetime expires.                        |
| `explosion_dont_damage_shooter` | `1`      | Prevents the projectile's explosion from harming the player who fired it.   |
| `on_collision_die`        | `0`          | The projectile does not die immediately upon collision; it explodes instead. |
| `lifetime`                | `500`        | Duration in frames before the projectile explodes.                          |
| `damage`                  | `0`          | Base damage of the projectile itself (the explosion handles damage).        |
| `damage_every_x_frames`   | `5`          | How often the damage tick occurs if applicable (less relevant here).        |
| `knockback_force`         | `0.1`        | The force applied to entities it hits.                                      |
| `penetrate_entities`      | `1`          | Allows the projectile to pass through multiple entities.                    |
| `camera_shake_when_shot`  | `0.4`        | Amount of camera shake when the projectile is fired.                        |
| `collide_with_entities`   | `0`          | The projectile itself does not collide with entities (its effect does).     |
| `friendly_fire`           | `1`          | The projectile can affect friendly entities.                                |
| `shoot_light_flash_radius`| `250`        | Radius of the light flash when fired.                                       |

#### `config_explosion`

Nested within `ProjectileComponent`, this defines the explosion properties.

| Attribute               | Value                                     | Description                                                              |
| :---------------------- | :---------------------------------------- | :----------------------------------------------------------------------- |
| `damage`                | `0`                                       | Base damage of the explosion.                                            |
| `camera_shake`          | `0`                                       | Camera shake caused by the explosion.                                    |
| `explosion_radius`      | `1`                                       | The radius of the explosion effect.                                      |
| `explosion_sprite`      | `data/particles/black_hole_big_circle_out.xml` | Visual effect for the explosion.                                         |
| `hole_enabled`          | `1`                                       | Enables the "black hole" effect.                                         |
| `damage_mortals`        | `1`                                       | The explosion deals damage to mortal entities.                           |
| `physics_throw_enabled` | `0`                                       | Disables physics-based throwing of entities by the explosion.            |
| `sparks_count_min`, `max` | `4`, `4`                                  | Number of sparks generated by the explosion.                             |
| `spark_material`        | `plasma_fading`                           | Material of the sparks.                                                  |

### SpriteComponent

Defines the visual appearance of the projectile.

| Attribute      | Value                                | Description                                  |
| :------------- | :----------------------------------- | :------------------------------------------- |
| `alpha`        | `0.1`                                | Transparency of the sprite.                  |
| `image_file`   | `data/projectiles_gfx/black_hole_big_circle.xml` | The sprite image used.                       |
| `emissive`     | `1`                                  | Sprite emits light.                          |
| `additive`     | `1`                                  | Sprite uses additive blending.               |

### ParticleEmitterComponent

Responsible for generating cosmetic particles around the projectile.

| Attribute               | Value      | Description                                                              |
| :---------------------- | :--------- | :----------------------------------------------------------------------- |
| `emitted_material_name` | `plasma_fading_pink` | The material of the particles emitted.                                   |
| `x_vel_min`, `max`      | `-8`, `8`  | Minimum and maximum horizontal velocity of emitted particles.            |
| `y_vel_min`, `max`      | `-8`, `8`  | Minimum and maximum vertical velocity of emitted particles.              |
| `count_min`, `max`      | `4`, `4`   | Number of particles emitted per burst.                                   |
| `attractor_force`       | `32`       | Force that pulls particles towards the projectile.                       |
| `lifetime_min`, `max`   | `0.1`, `1.5` | Minimum and maximum lifetime of emitted particles.                       |
| `emit_cosmetic_particles` | `1`      | Ensures only cosmetic particles are emitted.                             |
| `emission_interval_min_frames`, `max` | `2`, `2` | How often particles are emitted (in frames).                             |

### LightComponent

Adds a light source to the projectile.

| Attribute | Value   | Description                               |
| :-------- | :------ | :---------------------------------------- |
| `r`, `g`, `b` | `255`, `40`, `255` | RGB color of the light.                   |
| `radius`  | `128`   | Radius of the light.                      |

### BlackHoleComponent

This component specifically enables and configures the black hole gravity effect.

| Attribute             | Value   | Description                                                              |
| :-------------------- | :------ | :----------------------------------------------------------------------- |
| `particle_attractor_force` | `6`     | The strength of the gravitational pull on particles.                     |
| `damage_probability`  | `0.25`  | The chance that entities within the radius take damage.                  |
| `radius`              | `1`     | The radius of the black hole effect.                                     |

### LuaComponent

Links to external Lua scripts that provide more complex behavior.

| Attribute           | Value                                     | Description                                                              |
| :------------------ | :---------------------------------------- | :----------------------------------------------------------------------- |
| `script_source_file`| `data/scripts/projectiles/black_hole_big.lua` | Primary script for the projectile's core logic.                          |
| `execute_every_n_frame` | `3`                                       | How often the primary script is executed.                                |
| `script_source_file`| `data/scripts/projectiles/black_hole_gravity.lua` | Script responsible for handling the gravity effect.                      |
| `execute_every_n_frame` | `1`                                       | How often the gravity script is executed.                                |

### AudioComponent & AudioLoopComponent

Handle the sound effects associated with the projectile.

| Attribute     | Value                                  | Description                                                              |
| :------------ | :------------------------------------- | :----------------------------------------------------------------------- |
| `file`        | `data/audio/Desktop/projectiles.bank`  | The audio bank containing the sound events.                              |
| `event_root`  | `player_projectiles/black_hole_big`    | The root event name for the projectile's sounds.                         |
| `event_name`  | `player_projectiles/black_hole_big/loop` | The specific event name for the looping sound.                           |
| `auto_play`   | `1`                                    | The loop sound starts automatically.                                     |

### Entity (Nested)

This nested entity contains components that affect the environment around the projectile's impact point.

#### LooseGroundComponent

| Attribute         | Value        | Description                                                              |
| :---------------- | :----------- | :----------------------------------------------------------------------- |
| `max_distance`    | `80`         | Maximum distance from the projectile where loose ground can be affected. |
| `probability`     | `0.2`        | Chance that loose ground will be affected.                               |
| `chunk_probability` | `0.03`       | Chance that chunks of loose ground are created.                          |

## Summary

The "Black Hole Big" projectile is a complex entity that combines visual effects, particle emission, light sources, and Lua scripting to create a powerful gravitational anomaly. Its key features include its ability to pull in entities and particles, its visual representation, and its sound design. Modders can adjust parameters within these components to alter its behavior, appearance, and impact.
