# Boss Centipede Tail Verlet 1

This entity represents a segment of the boss centipede's tail, specifically a "verlet" state.

## Sprite Component

This component defines the visual representation of the tail segment.

### Key Attributes:

*   **`image_file`**: `data/entities/animals/boss_centipede/tail/tail_1.xml` - Specifies the sprite image to be used.
*   **`offset_x`**: `2` - Horizontal offset of the sprite.
*   **`offset_y`**: `15` - Vertical offset of the sprite.
*   **`update_transform`**: `0` - Disables automatic transform updates.
*   **`update_transform_rotation`**: `0` - Disables automatic rotation updates.
*   **`z_index`**: `1.2` - Determines the rendering order (higher values are drawn on top).
