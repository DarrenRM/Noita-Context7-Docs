# Orb Particle Emitter Base

This entity defines a base configuration for particle emitters, commonly used for visual effects associated with orbs in Noita. It specifies how particles are generated, their properties, and how they interact with the environment.

## Key Components

### ParticleEmitterComponent

This is the core component responsible for generating and managing particles.
