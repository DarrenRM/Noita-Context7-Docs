# Wraith Cape Part 4

This document describes a specific part of the Wraith's cape in Noita, focusing on its visual representation.

## SpriteComponent

This component defines the visual appearance of the entity.

### Key Attributes:

*   **image\_file**: `data/entities/verlet_chains/wraith_cape/wraith/parts/cape_4.xml` - Specifies the image file used for the sprite.
*   **offset\_x**: `1` - Horizontal offset of the sprite.
*   **offset\_y**: `12` - Vertical offset of the sprite.
*   **z\_index**: `1.5` - Determines the drawing order of the sprite. Higher values are drawn on top.
*   **update\_transform**: `0` - Disables automatic transformation updates for the sprite.
*   **update\_transform\_rotation**: `0` - Disables automatic rotation updates for the sprite.
