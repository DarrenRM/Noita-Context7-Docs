# Root Verlet 2 Entity

This document describes the `root_verlet_2.xml` entity, a component of the rootverlet chain in Noita.

## SpriteComponent

This component defines the visual representation of the entity.

### Key Attributes:

*   **image_file**: `data/entities/verlet_chains/root/root_parts/root_2.xml` - Specifies the sprite image to be used.
*   **offset_x**: `0` - Horizontal offset for the sprite.
*   **offset_y**: `3` - Vertical offset for the sprite.
*   **update_transform**: `0` - Disables automatic transform updates for the sprite.
*   **update_transform_rotation**: `0` - Disables automatic rotation updates for the sprite.
