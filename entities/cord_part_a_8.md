# Cord Part A 8

This entity defines a single segment of a cord in Noita, specifically designed for Verlet chain physics.

## SpriteComponent

This component handles the visual representation of the cord segment.

### Key Attributes:

*   **image\_file**: `data/entities/verlet_chains/cords/cord_parts/cord_a_8.xml` - Specifies the image file used for this cord segment.
*   **offset\_x**: `0` - Horizontal offset for the sprite.
*   **offset\_y**: `1.5` - Vertical offset for the sprite.
*   **update\_transform**: `0` - Disables automatic transform updates for the sprite.
*   **update\_transform\_rotation**: `0` - Disables automatic rotation updates for the sprite.
