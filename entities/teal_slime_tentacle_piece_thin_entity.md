# Teal Slime Tentacle Piece (Thin) Entity

This document describes the `tealslime_tentacle_piece_thin.xml` entity, which represents a thin segment of a teal slime tentacle in Noita.

## Sprite Component

The `SpriteComponent` defines the visual representation of the tentacle piece.

### Key Attributes:

*   **`image_file`**: Specifies the texture file used for the sprite.
    *   `data/entities/verlet_chains/tealslime_tentacle/tealslime_tentacle_piece_thin.png`
*   **`offset_x`**: Horizontal offset of the sprite.
    *   `0`
*   **`offset_y`**: Vertical offset of the sprite.
    *   `0.5`
*   **`update_transform`**: Determines if the sprite's transform (position, scale) is updated.
    *   `0` (Disabled)
*   **`update_transform_rotation`**: Determines if the sprite's rotation is updated.
    *   `0` (Disabled)
