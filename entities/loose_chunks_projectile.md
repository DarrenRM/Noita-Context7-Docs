# Loose Chunks Projectile

This entity defines the behavior of loose ground chunks that can be spawned as projectiles.

## Entity Definition

The core of this entity is the `LooseGroundComponent`, which dictates how these chunks behave.

### LooseGroundComponent

This component controls the spawning and physics of loose ground chunks.
