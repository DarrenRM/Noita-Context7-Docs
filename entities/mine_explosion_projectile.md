# Mine Explosion Projectile

This document details the configuration of a mine explosion projectile entity in Noita, designed for AI-assisted modding.

## Entity Definition

The core of this entity is a projectile with specific explosion behaviors.

```xml
<Entity 
  name="$projectile_default" 
   >
  <Base file="data/entities/base_projectile.xml" >
    <VelocityComponent>
    </VelocityComponent> 
  </Base>

  <ProjectileComponent 
    _enabled="1" 
    lob_min="0.8"
    lob_max="1.0"
    die_on_low_velocity="1"
    on_death_explode="1"
    on_death_gfx_leave_sprite="0" 
    on_lifetime_out_explode="1"
    explosion_dont_damage_shooter="0"
    damage="0.25"
    on_collision_die="1"
    lifetime="0" >
    
    <!-- Explosion Configuration -->
    <config_explosion
      never_cache="1" 
      camera_shake="25.0" 
      explosion_radius="30" 
      explosion_sprite="data/particles/explosion_040_poof.xml"
      load_this_entity="data/entities/particles/particle_explosion/main.xml"
      explosion_sprite_lifetime="0" 
      create_cell_probability="80" 
      create_cell_material="fire" 
      ray_energy="170000"
      hole_destroy_liquid="0" 
      damage="4"
      hole_enabled="1" 
      hole_image="data/temp/explosion_hole.png" 
      particle_effect="1" 
      damage_mortals="1" 
      physics_explosion_power.min="2.3"
      physics_explosion_power.max="3.6" 
      physics_throw_enabled="1" 
      shake_vegetation="1" 
      sparks_enabled="1" 
      sparks_count_max="30" 
      sparks_count_min="10"
      light_fade_time="0.8" 
      stains_enabled="1" 
      stains_image="data/temp/explosion_stain.png" >
    </config_explosion>
  </ProjectileComponent>
</Entity>
```

### Key ProjectileComponent Attributes

*   **`lob_min` / `lob_max`**: Controls the minimum and maximum lob angle of the projectile.
*   **`die_on_low_velocity`**: If true, the projectile will explode when its velocity drops below a certain threshold.
*   **`on_death_explode`**: Triggers an explosion upon the projectile's death.
*   **`on_lifetime_out_explode`**: Triggers an explosion when the projectile's lifetime expires.
*   **`explosion_dont_damage_shooter`**: If true, the explosion will not damage the entity that fired it.
*   **`damage`**: The base damage of the projectile itself before exploding.
*   **`on_collision_die`**: If true, the projectile will die and explode upon colliding with something.
*   **`lifetime`**: The duration the projectile exists before expiring (set to 0 for immediate explosion on other triggers).

### Key `config_explosion` Attributes

*   **`camera_shake`**: The intensity of camera shake caused by the explosion.
*   **`explosion_radius`**: The radius of the explosion's effect.
*   **`explosion_sprite`**: The visual sprite used for the explosion effect.
*   **`load_this_entity`**: Specifies another entity to load and spawn as part of the explosion (e.g., particle effects).
*   **`create_cell_probability`**: The chance (in percent) that the explosion will create new material cells.
*   **`create_cell_material`**: The material to create when cells are generated by the explosion.
*   **`ray_energy`**: The energy value associated with the explosion's raycast effects.
*   **`damage`**: The damage dealt by the explosion itself.
*   **`hole_enabled`**: If true, the explosion will create holes in surfaces.
*   **`hole_image`**: The texture used for the explosion holes.
*   **`particle_effect`**: If true, particle effects will be generated by the explosion.
*   **`damage_mortals`**: If true, the explosion will damage living entities.
*   **`physics_explosion_power.min` / `physics_explosion_power.max`**: The minimum and maximum force applied to physics objects by the explosion.
*   **`sparks_enabled`**: If true, sparks will be generated by the explosion.
*   **`sparks_count_min` / `sparks_count_max`**: The range for the number of sparks generated.
*   **`stains_enabled`**: If true, explosion stains will be left on surfaces.
*   **`stains_image`**: The texture used for the explosion stains.
