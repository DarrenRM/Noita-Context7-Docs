# Swarm Fly Projectile

This document details the `swarm_fly.xml` entity, representing a projectile in Noita. It's designed to be a swarm-like projectile with homing capabilities and a unique death explosion.

## Core Components

### Base Projectile (`BaseComponent`)

Defines fundamental projectile physics.

| Attribute        | Value | Description                                     |
| :--------------- | :---- | :---------------------------------------------- |
| `gravity_y`      | `50`  | Downward acceleration due to gravity.           |
| `air_friction`   | `2`   | Resistance to movement in the air.              |
| `mass`           | `0.03`| The projectile's mass, affecting its momentum. |

### Homing Component (`HomingComponent`)

Enables the projectile to track and move towards a target.

| Attribute                | Value | Description                                                              |
| :----------------------- | :---- | :----------------------------------------------------------------------- |
| `target_who_shot`        | `1`   | The projectile will home towards the entity that shot it.                |
| `homing_targeting_coeff` | `10.0`| Strength of the homing behavior. Higher values mean more aggressive tracking. |
| `detect_distance`        | `300` | The maximum distance at which the projectile can detect its target.      |
