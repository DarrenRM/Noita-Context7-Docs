# Test Physics Heavy Prop

This entity is a simple test prop designed to demonstrate physics interactions. It's intended for modders to experiment with physics properties.

## Entity Structure

The entity is composed of the following core components:

*   **`PhysicsBodyComponent`**: Defines the physical properties of the entity.
*   **`PhysicsImageShapeComponent`**: Defines the visual shape and material of the physics body.
*   **`LuaComponent`**: Executes a Lua script associated with the entity.

## Key Components and Attributes

### `PhysicsBodyComponent`

This component governs how the entity interacts with the physics engine.
