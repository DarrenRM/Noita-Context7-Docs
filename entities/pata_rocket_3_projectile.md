# Pata Rocket 3 Projectile

This document details the configuration of the "Pata Rocket 3" projectile entity in Noita, focusing on its behavior, visual effects, and sound.

## Core Projectile Properties

The `ProjectileComponent` defines the fundamental behavior of the projectile.

### Key Attributes:

| Attribute                 | Value   | Description                                                                 |
| :------------------------ | :------ | :-------------------------------------------------------------------------- |
| `lob_min`, `lob_max`      | `0.8`, `1.0` | Controls the initial upward angle (lob) of the projectile.                  |
| `speed_min`, `speed_max`  | `70`, `100` | Defines the range of initial projectile speeds.                             |
| `on_death_explode`        | `1`     | The projectile will explode upon death.                                     |
| `on_lifetime_out_explode` | `1`     | The projectile will explode when its lifetime expires.                      |
| `on_collision_die`        | `1`     | The projectile will die upon colliding with something.                      |
| `lifetime`                | `40`    | The base duration (in frames) the projectile exists before expiring.        |
| `lifetime_randomness`     | `20`    | Adds random variation to the projectile's lifetime.                         |
| `damage`                  | `5`     | The base damage dealt by the projectile.                                    |
| `knockback_force`         | `4.0`   | The force applied to entities when the projectile hits or explodes.         |
| `camera_shake_when_shot`  | `15.0`  | The intensity of camera shake when this projectile is fired.                |

### Explosion Configuration (`config_explosion`)

This nested element defines the properties of the explosion that occurs when the projectile dies.

| Attribute                 | Value        | Description                                                                 |
| :------------------------ | :----------- | :-------------------------------------------------------------------------- |
| `camera_shake`            | `90`         | Intensity of camera shake during the explosion.                             |
| `explosion_radius`        | `30`         | The radius of the explosion's effect.                                       |
| `explosion_sprite`        | `...slime.xml` | The sprite used for the initial explosion visual.                           |
| `load_this_entity`        | `...giga.xml` | The entity to spawn upon explosion (e.g., a larger particle effect).        |
| `ray_energy`              | `6200000`    | The energy of the destructive rays emitted by the explosion.                |
| `damage`                  | `5.2`        | The damage dealt by the explosion itself.                                   |
| `physics_explosion_power` | `1.0` - `1.4`| The force of the physics-based explosion.                                   |
| `sparks_count_min/max`    | `60`, `90`   | The range of sparks generated by the explosion.                             |
| `spark_material`          | `spark_green`| The material type of the sparks.                                            |
| `audio_event_name`        | `explosions/...` | The sound event triggered by the explosion.                                 |
