# Boss Wizard Summon Projectile

This entity defines the projectile used by the Boss Wizard to summon other entities. It's a projectile that explodes on impact or after a set lifetime, spawning a specific entity.

## Key Components

### ProjectileComponent

This is the core component defining the projectile's behavior.

| Attribute              | Description                                                                                                |
| :--------------------- | :--------------------------------------------------------------------------------------------------------- |
| `speed_min`            | Minimum projectile speed.                                                                                  |
| `speed_max`            | Maximum projectile speed.                                                                                  |
| `die_on_low_velocity`  | If `1`, the projectile dies when its velocity drops below a certain threshold.                             |
| `on_death_explode`     | If `1`, the projectile explodes when it dies.                                                              |
| `on_lifetime_out_explode` | If `1`, the projectile explodes when its lifetime expires.                                                 |
| `direction_random_rad` | Randomness applied to the projectile's direction in radians.                                               |
| `on_collision_die`     | If `1`, the projectile dies upon collision.                                                                |
| `damage`               | Damage dealt by the projectile itself (set to `0` here, damage comes from explosion).                      |
| `lifetime`             | Duration in frames before the projectile expires.                                                          |
| `on_collision_spawn_entity` | If `1`, an entity is spawned upon collision.                                                               |
| `spawn_entity`         | Path to the entity XML file to spawn upon collision. Here, it's `data/entities/animals/boss_wizard/spawner.xml`. |

#### `config_explosion` (Nested within `ProjectileComponent`)

Defines the parameters of the explosion that occurs when the projectile dies.

| Attribute              | Description                                                                                                |
| :--------------------- | :--------------------------------------------------------------------------------------------------------- |
| `damage`               | Damage dealt by the explosion.                                                                             |
| `camera_shake`         | Intensity of camera shake caused by the explosion.                                                         |
| `explosion_radius`     | The radius of the explosion.                                                                               |
| `explosion_sprite`     | Path to the sprite XML for the explosion visual effect.                                                    |
| `explosion_sprite_lifetime` | Duration of the explosion sprite.                                                                          |
| `hole_enabled`         | If `1`, the explosion creates a hole in the terrain.                                                       |
| `hole_image`           | Path to the image used for the terrain hole.                                                               |
| `particle_effect`      | If `1`, particle effects are generated by the explosion.                                                   |
| `damage_mortals`       | If `1`, the explosion damages mortal entities.                                                             |
| `physics_explosion_power` | Controls the physics force applied by the explosion.                                                       |
| `shake_vegetation`     | If `1`, the explosion shakes vegetation.                                                                   |
| `sparks_count_min`/`max` | Minimum and maximum number of sparks generated.                                                            |
| `stains_enabled`       | If `1`, the explosion leaves stains on the terrain.                                                        |
| `stains_radius`        | The radius of the stains left by the explosion.                                                            |

### SpriteComponent

Defines the visual appearance of the projectile.

| Attribute      | Description                                                                                                |
| :------------- | :--------------------------------------------------------------------------------------------------------- |
| `image_file`   | Path to the sprite image file. Here, it's a shine effect.                                                  |
| `additive`     | If `1`, uses additive blending for the sprite.                                                             |
| `emissive`     | If `1`, the sprite is emissive (glows).                                                                    |

### ParticleEmitterComponent

Adds a particle effect to the projectile.

| Attribute              | Description                                                                                                |
| :--------------------- | :--------------------------------------------------------------------------------------------------------- |
| `emitted_material_name` | The material name of the particles to emit (e.g., `spark_yellow`).                                       |
| `lifetime_min`/`max`   | Minimum and maximum lifetime of emitted particles.                                                         |
| `count_min`/`max`      | Minimum and maximum number of particles emitted per emission.                                              |
| `area_circle_radius.max` | Maximum radius of the emission area.                                                                       |
| `airflow_force`        | Force applied to particles by airflow.                                                                     |
| `emission_interval_min_frames`/`max_frames` | Controls how often particles are emitted.                                                              |
| `emit_cosmetic_particles` | If `1`, emits cosmetic particles (particles that don't affect gameplay).                                   |
| `is_emitting`          | If `1`, the particle emitter is active.                                                                    |

### VelocityComponent

Defines the projectile's velocity properties.

| Attribute   | Description                                                                                                |
| :---------- | :--------------------------------------------------------------------------------------------------------- |
| `gravity_y` | The gravitational force applied to the projectile on the Y-axis. Set to `0` for no gravity.                |

### AudioComponent

Defines the sound played by the projectile.

| Attribute   | Description                                                                                                |
| :---------- | :--------------------------------------------------------------------------------------------------------- |
| `file`      | Path to the audio bank file.                                                                               |
| `event_root` | The root event name for the sound effect.                                                                  |

### VariableStorageComponent

Stores variables associated with the entity.

| Attribute    | Description                                                                                                |
| :----------- | :--------------------------------------------------------------------------------------------------------- |
| `name`       | The name of the variable.                                                                                  |
| `value_string` | The string value of the variable. Here, it stores the `projectile_file` which is the same as `spawn_entity`. |
