# Iceskull Explosion Projectile

This entity defines the behavior of the projectile that is spawned when an "Iceskull" explodes. It's primarily a wrapper for a Lua script that handles the actual explosion logic.

## Key Components

### ProjectileComponent

This component defines the fundamental properties of the projectile itself.
