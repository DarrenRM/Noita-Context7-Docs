# Orange Firework Particle

This entity defines the visual components of an orange firework explosion. It primarily uses `LoadEntitiesComponent` to spawn other particle entities, creating a layered explosion effect.

## Key Components

### LoadEntitiesComponent

This component is responsible for loading and spawning other entities. For this firework, it's used to create various visual effects like flares, fog of war holes, smoke trails, sparks, and smoke pillars.
