# Physics Fungus (Acid)

This document details the `physics_fungus_acid.xml` entity, a physics-enabled prop in Noita that functions as a source of acid.

## Core Components

### PhysicsBody2Component
Manages the physical properties of the fungus.

| Attribute        | Description                                     |
| :--------------- | :---------------------------------------------- |
| `is_static`      | `0` - The fungus is not fixed in place.         |
| `allow_sleep`    | `1` - Allows the physics body to sleep when idle. |
| `linear_damping` | `0.5` - Slows down linear movement.             |
| `init_offset_y`  | `40` - Initial vertical offset.                 |

### PhysicsImageShapeComponent
Defines the visual representation and physical shape of the fungus. Multiple components are used to construct the fungus from its cap, stalk, and foot.

| Attribute    | Description                                     |
| :----------- | :---------------------------------------------- |
| `image_file` | Path to the sprite for the specific part.       |
| `material`   | `fungus_loose_green` - The base material.       |
| `body_id`    | Unique identifier for each physics body part.   |
| `offset_x`   | Horizontal offset for the part.                 |
| `offset_y`   | Vertical offset for the part.                   |

### MaterialInventoryComponent
Handles the material contained within the fungus.

| Attribute             | Description                                                              |
| :-------------------- | :----------------------------------------------------------------------- |
| `on_death_spill`      | `1` - Spills its contents when destroyed.                                |
| `leak_on_damage_percent` | `1` - Leaks material when damaged.                                       |
| `leak_pressure_min`   | `0.5` - Minimum pressure for leaking.                                    |
| `leak_pressure_max`   | `1.0` - Maximum pressure for leaking.                                    |
| `kill_when_empty`     | `1` - The fungus entity is destroyed when its material is depleted.      |
| `count_per_material_type` | Contains a list of materials and their quantities.                       |
| `Material material="acid" count="900"` | The fungus contains 900 units of acid.                                   |

### DamageModelComponent
Defines how the fungus takes damage and its reactions.

| Attribute              | Description                                                              |
| :--------------------- | :----------------------------------------------------------------------- |
| `hp`                   | `0.6` - Very low health, easily destroyed.                               |
| `ragdoll_material`     | `fungus_loose_green` - Material used for ragdoll effects.                |
| `blood_sprite_material`| `acid` - The "blood" sprayed when damaged is acid.                       |
| `damage_multipliers`   | Modifiers for different damage types.                                    |
| `fire="40.0"`          | Takes significantly more damage from fire.                               |

### ExplodeOnDamageComponent
Controls the explosion behavior when the fungus is damaged or destroyed.

| Attribute                     | Description                                                              |
| :---------------------------- | :----------------------------------------------------------------------- |
| `explode_on_death_percent`    | `1` - Explodes upon death.                                               |
| `physics_body_destruction_required` | `0.25` - Explosion is triggered when 25% of its physics body is destroyed. |
| `config_explosion`            | Contains detailed settings for the explosion.                            |
| `explosion_radius`            | `30` - The radius of the explosion.                                      |
| `explosion_sprite`            | `data/particles/explosion_025_fuel.xml` - Visual effect for the explosion. |
| `create_cell_material="fire"` | Creates fire cells upon explosion.                                       |
| `spark_material="acid"`       | Sparks generated by the explosion are acid.                              |
| `audio_event_name="explosions/slime"` | Sound effect played during the explosion.                                |

### LightComponent
Adds a light source to the entity.

| Attribute | Description                               |
| :-------- | :---------------------------------------- |
| `radius`  | `80` - The radius of the light.           |
| `r`, `g`, `b` | `32`, `255`, `250` - Greenish-white color. |

### LuaComponent
Attaches a Lua script to the entity for custom behavior.

| Attribute        | Description                               |
| :--------------- | :---------------------------------------- |
| `script_source_file` | `data/scripts/props/physics_fungus.lua` |
| `execute_every_n_frame` | `1` - Executes the script every frame.    |

## Physics Joints

The fungus is constructed from multiple physics bodies connected by joints, allowing for a flexible, segmented structure.

### PhysicsJoint2MutatorComponent
These components appear to control motor behavior for the joints, though their specific impact here is less clear without further context on `motor_speed` and `motor_max_torque` in this configuration.

### PhysicsJoint2Component
These define the connections between different parts of the fungus.

| Type             | `joint_id` | `body1_id` | `body2_id` | `break_force` | `break_distance` | Description                               |
| :--------------- | :--------- | :--------- | :--------- | :------------ | :--------------- | :---------------------------------------- |
| `REVOLUTE_JOINT` | `1` to `5` | Varies     | Varies     | `10`          | `5`              | Connects segments of the fungus stalk.    |
| `REVOLUTE_JOINT_ATTACH_TO_NEARBY_SURFACE` | N/A        | `105`      | N/A        | `35`          | `8`              | Attaches the foot of the fungus to the ground. |
