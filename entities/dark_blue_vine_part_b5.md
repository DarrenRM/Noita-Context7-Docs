# Dark Blue Vine Part B5

This entity defines a single segment of a dark blue vine, specifically part B5. It primarily utilizes a `SpriteComponent` to render its visual appearance.

## Sprite Component

The `SpriteComponent` is responsible for drawing the vine segment.

### Key Attributes:

*   **`image_file`**: Specifies the graphical asset used for the sprite.
    *   Value: `data/entities/verlet_chains/vines/vine_parts_blue/vine_dark_b_5.xml`
*   **`offset_x`**: Horizontal offset for the sprite.
    *   Value: `0`
*   **`offset_y`**: Vertical offset for the sprite.
    *   Value: `1.5`
*   **`update_transform`**: Controls whether the sprite's transform (position, scale, rotation) is updated.
    *   Value: `0` (Disabled)
*   **`update_transform_rotation`**: Controls whether the sprite's rotation is updated.
    *   Value: `0` (Disabled)
