# Root Verlet Part 3

This document describes the `root_verlet_3.xml` entity, a component of the root verlet chain in Noita.

## Entity Definition

The core of this entity is a `SpriteComponent` which defines its visual representation.

### SpriteComponent

*   **image_file**: `data/entities/verlet_chains/root/root_parts/root_3.xml` - Specifies the image file used for the sprite.
*   **offset_x**: `0` - Horizontal offset of the sprite.
*   **offset_y**: `3` - Vertical offset of the sprite.
*   **update_transform**: `0` - Disables automatic transform updates.
*   **update_transform_rotation**: `0` - Disables automatic rotation updates.
