# Blue Vine Part B3

This entity defines a specific segment of a blue vine, intended for use inverlet chains.

## Sprite Component

The `SpriteComponent` handles the visual representation of this vine part.

### Key Attributes:

*   **`image_file`**: `data/entities/verlet_chains/vines/vine_parts_blue/vine_b_3.xml` - Specifies the sprite asset used for this vine segment.
*   **`offset_x`**: `0` - Horizontal offset for the sprite.
*   **`offset_y`**: `1.5` - Vertical offset for the sprite.
*   **`update_transform`**: `0` - Disables automatic transform updates for the sprite.
*   **`update_transform_rotation`**: `0` - Disables automatic rotation updates for the sprite.
