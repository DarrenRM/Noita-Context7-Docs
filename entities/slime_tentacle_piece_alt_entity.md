# Slime Tentacle Piece (Alt) Entity

This document describes the `slime_tentacle_piece_alt.xml` entity, which represents a single segment of an alternative slime tentacle in Noita.

## Sprite Component

The `SpriteComponent` defines the visual appearance of the tentacle piece.

### Key Attributes:

*   **`image_file`**: Specifies the texture file used for the sprite.
    *   Value: `data/entities/verlet_chains/slime_tentacle/slime_tentacle_piece_alt.png`
*   **`offset_x`**: Horizontal offset of the sprite.
    *   Value: `0`
*   **`offset_y`**: Vertical offset of the sprite.
    *   Value: `1`
*   **`update_transform`**: Determines if the sprite's transform (position, scale, rotation) should be updated.
    *   Value: `0` (Disabled)
*   **`update_transform_rotation`**: Determines if the sprite's rotation should be updated.
    *   Value: `0` (Disabled)
