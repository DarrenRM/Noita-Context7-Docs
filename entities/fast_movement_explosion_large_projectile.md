# Fast Movement Explosion Large Projectile

This entity defines a projectile that, upon death (either from low velocity, lifetime expiration, or collision), triggers a large, damaging explosion. It's designed to be a fast-moving projectile that leaves a significant impact.

## Key Components

### ProjectileComponent

This is the core component that defines the projectile's behavior.
