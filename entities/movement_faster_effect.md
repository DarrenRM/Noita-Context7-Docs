# Movement Faster Effect

This entity defines a game effect that increases the player's movement speed.

## Key Components

### GameEffectComponent
This component is responsible for applying the actual game effect.

| Attribute | Value | Description |
|---|---|---|
| `effect` | `MOVEMENT_FASTER` | Specifies the type of effect to apply. |
| `frames` | `600` | The duration of the effect in game frames (60 frames per second). |
