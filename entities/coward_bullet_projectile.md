# Coward Bullet Projectile

This document details the configuration of the "Coward Bullet" projectile entity in Noita, focusing on its behavior, visual effects, and associated scripts for AI-assisted modding.

## Core Projectile Properties

The `ProjectileComponent` defines the fundamental behavior of the projectile.

| Attribute           | Value      | Description                                                                 |
| :------------------ | :--------- | :-------------------------------------------------------------------------- |
| `lob_min`           | `0.8`      | Minimum lob angle (in radians) for projectile trajectory.                   |
| `lob_max`           | `1.0`      | Maximum lob angle (in radians) for projectile trajectory.                   |
| `speed_min`         | `300`      | Minimum projectile speed.                                                   |
| `speed_max`         | `400`      | Maximum projectile speed.                                                   |
| `die_on_low_velocity` | `0`        | Projectile does not die when its velocity becomes low.                      |
| `on_death_explode`  | `1`        | Projectile explodes upon death.                                             |
| `on_lifetime_out_explode` | `1`        | Projectile explodes when its lifetime expires.                              |
| `explosion_dont_damage_shooter` | `1`        | The explosion caused by this projectile does not damage the shooter.        |
| `direction_random_rad` | `0.5`      | Random deviation in projectile direction (in radians).                      |
| `collide_with_entities` | `0`        | Projectile does not collide with entities.                                  |
| `damage`            | `0`        | Base damage of the projectile (this projectile's damage is handled by script). |
| `on_collision_die`  | `1`        | Projectile dies upon collision.                                             |
| `lifetime`          | `20`       | Duration (in seconds) before the projectile expires and explodes.           |
| `knockback_force`   | `2.0`      | Force applied to entities when the projectile hits them (if collision was enabled). |
