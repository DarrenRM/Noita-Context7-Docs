# Verlet Vine Entity

This document describes the `verlet_vine.xml` entity, which defines a dynamic vine structure in Noita. It utilizes the Verlet physics system to simulate chain-like behavior, allowing for flexible and reactive vines.

## Entity Components

### VerletPhysicsComponent

This component governs the physical simulation of the vine as a Verlet chain.
