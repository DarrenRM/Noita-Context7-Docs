# Iron Stomach Effect

This entity defines the "Iron Stomach" game effect in Noita.

## GameEffectComponent

This is the primary component that defines the effect.

### Key Attributes:

*   **`effect`**: Specifies the type of effect. In this case, it's `"IRON_STOMACH"`.
*   **`frames`**: The duration of the effect in game frames. `600` frames is equivalent to 10 seconds (60 frames per second).

This component is present but has no specific attributes defined, indicating it inherits default transform properties.
