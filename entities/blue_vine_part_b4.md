# Blue Vine Part B4

This entity defines a single segment of a blue vine, specifically the 'B4' variant. It's a visual component used inverlet chains to create dynamic vine structures.

## Key Components

### SpriteComponent
This is the primary component responsible for rendering the visual representation of the vine segment.

*   **`image_file`**: `data/entities/verlet_chains/vines/vine_parts_blue/vine_b_4.xml`
    *   Specifies the XML file containing the sprite's visual data.
*   **`offset_x`**: `0`
    *   Horizontal offset for the sprite.
*   **`offset_y`**: `1.5`
    *   Vertical offset for the sprite.
*   **`update_transform`**: `0`
    *   Disables automatic transform updates for the sprite.
*   **`update_transform_rotation`**: `0`
    *   Disables automatic rotation updates for the sprite.
