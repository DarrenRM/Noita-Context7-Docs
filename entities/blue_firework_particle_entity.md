# Blue Firework Particle Entity

This entity defines the behavior and visual components of a blue firework particle effect in Noita. It primarily uses `LoadEntitiesComponent` to spawn other particle entities, creating a chain reaction of visual effects.

## Key Components

### LoadEntitiesComponent

This component is responsible for loading and spawning other entities. In this firework, it's used to create the visual elements of the explosion.

                                           1.  **`count.min`**: The minimum number of entities to spawn.

### Specific `entity_file` values used:

*   **`data/entities/particles/particle_explosion/explosion_flare_blue.xml`**: Likely spawns a bright, blue flare effect, representing the initial burst of the firework.
*   **`data/entities/particles/particle_explosion/explosion_flare_fog_of_war_hole.xml`**: This suggests a particle that creates a temporary "hole" in the fog of war, perhaps for visual clarity or a specific gameplay mechanic related to the firework's visibility.
*   **`data/entities/particles/particle_explosion/explosion_trail_spark_blue.xml`**: This component spawns a variable number (30-42) of blue sparks, creating the trailing effect characteristic of fireworks. The `kill_entity="1"` on this component means the firework entity itself will be destroyed after these sparks are spawned.
