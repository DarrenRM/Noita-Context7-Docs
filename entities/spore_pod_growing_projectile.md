# Spore Pod Growing Projectile

This document details the `spore_pod_growing.xml` entity, which defines a projectile in Noita that grows and explodes. It's designed to be used in AI-assisted modding by highlighting key attributes and their functions.

## Entity Definition

The core of this entity is a projectile with specific behaviors and visual components.

```xml
<Entity name="$projectile_default" tags="teleportable_NOT,projectile">
    <!-- ... components ... -->
</Entity>
```

*   **`name="$projectile_default"`**: This is a placeholder name, indicating it inherits default projectile properties.
*   **`tags="teleportable_NOT,projectile"`**:
    *   `teleportable_NOT`: Prevents this projectile from being teleported.
    *   `projectile`: Identifies this entity as a projectile.

## Key Components

### ProjectileComponent

This component governs the projectile's fundamental behavior, including its lifespan, damage, and explosion properties.

```xml
<ProjectileComponent
    lifetime="60"
    lifetime_randomness="10"
    damage="0"
    muzzle_flash_file="data/entities/particles/muzzle_flashes/muzzle_flash_launcher_large.xml"
    shoot_light_flash_radius="120"
    shoot_light_flash_r="255"
    shoot_light_flash_g="240"
    shoot_light_flash_b="30"
    on_death_explode="1"
    on_lifetime_out_explode="1"
    on_death_gfx_leave_sprite="0"
    >
    <!-- config_explosion -->
</ProjectileComponent>
```

*   **`lifetime="60"`**: The base duration (in frames) the projectile exists.
*   **`lifetime_randomness="10"`**: Adds variability to the projectile's lifetime.
*   **`damage="0"`**: The projectile itself deals no direct damage. Damage is applied via its explosion.
*   **`muzzle_flash_file`**: Specifies the particle effect for the muzzle flash when fired.
*   **`shoot_light_flash_radius`, `shoot_light_flash_r`, `shoot_light_flash_g`, `shoot_light_flash_b`**: Define the radius and color of the light flash upon shooting.
*   **`on_death_explode="1"`**: Triggers an explosion when the projectile is destroyed by other means (e.g., hitting a wall).
*   **`on_lifetime_out_explode="1"`**: Triggers an explosion when the projectile's lifetime expires.
*   **`on_death_gfx_leave_sprite="0"`**: Prevents graphical effects from lingering after death.

#### `config_explosion`

This nested element defines the properties of the explosion that occurs upon the projectile's death or lifetime expiration.

```xml
<config_explosion
    never_cache="0"
    damage="5"
    camera_shake="5"
    explosion_radius="12"
    explosion_sprite="data/particles/explosion_008.xml"
    explosion_sprite_lifetime="10"
    load_this_entity="data/entities/particles/particle_explosion/spore_pod_spikes.xml"
    spark_material="spark_blue"
    audio_event_name="explosions/slime"
    create_cell_probability="40"
    hole_destroy_liquid="0"
    hole_enabled="1"
    ray_energy="200000"
    particle_effect="1"
    damage_mortals="1"
    physics_explosion_power.min="2.2"
    physics_explosion_power.max="3.6"
    physics_throw_enabled="1"
    shake_vegetation="1"
    sparks_count_min="12"
    sparks_count_max="15"
    sparks_enabled="1"
    stains_enabled="1"
    stains_radius="15"
    max_durability_to_destroy="8"
    audio_event_name="explosions/slime" >
</config_explosion>
```

*   **`damage="5"`**: The damage dealt by the explosion.
*   **`camera_shake="5"`**: The intensity of camera shake caused by the explosion.
*   **`explosion_radius="12"`**: The radius of the explosion's effect.
*   **`explosion_sprite`**: The visual sprite for the explosion.
*   **`load_this_entity`**: Specifies a custom entity to load upon explosion, likely for unique particle effects.
*   **`spark_material`**: The material used for sparks generated by the explosion.
*   **`audio_event_name`**: The sound event triggered by the explosion.
*   **`create_cell_probability="40"`**: The chance (in percent) of creating a "cell" effect.
*   **`hole_enabled="1"`**: Enables the creation of holes in surfaces.
*   **`ray_energy="200000"`**: The energy of any rays emitted by the explosion.
*   **`physics_explosion_power`**: The minimum and maximum force applied to physics objects by the explosion.
*   **`sparks_count_min`, `sparks_count_max`**: The range for the number of sparks generated.
*   **`stains_enabled="1"`**: Enables the creation of stains on surfaces.
*   **`max_durability_to_destroy`**: The maximum durability of objects that can be destroyed by the explosion.

### SpriteComponent

Defines the visual appearance of the projectile.

```xml
<SpriteComponent
    image_file="data/projectiles_gfx/spore_pod.xml"
    >
</SpriteComponent>
```

*   **`image_file="data/projectiles_gfx/spore_pod.xml"`**: Points to the graphical asset used for the spore pod.

### LuaComponent

This component links a Lua script to the projectile, enabling custom logic and behavior.

```xml
<LuaComponent
    script_source_file="data/scripts/projectiles/spore_pod_grow.lua"
    execute_every_n_frame="1"
    >
</LuaComponent>
```

*   **`script_source_file="data/scripts/projectiles/spore_pod_grow.lua"`**: The path to the Lua script that controls the projectile's growth and other dynamic behaviors.
*   **`execute_every_n_frame="1"`**: The script will execute every frame, allowing for continuous updates.

### ParticleEmitterComponent (x2)

These components are responsible for emitting particles, contributing to the visual effects of the projectile.

**First Emitter (Spore Pod Stalk):**

```xml
<ParticleEmitterComponent
    emitted_material_name="spore_pod_stalk"
    offset.x="0"
    offset.y="0"
    x_pos_offset_min="-1"
    x_pos_offset_max="2"
    y_pos_offset_min="-1"
    y_pos_offset_max="2"
    x_vel_min="0"
    x_vel_max="0"
    y_vel_min="0"
    y_vel_max="0"
    count_min="6"
    count_max="6"
    lifetime_min="0.1"
    lifetime_max="0.3"
    create_real_particles="1"
    emission_interval_min_frames="1"
    emission_interval_max_frames="1"
    is_emitting="1"
    friction="0"
    render_on_grid="1"
    attractor_force="0"
    >
</ParticleEmitterComponent>
```

*   **`emitted_material_name="spore_pod_stalk"`**: Specifies the material of the particles being emitted, likely for visual representation of growth.
*   **`count_min="6"`, `count_max="6"`**: Emits a fixed number of particles per emission cycle.
*   **`lifetime_min="0.1"`, `lifetime_max="0.3"`**: Short lifespan for these particles.
*   **`is_emitting="1"`**: The emitter is active.
