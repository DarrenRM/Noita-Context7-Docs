# Spore Pod Projectile

This entity defines the behavior and appearance of the Spore Pod projectile in Noita. When it lands, it transforms into a growing spore pod that leaves a trail of material and eventually explodes into spikes.

## Key Attributes

### ProjectileComponent

This component governs the projectile's physical properties and interactions.

## Description

When this projectile lands, it is switched to `spore_pod_growing` which moves away from surfaces, leaving a trail of material. After a while, it explodes into a bunch of `spore_pod_spike` entities (via `spore_pod_spikes.xml`).
