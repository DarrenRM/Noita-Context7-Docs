# Mine Projectile

This document details the configuration of the "Mine" projectile entity in Noita, focusing on attributes relevant to AI-assisted modding.

## Core Components

### ProjectileComponent

This component defines the fundamental behavior and physics of the projectile.

| Attribute             | Description                                                                 |
| :-------------------- | :-------------------------------------------------------------------------- |
| `lob_min`, `lob_max`  | Controls the arc of the projectile.                                         |
| `speed_min`, `speed_max`| Sets the initial velocity range.                                            |
| `friction`            | How quickly the projectile loses speed.                                     |
| `direction_random_rad`| Adds a small random deviation to the projectile's trajectory.               |
| `on_death_explode`    | If set to 1, the projectile explodes upon death.                            |
| `bounces_left`        | The number of times the projectile can bounce off surfaces.                 |
| `camera_shake_when_shot`| The intensity of camera shake when the projectile is fired.                 |
| `physics_impulse_coeff`| Affects the force applied to objects it hits.                               |

#### `config_explosion` (Nested within `ProjectileComponent`)

Defines the parameters of the explosion that occurs when `on_death_explode` is true.

| Attribute              | Description                                                                 |
| :--------------------- | :-------------------------------------------------------------------------- |
| `explosion_radius`     | The area of effect for the explosion.                                       |
| `damage`               | The damage dealt by the explosion.                                          |
| `ray_energy`           | The energy of the destructive rays emitted by the explosion.                |
| `hole_enabled`         | If set to 1, the explosion creates holes in terrain.                        |
| `physics_explosion_power.min`, `physics_explosion_power.max` | The range of force applied to physics objects by the explosion. |
| `sparks_enabled`       | If set to 1, sparks are generated by the explosion.                         |
| `audio_event_name`     | The sound event triggered by the explosion.                                 |

### HitboxComponent

Defines the collision boundaries of the projectile.

| Attribute     | Description                                                                 |
| :------------ | :-------------------------------------------------------------------------- |
| `aabb_min_x`, `aabb_max_x`, `aabb_min_y`, `aabb_max_y` | Defines the bounding box for collision detection. |
| `is_enemy`    | If set to 1, the hitbox is considered an enemy for certain interactions.    |

### DamageModelComponent

Determines how the projectile interacts with damage and its own health.

| Attribute   | Description                                                                 |
| :---------- | :-------------------------------------------------------------------------- |
| `hp`        | The health of the projectile.                                               |
| `fire_damage_amount` | The amount of fire damage it deals if it ignites.                       |

### CollisionTriggerComponent

Handles specific collision events and triggers.

| Attribute           | Description                                                                 |
| :------------------ | :-------------------------------------------------------------------------- |
| `width`, `height`, `radius` | Defines the trigger area dimensions.                                    |
| `required_tag`      | The tag an entity must have to trigger this component.                      |
| `timer_for_destruction` | The number of frames before the projectile is destroyed after triggering. |

### LuaComponent

Allows for custom scripting to extend projectile behavior.

| Attribute              | Description                                                                 |
| :--------------------- | :-------------------------------------------------------------------------- |
| `script_collision_trigger_hit` | Script executed when the `CollisionTriggerComponent` is hit.            |
| `script_damage_received` | Script executed when the projectile receives damage.                        |
| `script_source_file`   | A general script file to execute.                                           |

### SpriteComponent

Defines the visual appearance of the projectile.

| Attribute             | Description                                                                 |
| :-------------------- | :-------------------------------------------------------------------------- |
| `image_file`          | The path to the sprite image.                                               |
| `rect_animation`      | The name of the animation to play (e.g., "stand").                        |
| `emissive`            | If set to 1, the sprite emits light.                                        |
| `additive`            | If set to 1, the sprite uses additive blending.                             |

### AudioComponent

Manages the sound effects associated with the projectile.

| Attribute      | Description                                                                 |
| :------------- | :-------------------------------------------------------------------------- |
| `file`         | The path to the audio bank.                                                 |
| `event_root`   | The root event name for sounds played by this component.                    |

## Other Notable Components

*   **Base:** Inherits base projectile properties.
*   **CameraBoundComponent:** Manages how the projectile interacts with the camera's view.
*   **ExplosionComponent:** A separate component that can also define explosion behavior, often used in conjunction with `ProjectileComponent`.

This documentation provides a focused overview of the "Mine" projectile's key attributes for modding purposes. For a complete understanding, refer to the original XML file and Noita's modding documentation.
