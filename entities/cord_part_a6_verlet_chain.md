# Cord Part A6 Verlet Chain

This entity defines a single segment of a verlet chain, specifically designed for use as a cord.

## Sprite Component

The `SpriteComponent` handles the visual representation of this cord segment.

### Key Attributes:

*   **`image_file`**: Specifies the sprite image to be used.
    *   `data/entities/verlet_chains/cords/cord_parts/cord_a_6.xml`
*   **`offset_x`**: Horizontal offset for the sprite.
    *   `0`
*   **`offset_y`**: Vertical offset for the sprite.
    *   `1.5`
*   **`update_transform`**: Controls whether the sprite's transform is updated.
    *   `0` (Disabled)
*   **`update_transform_rotation`**: Controls whether the sprite's rotation is updated.
    *   `0` (Disabled)
