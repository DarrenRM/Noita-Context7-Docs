# Heavy Bullet Projectile

This document details the configuration of the "Heavy Bullet" projectile entity in Noita, designed for AI-assisted modding.

## Core Components

The Heavy Bullet projectile is defined by several key components that govern its behavior, appearance, and effects.

### Entity Definition

```xml
<Entity name="$projectile_default" tags="projectile_player" >
```

*   **`name="$projectile_default"`**: This is a placeholder name, indicating it inherits default projectile properties.
*   **`tags="projectile_player"`**: This tag identifies the projectile as originating from the player.

### Base Projectile Properties

```xml
<Base file="data/entities/base_projectile.xml" >
	<VelocityComponent
		air_friction="0.3"
		gravity_y="200"
		mass="0.08"
		>
	</VelocityComponent>
</Base>
```

*   **`VelocityComponent`**: Defines the physical movement characteristics.
    *   `air_friction`: Resistance to movement in the air.
    *   `gravity_y`: The force of gravity applied along the Y-axis.
    *   `mass`: The projectile's mass, affecting its inertia.

### Projectile Behavior (`ProjectileComponent`)

This is the primary component defining the projectile's unique attributes.

```xml
<ProjectileComponent
    _enabled="1"
    lob_min="0.8"
    lob_max="1.0"
    speed_min="600"
    speed_max="750"
    friction="1"
    direction_random_rad="0.05"
    on_death_explode="1"
    on_death_gfx_leave_sprite="0"
    on_lifetime_out_explode="1"
    explosion_dont_damage_shooter="1"
    on_collision_die="1"
    lifetime="30"
    damage="0.51"
    velocity_sets_scale="1"
    lifetime_randomness="7"
    ragdoll_force_multiplier="0.05"
    hit_particle_force_multiplier="0.25"
    camera_shake_when_shot="0.1"
    muzzle_flash_file="data/entities/particles/muzzle_flashes/muzzle_flash_magic_launcher.xml"
    shoot_light_flash_r="120"
    shoot_light_flash_g="235"
    shoot_light_flash_b="90"
    shoot_light_flash_radius="100"
    knockback_force="2.6"
    physics_impulse_coeff="3000"
    >
    <config_explosion
      never_cache="1"
      damage="0"
      camera_shake="2.0"
      explosion_radius="6"
      explosion_sprite="data/particles/explosion_016_slime.xml"
      explosion_sprite_lifetime="0"
      create_cell_probability="0"
      explosion_sprite_additive="1"
      hole_destroy_liquid="0"
      hole_enabled="1"
      ray_energy="300000"
      particle_effect="0"
      damage_mortals="1"
      physics_explosion_power.min="0.1"
      physics_explosion_power.max="0.3"
      physics_throw_enabled="1"
      shake_vegetation="1"
      sparks_count_max="1"
      sparks_count_min="3"
      sparks_enabled="1"
      material_sparks_enabled="1"
      material_sparks_count_max="3"
      material_sparks_count_min="0"
      light_enabled="0"
      stains_enabled="1"
      stains_radius="1"
      spark_material="spark_green"
      >
    </config_explosion>
</ProjectileComponent>
```

*   **`lob_min`, `lob_max`**: Controls the arc of the projectile.
*   **`speed_min`, `speed_max`**: Defines the projectile's initial velocity range.
*   **`friction`**: How much the projectile slows down over time.
*   **`direction_random_rad`**: Adds a small random deviation to the projectile's direction.
*   **`on_death_explode`**: Triggers an explosion upon death (e.g., hitting a target or expiring).
*   **`on_lifetime_out_explode`**: Triggers an explosion when the projectile's lifetime ends.
*   **`on_collision_die`**: The projectile dies upon collision.
*   **`lifetime`**: The maximum duration the projectile exists in seconds.
*   **`damage`**: The base damage dealt by the projectile.
*   **`knockback_force`**: The force applied to knock back entities upon impact.
*   **`physics_impulse_coeff`**: Coefficient for physics impulse calculations.
*   **`config_explosion`**: Nested configuration for explosion effects.
    *   `camera_shake`: Amount of camera shake during the explosion.
    *   `explosion_radius`: The area of effect for the explosion.
    *   `explosion_sprite`: The visual sprite used for the explosion.
    *   `hole_enabled`: Whether the explosion creates holes in terrain.
    *   `ray_energy`: Energy value for raycasting effects from the explosion.
    *   `damage_mortals`: Whether the explosion damages living entities.
    *   `physics_explosion_power`: The strength of the physics-based explosion.
    *   `stains_enabled`, `stains_radius`: Controls the creation and size of impact stains.
    *   `spark_material`: The material used for sparks generated by the explosion.

### Visual Representation (`SpriteComponent`)

```xml
<SpriteComponent
    _enabled="1"
    alpha="1"
    image_file="data/projectiles_gfx/fireball_smaller_green.xml"
    rect_animation="fireball"
    additive="1"
     >
</SpriteComponent>
```

*   **`image_file`**: Specifies the graphical asset for the projectile.
*   **`rect_animation`**: Refers to a specific animation frame or sequence within the image file.
*   **`additive`**: Indicates that the sprite uses additive blending for a brighter, more emissive look.


*   **`file`**: The audio bank containing the sound effects.
*   **`event_root`**: The specific sound event to trigger for this projectile.

### Particle Emitters (`ParticleEmitterComponent`)

Two particle emitters are used to create visual effects:

**Emitter 1 (Trail/Subtle Sparks)**

```xml
<ParticleEmitterComponent
    emitted_material_name="spark_yellow"
    x_pos_offset_min="-3"
    x_pos_offset_max="3"
    y_pos_offset_min="-1"
    y_pos_offset_max="1"
    gravity.y="0"
    x_vel_min="-2"
    x_vel_max="2"
    count_min="1"
    count_max="1"
    is_trail="1"
    trail_gap="0.6"
    lifetime_min="0.1"
    lifetime_max="0.2"
    airflow_force="10.5"
    airflow_time="1.401"
    airflow_scale="0.05"
    emit_cosmetic_particles="1"
    emission_interval_min_frames="1"
    emission_interval_max_frames="2"
    is_emitting="1" >
</ParticleEmitterComponent>
```

*   **`emitted_material_name`**: The type of particle to emit (yellow sparks).
*   **`is_trail`**: Indicates this emitter is for a trail effect.
*   **`trail_gap`**: Spacing between trail particles.
*   **`lifetime_min`, `lifetime_max`**: Duration of individual particles.
*   **`airflow_force`, `airflow_time`, `airflow_scale`**: Parameters controlling how air currents affect the particles.

**Emitter 2 (Impact/Burst Sparks)**

```xml
<ParticleEmitterComponent
    emitted_material_name="spark_green"
    x_pos_offset_min="-1"
    x_pos_offset_max="1"
    y_pos_offset_min="-2"
    y_pos_offset_max="2"
    gravity.y="0"
    x_vel_min="20"
    x_vel_max="80"
    count_min="1"
    count_max="4"
    is_trail="1"
    trail_gap="1.2"
    lifetime_min="0.2"
    lifetime_max="0.6"
    airflow_force="1.5"
    airflow_time="0.401"
    airflow_scale="0.05"
    emit_cosmetic_particles="1"
    emission_interval_min_frames="1"
    emission_interval_max_frames="2"
    is_emitting="1" >
</ParticleEmitterComponent>
```

*   **`emitted_material_name`**: The type of particle to emit (green sparks).
*   **`x_vel_min`, `x_vel_max`**: Controls the horizontal velocity of emitted particles.
*   **`count_min`, `count_max`**: The number of particles emitted per burst.

### Lighting (`LightComponent`)
