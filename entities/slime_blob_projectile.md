# Slime Blob Projectile

This document details the configuration of the `slimeblob.xml` entity, which defines the behavior and appearance of the slime blob projectile in Noita.

## Core Functionality

The `slimeblob.xml` entity inherits from `base_projectile.xml`, gaining fundamental projectile properties.

### `ProjectileComponent`

This component governs the projectile's movement, collision, and death behaviors.
