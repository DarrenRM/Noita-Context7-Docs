# Wraith Glowing Laser Projectile

This document details the configuration for the "Wraith Glowing Laser" projectile entity in Noita, intended for AI-assisted modding.

## Core Entity Configuration

The projectile is based on the `base_projectile.xml` entity, inheriting its fundamental properties.

```xml
<Entity 
  name="$projectile_default" 
>
	<Base file="data/entities/base_projectile.xml" >
		<VelocityComponent
			air_friction="-1"
			gravity_y="0" 
			mass="0.09"
			>
		</VelocityComponent>
	</Base>
    <!-- ... other components ... -->
</Entity>
```

## Projectile Component Key Attributes

This component defines the projectile's behavior, speed, damage, and interaction properties.

| Attribute                 | Value      | Description                                                                 |
| :------------------------ | :--------- | :-------------------------------------------------------------------------- |
| `lob_min`, `lob_max`      | `0.8`, `1.0` | Controls the projectile's arc. Values closer to 1 indicate a straighter path. |
| `speed_min`, `speed_max`  | `800`, `900` | The range of initial velocities for the projectile.                         |
| `friction`                | `1`        | How much the projectile slows down over time.                               |
| `direction_random_rad`    | `0.03`     | Adds a small amount of randomness to the projectile's initial direction.    |
| `on_death_explode`        | `1`        | The projectile explodes upon death (e.g., hitting a surface).               |
| `on_lifetime_out_explode` | `1`        | The projectile explodes when its lifetime expires.                          |
| `explosion_dont_damage_shooter` | `1`    | The explosion from this projectile will not harm the entity that fired it.  |
| `die_on_low_velocity`     | `1`        | The projectile will be destroyed if its velocity drops too low.             |
| `on_collision_die`        | `0`        | The projectile does not die immediately upon collision (unless it explodes). |
| `lifetime`                | `90`       | The maximum duration of the projectile in frames.                           |
| `lifetime_randomness`     | `7`        | Adds randomness to the projectile's lifetime.                               |
| `damage`                  | `0`        | Base damage of the projectile itself (before explosion or type modifiers).  |
| `knockback_force`         | `2.8`      | The force applied to entities that are hit by the projectile.               |
| `physics_impulse_coeff`   | `4000`     | Coefficient for physics impulse applied on collision.                       |
| `camera_shake_when_shot`  | `0.4`      | Amount of camera shake when the projectile is fired.                        |
| `shoot_light_flash_radius`| `100`      | Radius of the light flash emitted when the projectile is shot.              |

### Explosion Configuration (`config_explosion`)

Details about the explosion that occurs when the projectile dies or its lifetime ends.

| Attribute                 | Value      | Description                                                                 |
| :------------------------ | :--------- | :-------------------------------------------------------------------------- |
| `damage`                  | `0.58`     | Damage dealt by the explosion.                                              |
| `camera_shake`            | `20.0`     | Amount of camera shake caused by the explosion.                             |
| `explosion_radius`        | `20`       | The radius of the explosion.                                                |
| `ray_energy`              | `2050000`  | Energy of the destructive rays emitted by the explosion.                    |
| `max_durability_to_destroy`| `11`      | Maximum durability of objects that can be destroyed by the explosion.       |
| `damage_mortals`          | `1`        | The explosion damages living entities.                                      |
| `physics_explosion_power.min`, `.max` | `0.15`, `0.35` | Range of physics force applied by the explosion.                            |
| `sparks_enabled`          | `1`        | Sparks are generated by the explosion.                                      |
| `stains_enabled`          | `1`        | Creates stains on surfaces where the explosion occurs.                      |
| `spark_material`          | `spark_white`| The material used for the sparks.                                           |

## Particle Emitter Components

These components are responsible for generating visual effects, such as trails and cosmetic particles.

### Trail Emitter

Generates a continuous trail of particles behind the projectile.

| Attribute                 | Value   | Description                               |
| :------------------------ | :------ | :---------------------------------------- |
| `emitted_material_name`   | `spark_white` | The material of the emitted particles.    |
| `lifetime_min`, `lifetime_max` | `0.1`, `0.3` | Duration of individual particles.         |
| `is_trail`                | `1`     | Marks this emitter as a trail effect.     |
| `trail_gap`               | `0.5`   | Spacing between trail particles.          |
| `emit_cosmetic_particles` | `1`     | Emits visual-only particles.              |
| `emission_interval_min_frames`, `max_frames` | `1`, `2` | Frequency of particle emission.           |
