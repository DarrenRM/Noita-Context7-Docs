# Melee Counter Effect

This entity defines a game effect that triggers a "melee counter" action.

## GameEffectComponent

This component is responsible for defining the specific game effect.

### Key Attributes:

*   **`effect`**: Specifies the type of effect. In this case, it's set to `"MELEE_COUNTER"`.
*   **`frames`**: Determines the duration of the effect in game frames. Here, it's set to `3` frames.
