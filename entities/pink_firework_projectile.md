# Pink Firework Projectile

This document details the configuration for the "firework_pink.xml" projectile entity in Noita, designed for AI-assisted modding.

## Base Projectile Configuration

The projectile inherits its fundamental properties from `base_projectile.xml`.

### Velocity Component

| Attribute      | Value   | Description                                     |
|----------------|---------|-------------------------------------------------|
| `gravity_y`    | `50`    | Downward acceleration due to gravity.           |
| `air_friction` | `-5.0`  | Resistance to movement in the air.              |
| `mass`         | `0.05`  | The mass of the projectile.                     |

## Projectile Component

This component defines the core behavior and effects of the pink firework projectile.

### Key Attributes

| Attribute                     | Value   | Description                                                                 |
|-------------------------------|---------|-----------------------------------------------------------------------------|
| `lob_min`, `lob_max`          | `0.8`, `1.0` | Controls the arc of the projectile's trajectory.                            |
| `speed_min`, `speed_max`      | `70`, `80`  | The initial speed range of the projectile.                                  |
| `on_death_explode`            | `1`     | The projectile explodes upon death (e.g., hitting a surface or expiring). |
| `on_lifetime_out_explode`     | `1`     | The projectile explodes when its lifetime expires.                          |
| `on_collision_die`            | `1`     | The projectile is destroyed upon collision.                                 |
| `lifetime`                    | `30`    | The base duration of the projectile before expiring.                        |
| `lifetime_randomness`         | `7`     | Adds randomness to the projectile's lifetime.                               |
| `damage`                      | `1`     | Base damage dealt by the projectile itself.                                 |
| `camera_shake_when_shot`      | `15.0`  | The intensity of camera shake when this projectile is fired.                |
| `shoot_light_flash_r`, `g`, `b` | `255`, `40`, `210` | RGB values for the light flash when the projectile is shot.               |
| `shoot_light_flash_radius`    | `90`    | The radius of the light flash when shot.                                    |
| `knockback_force`             | `3.0`   | The force applied to knock back entities upon impact.                       |

### Explosion Configuration (`config_explosion`)

This nested element defines the properties of the explosion that occurs when the projectile dies.

| Attribute                 | Value   | Description                                                                 |
|---------------------------|---------|-----------------------------------------------------------------------------|
| `never_cache`             | `1`     | Prevents the explosion from being cached, ensuring fresh effects.           |
| `camera_shake`            | `20`    | The intensity of camera shake caused by the explosion.                      |
| `explosion_radius`        | `15`    | The radius of the explosion's effect.                                       |
| `load_this_entity`        | `data/entities/particles/fireworks/firework_pink.xml` | The entity to load for the explosion's visual effects.                      |
| `create_cell_probability` | `10`    | Probability of creating cell-based effects during the explosion.            |
| `damage`                  | `1.2`   | Damage dealt by the explosion.                                              |
| `physics_explosion_power.min`, `max` | `1.0`, `1.4` | The range of physics force applied by the explosion.                        |
| `spark_material`          | `plasma_fading_pink` | The material used for sparks generated by the explosion.                    |
| `sparks_count_min`, `max` | `7`, `20`   | The range for the number of sparks generated.                               |
| `audio_event_name`        | `explosions/glitter` | The audio event triggered by the explosion.                                 |

## Light Component

This component adds a persistent light source to the projectile.

| Attribute | Value   | Description                               |
|-----------|---------|-------------------------------------------|
| `r`, `g`, `b` | `255`, `30`, `210` | RGB values for the light.                 |
| `radius`  | `150`   | The radius of the light emitted.          |

## Particle Emitter Components

Multiple `ParticleEmitterComponent` instances are used to create various visual effects.

### Smoke Emitter

*   **`emitted_material_name`**: `smoke`
*   **`count_min`/`max`**: `5`
*   **`lifetime_min`/`max`**: `0.1`, `0.3`
*   **`create_real_particles`**: `1`
*   **`is_emitting`**: `1`

### Pink Plasma Trail Emitters (3 instances)

These emitters create the characteristic pink plasma trails of the firework. They differ primarily in their `gravity.y` value, creating upward, downward, and neutral movement of particles.

*   **`emitted_material_name`**: `plasma_fading_pink`
*   **`count_min`/`max`**: `1`, `8`
*   **`lifetime_min`/`max`**: `0.3`, `0.5`
*   **`is_trail`**: `1`
*   **`trail_gap`**: `0.5`
*   **`create_real_particles`**: `0`
*   **`emit_cosmetic_particles`**: `1`
*   **`is_emitting`**: `1`

### Sparse Pink Plasma Emitter

This emitter creates fewer, longer-lasting plasma particles with a specific velocity.

*   **`emitted_material_name`**: `plasma_fading_pink`
*   **`count_min`/`max`**: `1`
*   **`lifetime_min`/`max`**: `1.4`, `1.5`
*   **`x_vel_min`/`max`**: `10`, `20`
*   **`is_trail`**: `0`
*   **`emission_interval_min_frames`/`max_frames`**: `8`, `12`
*   **`create_real_particles`**: `0`
*   **`emit_cosmetic_particles`**: `1`
*   **`is_emitting`**: `1`
