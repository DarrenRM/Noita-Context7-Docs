# Critical Hit Boost Effect

This entity defines a game effect that provides a temporary boost to critical hit chance.

## GameEffectComponent

This component governs the core functionality of the game effect.

### Key Attributes:

*   **effect**: `CRITICAL_HIT_BOOST` - Specifies the type of effect being applied.
*   **frames**: `720` - The duration of the effect in game frames (720 frames is equivalent to 12 seconds at 60 FPS).

## InheritTransformComponent

This component is present but has no specific attributes configured, indicating it doesn't alter the entity's transform properties in this instance.
