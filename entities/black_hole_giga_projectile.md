# Black Hole Giga Projectile

This document details the `black_hole_giga.xml` entity, representing a powerful black hole projectile in Noita. It focuses on key attributes that define its behavior, appearance, and effects.

## Core Functionality

The Black Hole Giga is a projectile that creates a gravitational pull, damages entities within its radius, and can convert materials.

### Projectile Component

This component defines the fundamental projectile properties.

| Attribute                 | Value                               | Description                                                                 |
| :------------------------ | :---------------------------------- | :-------------------------------------------------------------------------- |
| `lob_min`, `lob_max`      | `0.8`, `1.0`                        | Controls the projectile's lobbing behavior.                                 |
| `speed_min`, `speed_max`  | `100`, `140`                        | The range of speeds the projectile can travel at.                           |
| `on_death_explode`        | `1`                                 | The projectile explodes when it dies.                                       |
| `on_lifetime_out_explode` | `1`                                 | The projectile explodes when its lifetime expires.                          |
| `explosion_dont_damage_shooter` | `1`                             | Prevents the projectile's explosion from damaging the entity that shot it.  |
| `on_collision_die`        | `0`                                 | The projectile does not die upon collision with entities.                   |
| `lifetime`                | `1000`                              | The maximum duration of the projectile in frames.                           |
| `damage`                  | `0`                                 | Base damage of the projectile (area damage is handled separately).          |
| `damage_every_x_frames`   | `5`                                 | How often the projectile applies damage (in frames).                        |
| `knockback_force`         | `0.1`                               | The force applied to entities when hit.                                     |
| `penetrate_entities`      | `1`                                 | Allows the projectile to pass through multiple entities.                    |
| `camera_shake_when_shot`  | `0.4`                               | The intensity of camera shake when the projectile is fired.                 |
| `collide_with_shooter_frames` | `10`                              | Frames the projectile is immune to colliding with its shooter.              |
| `friendly_fire`           | `1`                                 | Allows the projectile to damage friendly entities.                          |
| `shoot_light_flash_radius`| `250`                               | Radius of the light flash when the projectile is shot.                      |
| `shoot_light_flash_r/g/b` | `255`, `180`, `230`                 | RGB values for the light flash color.                                       |

### `config_explosion`

Defines the properties of the explosion that occurs when the projectile dies or its lifetime ends.

| Attribute                 | Value                               | Description                                                                 |
| :------------------------ | :---------------------------------- | :-------------------------------------------------------------------------- |
| `damage`                  | `0`                                 | Base damage of the explosion.                                               |
| `camera_shake`            | `0`                                 | Camera shake intensity of the explosion.                                    |
| `explosion_radius`        | `1`                                 | The radius of the explosion effect.                                         |
| `explosion_sprite`        | `data/particles/black_hole_big_circle_out.xml` | The sprite used for the explosion effect.                                   |
| `hole_enabled`            | `1`                                 | Enables the black hole effect during the explosion.                         |
| `damage_mortals`          | `1`                                 | The explosion damages mortal entities.                                      |
| `shake_vegetation`        | `1`                                 | The explosion shakes vegetation.                                            |
| `sparks_count_min/max`    | `4`, `1`                            | Minimum and maximum number of sparks generated by the explosion.            |
| `spark_material`          | `plasma_fading`                     | The material of the sparks generated.                                       |

## Special Effects and Interactions

### MagicConvertMaterialComponent

Allows the projectile to convert materials in its vicinity.

| Attribute       | Value | Description                                     |
| :-------------- | :---- | :---------------------------------------------- |
| `extinguish_fire` | `1`   | Extinguishes fire in its area.                  |
| `steps_per_frame` | `16`  | How many conversion steps per frame.            |
| `is_circle`     | `1`   | The conversion area is circular.                |
| `radius`        | `120` | The radius of the material conversion effect.   |
| `loop`          | `1`   | The conversion effect loops.                    |

| :---------------- | :---------------------------------- | :---------------------------------------- |

| `eat_probability` | `80`                                | The chance of eating a cell.              |

| `update_every_n_frame`| `1`                               | How often the damage is applied.          |
| `entities_with_tag`| `mortal`                            | Targets entities with the "mortal" tag.   |

| `damage_type`     | `DAMAGE_CURSE`                      | The type of damage applied.               |

| `x_vel_min/max`, `y_vel_min/max` | `-80` to `80`                       | The velocity range of emitted particles.                                    |
| `area_circle_radius.min/max`| `24`, `72`                          | The radius range of the particle emission area.                             |
| `velocity_always_away_from_center` | `300`                             | Particles are always pushed away from the center.                           |

| `lifetime_min/max`          | `0.1`, `1.4`                        | The lifetime range of emitted particles.                                    |
| `create_real_particles`     | `0`                                 | Does not create actual physics-simulated particles.                         |
| `emit_cosmetic_particles`   | `1`                                 | Emits particles for visual effect only.                                     |

| `render_on_grid`            | `1`                                 | Particles are rendered on the grid.                                         |
| `emission_interval_min/max_frames` | `1`, `1`                          | Particles are emitted every frame.                                          |
| `attractor_force`           | `16`                                | Force that attracts particles towards a point.                              |
