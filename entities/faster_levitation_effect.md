# Faster Levitation Effect

This entity defines a game effect that grants the player faster levitation.

## Key Components

### GameEffectComponent
This component is responsible for applying the game effect.

*   **effect**: `FASTER_LEVITATION` - Specifies the type of effect to apply.
*   **frames**: `600` - The duration of the effect in frames (approximately 10 seconds at 60 FPS).
