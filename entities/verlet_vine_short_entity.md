# Verlet Vine Short Entity

This document describes the `verlet_vine_short.xml` entity, which defines a short, physics-simulated vine segment in Noita. It utilizes the `VerletPhysicsComponent` to create a chain of connected points that react to physics, wind, and gravity.

## Key Components

### VerletPhysicsComponent

This component governs the physics simulation of the vine.
