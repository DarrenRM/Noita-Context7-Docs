# Edit Wands Everywhere Effect

This entity grants the player the ability to edit their wands from anywhere in the game.

## Key Components

### GameEffectComponent
This component defines the special effect applied to the player.

| Attribute | Value | Description |
|---|---|---|
| `effect` | `EDIT_WANDS_EVERYWHERE` | Specifies the type of game effect. |
| `frames` | `600` | The duration of the effect in frames (approximately 10 seconds). |
