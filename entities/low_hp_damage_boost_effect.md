# Low HP Damage Boost Effect

This entity defines a game effect that grants a damage boost when the player's health is low.

## Key Components

### GameEffectComponent

This component applies the core game effect.

| Attribute | Value | Description |
|---|---|---|
| `effect` | `LOW_HP_DAMAGE_BOOST` | Specifies the type of effect to apply. |
| `frames` | `600` | The duration of the effect in frames (approximately 10 seconds). |
