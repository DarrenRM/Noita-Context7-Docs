# Slime Tentacle Piece (Thin Alt)

This entity defines a thin, alternative visual piece for a slime tentacle in Noita. It primarily consists of a sprite component.

## Sprite Component

This component dictates the visual appearance of the entity.

### Key Attributes:

*   **`image_file`**: `data/entities/verlet_chains/slime_tentacle/slime_tentacle_piece_thin_alt.png`
    *   Specifies the image file used for the sprite.
*   **`offset_x`**: `0`
    *   Horizontal offset of the sprite.
*   **`offset_y`**: `0.5`
    *   Vertical offset of the sprite.
*   **`update_transform`**: `0`
    *   Indicates whether the sprite's transform (position, scale) should be updated. `0` means no.
*   **`update_transform_rotation`**: `0`
    *   Indicates whether the sprite's rotation should be updated. `0` means no.
