# Meatvine Part A4 Sprite

This document describes the sprite component for a specific part of the meatvine entity in Noita.

## SpriteComponent

This component defines the visual representation of the entity.

### Key Attributes

*   **image\_file**: Specifies the path to the image file used for the sprite.
    *   `data/entities/verlet_chains/meatvine/vine_parts/vine_a_4.xml`
*   **offset\_x**: Horizontal offset for the sprite.
    *   `0`
*   **offset\_y**: Vertical offset for the sprite.
    *   `1.5`
*   **update\_transform**: Determines if the sprite's transform should be updated.
    *   `0` (Disabled)
*   **update\_transform\_rotation**: Determines if the sprite's rotation should be updated.
    *   `0` (Disabled)
