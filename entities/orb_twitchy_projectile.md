# Orb Twitchy Projectile

This document details the configuration of the "Orb Twitchy" projectile entity in Noita, focusing on its behavior, visual representation, and effects.

## Core Components

### Base Projectile (`<Base file="data/entities/base_projectile.xml">`)

This component defines fundamental projectile properties.

*   **`gravity_y="0"`**: The projectile is not affected by gravity.
*   **`air_friction="-0.1"`**: A slight negative air friction, potentially affecting its movement in unusual ways.
*   **`mass="0.05"`**: The projectile has a very low mass.

### Homing Component (`<HomingComponent>`)

This component enables the projectile to track targets.

*   **`target_tag="prey"`**: The projectile will attempt to home in on entities tagged as "prey".
*   **`homing_targeting_coeff="9"`**: A high coefficient, indicating strong homing behavior.
*   **`detect_distance="350"`**: The projectile can detect targets within this range.
*   **`homing_velocity_multiplier="1.0"`**: The homing mechanism does not alter the projectile's base velocity.

### Projectile Component (`<ProjectileComponent>`)

This is the primary component defining the projectile's offensive and survival characteristics.

*   **`lob_min="0.8"`, `lob_max="1.0"`**: Controls the lobbing behavior, with values close to 1 suggesting a relatively straight trajectory.
*   **`speed_min="100"`, `speed_max="120"`**: Defines the projectile's speed range.
*   **`die_on_low_velocity="0"`**: The projectile will not die if its velocity drops too low.
*   **`on_death_explode="1"`**: The projectile explodes upon death.
*   **`on_lifetime_out_explode="1"`**: The projectile explodes when its lifetime expires.
*   **`damage="0.3"`**: The base damage dealt by the projectile upon collision.
*   **`on_collision_die="1"`**: The projectile dies upon colliding with an entity.
*   **`lifetime="150"`**: The projectile exists for 150 frames before expiring.

#### Explosion Configuration (`<config_explosion>`)

Defines the parameters of the explosion that occurs upon the projectile's death.

*   **`never_cache="1"`**: Ensures the explosion effect is always generated fresh.
*   **`camera_shake="0.5"`**: The explosion causes moderate camera shake.
*   **`explosion_radius="1"`**: The radius of the explosion's effect.
*   **`explosion_sprite="data/particles/explosion_016_plasma.xml"`**: The visual sprite used for the explosion.
*   **`create_cell_probability="1"`**: There is a 100% chance to create cells upon explosion.
*   **`create_cell_material="spark_blue"`**: The material of the cells created is "spark_blue".
*   **`ray_energy="10000"`**: A high energy value for potential ray interactions.
*   **`hole_destroy_liquid="1"`**: The explosion can destroy liquids.
*   **`hole_enabled="1"`**: The explosion creates holes in surfaces.
*   **`hole_image="data/temp/explosion_hole.png"`**: The image used for the explosion hole.
*   **`damage="0.1"`**: The explosion itself deals a small amount of damage.
*   **`sparks_enabled="1"`**: Sparks are generated by the explosion.
*   **`spark_material="spark_white"`**: The material of the sparks is "spark_white".
*   **`sparks_count_min="12"`, `sparks_count_max="16"`**: The number of sparks generated.
*   **`light_r="180"`, `light_g="180"`, `light_b="180"`**: The explosion emits white light.
*   **`stains_enabled="1"`**: The explosion leaves stains.

### Sprite Component (`<SpriteComponent>`)

Defines the visual appearance of the projectile.

*   **`image_file="data/projectiles_gfx/orb_grey.xml"`**: Specifies the graphical asset for the projectile.
*   **`rect_animation="fireball"`**: Uses a "fireball" animation.
*   **`emissive="1"`, `additive="1"`**: The sprite is emissive and uses additive blending for a glowing effect.
*   **`offset_x="5"`, `offset_y="5"`**: The sprite is offset by 5 pixels horizontally and vertically.

### Light Component (`<LightComponent>`)

Adds a light source to the projectile.

*   **`radius="150"`**: The light extends to a radius of 150 units.
*   **`r="40"`, `g="18"`, `b="220"`**: The light color is a purplish-blue.

### Audio Components (`<AudioComponent>`, `<AudioLoopComponent>`)

Handle the sound effects associated with the projectile.

*   **`file="data/audio/Desktop/projectiles.bank"`**: Specifies the audio bank.
*   **`event_root="projectiles/magic"`**: Sets the root event for general projectile sounds.
*   **`event_name="projectiles/magic_orb/loop"`**: Defines a looping sound for the orb.

### Hit Effect Component (`<HitEffectComponent>`)

Determines the special effect applied when the projectile hits something.

*   **`effect_hit="LOAD_UNIQUE_CHILD_ENTITY"`**: Loads a unique child entity as the hit effect.
*   **`value_string="data/entities/misc/effect_twitchy.xml"`**: The specific entity to load is `effect_twitchy.xml`.

### Variable Storage Component (`<VariableStorageComponent>`)

Stores a variable within the entity.

*   **`name="projectile_file"`**: The name of the variable.
*   **`value_string="data/entities/projectiles/orb_twitchy.xml"`**: The value stored is the path to this projectile's definition file.
