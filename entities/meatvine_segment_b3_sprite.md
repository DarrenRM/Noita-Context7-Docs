# Meatvine Segment B3 Sprite

This document describes the sprite component for a specific segment of the Meatvine entity in Noita.

## SpriteComponent

This component defines the visual representation of the entity.

### Key Attributes:

*   **image_file**: `data/entities/verlet_chains/meatvine/vine_parts/vine_b_3.xml` - Specifies the XML file containing the sprite's visual data.
*   **offset_x**: `0` - The horizontal offset of the sprite from the entity's origin.
*   **offset_y**: `1.5` - The vertical offset of the sprite from the entity's origin.
*   **update_transform**: `0` - Indicates whether the sprite's transform (position, rotation, scale) should be updated automatically. `0` means it's static relative to the entity's transform.
*   **update_transform_rotation**: `0` - Indicates whether the sprite's rotation should be updated automatically. `0` means it's static relative to the entity's rotation.
