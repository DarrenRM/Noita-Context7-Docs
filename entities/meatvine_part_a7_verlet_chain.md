# Meatvine Part A7 Verlet Chain

This entity defines a specific part of the meatvine's verlet chain, responsible for its visual representation.

## Sprite Component

The `SpriteComponent` handles the visual aspects of this entity.

### Key Attributes:

*   **`image_file`**: Specifies the image file used for the sprite.
    *   `data/entities/verlet_chains/meatvine/vine_parts/vine_a_7.xml`
*   **`offset_x`**: Horizontal offset for the sprite.
    *   `0`
*   **`offset_y`**: Vertical offset for the sprite.
    *   `1.5`
*   **`update_transform`**: Determines if the sprite's transform should be updated.
    *   `0` (Disabled)
*   **`update_transform_rotation`**: Determines if the sprite's rotation should be updated.
    *   `0` (Disabled)
