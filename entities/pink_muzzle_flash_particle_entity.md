# Pink Muzzle Flash Particle Entity

This entity defines a visual effect for a muzzle flash, specifically a pink variant. It utilizes a sprite animation and a particle emitter to create the visual and particle effects associated with a weapon firing.

## Key Components

### SpriteComponent

This component handles the visual representation of the muzzle flash.
