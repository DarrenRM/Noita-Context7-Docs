# Root Verlet 5 Entity

This document describes the `root_verlet_5.xml` entity, which is part of the rootverlet chain in Noita.

## Sprite Component

The `SpriteComponent` defines the visual representation of the entity.

### Key Attributes:

*   **`image_file`**: Specifies the sprite image to be used.
    *   Value: `"data/entities/verlet_chains/root/root_parts/root_5.xml"`
*   **`offset_x`**: Horizontal offset for the sprite.
    *   Value: `"0"`
*   **`offset_y`**: Vertical offset for the sprite.
    *   Value: `"3"`
*   **`update_transform`**: Controls whether the sprite's transform is updated.
    *   Value: `"0"` (Disabled)
*   **`update_transform_rotation`**: Controls whether the sprite's rotation is updated.
    *   Value: `"0"` (Disabled)
