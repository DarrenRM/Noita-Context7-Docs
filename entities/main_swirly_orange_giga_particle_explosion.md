# Main Swirly Orange Giga Particle Explosion

This entity defines a large, swirly orange particle explosion effect in Noita. It primarily functions by loading other pre-defined particle entities to construct the visual appearance of the explosion.

## Key Components

The core functionality of this entity is driven by `LoadEntitiesComponent`, which spawns other entities at specific times or conditions.

### LoadEntitiesComponent

This component is responsible for instantiating other entity files.
