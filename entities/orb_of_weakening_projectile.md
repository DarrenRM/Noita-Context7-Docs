# Orb of Weakening Projectile

This document details the configuration for the "Orb of Weakening" projectile entity in Noita, designed for AI-assisted modding.

## Core Entity Configuration

The projectile is based on the `base_projectile.xml` entity, inheriting fundamental projectile behaviors.

```xml
<Entity 
  name="$projectile_default" 
   >

	<Base file="data/entities/base_projectile.xml" >
		<VelocityComponent
    		gravity_y="0"
			air_friction="-0.1"
			mass="0.05"
			>
    	</VelocityComponent> 
	</Base>
    
    <!-- Other components follow -->
</Entity>
```

### Key Attributes:

*   **`gravity_y="0"`**: The projectile is not affected by gravity, allowing for straight-line travel.
*   **`air_friction="-0.1"`**: A slight negative air friction might be intended to maintain velocity or counteract other forces, though typically friction is positive.
*   **`mass="0.05"`**: A very low mass, indicating it's lightweight.

## Projectile Behavior (`ProjectileComponent`)

This component defines the projectile's movement, lifespan, and death behaviors.

```xml
  <ProjectileComponent 
    _enabled="1" 
	lob_min="0.8"
  	lob_max="1.0"
  	speed_min="100"
  	speed_max="120"
    die_on_low_velocity="0"
    on_death_explode="1"
    on_death_gfx_leave_sprite="0" 
    on_lifetime_out_explode="1"
	explosion_dont_damage_shooter="1"
    damage="0.3"
    on_collision_die="1"
    lifetime="150" >
    
    <!-- Explosion configuration follows -->
  </ProjectileComponent>
```

### Key Attributes:

*   **`lob_min="0.8"`, `lob_max="1.0"`**: Controls the projectile's lobbing behavior, suggesting a slight arc.
*   **`speed_min="100"`, `speed_max="120"`**: Defines the projectile's speed range.
*   **`die_on_low_velocity="0"`**: The projectile will not die simply due to low velocity.
*   **`on_death_explode="1"`**: The projectile explodes upon death.
*   **`on_lifetime_out_explode="1"`**: The projectile also explodes when its lifetime expires.
*   **`explosion_dont_damage_shooter="1"`**: The explosion will not harm the entity that fired it.
*   **`damage="0.3"`**: The base damage dealt by the projectile on collision.
*   **`on_collision_die="1"`**: The projectile dies upon colliding with something.
*   **`lifetime="150"`**: The projectile exists for 150 frames before expiring.

### Explosion Configuration (`config_explosion`)

Defines the visual and physical effects of the projectile's explosion.

```xml
    <config_explosion
      never_cache="1" 
      camera_shake="0.5" 
      explosion_radius="1" 
      explosion_sprite="data/particles/explosion_016_electric.xml" 
      explosion_sprite_lifetime="0.0" 
      create_cell_probability="0" 
      create_cell_material="spark_blue" 
	  ray_energy="10000"
      hole_destroy_liquid="1" 
      hole_enabled="1" 
	  damage = "0.1"
      hole_image="data/temp/explosion_hole.png"
	  explosion_sprite_emissive="1"
	  explosion_sprite_additive="1"
      particle_effect="0" 
      damage_mortals="1"
	  physics_explosion_power.min="0"
      physics_explosion_power.max="0"
	  physics_throw_enabled="0"
      shake_vegetation="1"  
      sparks_enabled="1" 
	  spark_material="spark_white"
      sparks_count_max="16" 
      sparks_count_min="12"
      light_fade_time="1.2" 
	  light_r="180"
	  light_g="180"
      light_b="180"
      stains_enabled="1" 
      stains_image="data/temp/explosion_stain.png" >
    </config_explosion>
```

#### Key Explosion Attributes:

*   **`camera_shake="0.5"`**: The explosion causes moderate camera shake.
*   **`explosion_radius="1"`**: The radius of the explosion effect.
*   **`explosion_sprite="data/particles/explosion_016_electric.xml"`**: Specifies the visual particle effect for the explosion.
*   **`ray_energy="10000"`**: High energy for potential ray interactions.
*   **`hole_destroy_liquid="1"`, `hole_enabled="1"`**: The explosion creates holes and can destroy liquids.
*   **`damage="0.1"`**: The damage dealt by the explosion itself.
*   **`damage_mortals="1"`**: The explosion damages living entities.
*   **`sparks_enabled="1"`, `spark_material="spark_white"`**: Sparks are generated by the explosion.
*   **`light_r="180"`, `light_g="180"`, `light_b="180"`**: The explosion emits white light.

## Visuals (`SpriteComponent`)

Defines the projectile's appearance.

```xml
  <SpriteComponent 
    _enabled="1" 
    alpha="1" 
    image_file="data/projectiles_gfx/orb_grey.xml" 
    rect_animation="fireball" 
	emissive="1"
    additive="1"
    offset_x="5" 
    offset_y="5" 
	>
  </SpriteComponent>
```

### Key Attributes:

*   **`image_file="data/projectiles_gfx/orb_grey.xml"`**: The sprite image used for the projectile.
*   **`rect_animation="fireball"`**: Specifies a rectangle animation sequence.
*   **`emissive="1"`, `additive="1"`**: The sprite is emissive and uses additive blending for a glowing effect.
*   **`offset_x="5"`, `offset_y="5"`**: Offsets the sprite's position.

## Particle Emitters (`SpriteParticleEmitterComponent`)

Generates additional visual particles around the projectile.

```xml
	<SpriteParticleEmitterComponent
		sprite_file="data/particles/spark_electric.xml"
		delay="0"
		lifetime="0"
		color.r="0" color.g="1" color.b="1" color.a="1"
		gravity.y="10"
		emission_interval_min_frames="3"
		emission_interval_max_frames="4"
		count_min="1" count_max="2"
		randomize_rotation.min="-3.1415"
		randomize_rotation.max="3.1415"
		randomize_position.min_x="-8"
		randomize_position.max_x="8"
		randomize_position.min_y="-8"
		randomize_position.max_y="8"
		>
	</SpriteParticleEmitterComponent>
```

### Key Attributes:

*   **`sprite_file="data/particles/spark_electric.xml"`**: The particle sprite used.
*   **`color.r="0" color.g="1" color.b="1" color.a="1"`**: Particles are cyan.
*   **`gravity.y="10"`**: Particles are affected by gravity.
*   **`emission_interval_min_frames="3"`, `emission_interval_max_frames="4"`**: Controls the rate of particle emission.
*   **`count_min="1"`, `count_max="2"`**: The number of particles emitted per interval.
*   **`randomize_position`**: Particles are emitted within a small random area around the projectile.

## Homing Behavior (`HomingComponent`)

Allows the projectile to track targets.

```xml
	<HomingComponent
		target_tag="prey"
		homing_targeting_coeff="9"
		detect_distance="350"
		homing_velocity_multiplier="1.0"
		>
	</HomingComponent>
```

### Key Attributes:

*   **`target_tag="prey"`**: The projectile targets entities with the "prey" tag.
*   **`homing_targeting_coeff="9"`**: The strength of the homing behavior.
*   **`detect_distance="350"`**: The distance at which the projectile can detect targets.

### Audio Components

Two audio components are present for projectile sounds: a general event and a looping sound.

```xml
	<AudioComponent
		file="data/audio/Desktop/projectiles.bank"
		event_root="projectiles/magic" >
	</AudioComponent>
	
	<AudioLoopComponent
		file="data/audio/Desktop/projectiles.bank"
		event_name="projectiles/magic_orb/loop"
		auto_play="1">
	</AudioLoopComponent>
```

### Hit Effect Component

Applies a specific effect upon hitting a target.

```xml
	<HitEffectComponent 
        effect_hit="LOAD_UNIQUE_CHILD_ENTITY"
        value_string="data/entities/misc/effect_weaken.xml" >
	</HitEffectComponent >
```

*   **`value_string="data/entities/misc/effect_weaken.xml"`**: Loads the "effect_weaken.xml" entity, which applies the weakening debuff.

### Variable Storage Component

Stores the projectile's own file path.

```xml
	<VariableStorageComponent

		>
	</VariableStorageComponent>
```
