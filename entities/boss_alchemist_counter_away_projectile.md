# Boss Alchemist Counter Away Projectile

This document describes the configuration for a projectile entity used by the Boss Alchemist in Noita, specifically designed to counter incoming projectiles.

## Key Components and Attributes

This entity primarily focuses on its hitbox and its interaction with damage.

### HitboxComponent

This component defines the physical presence and interaction properties of the projectile.

| Attribute           | Value   | Description                                                                 |
| :------------------ | :------ | :-------------------------------------------------------------------------- |
| `damage_multiplier` | `1.0`   | The multiplier applied to any damage this entity might deal.                |
