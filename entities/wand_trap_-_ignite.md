# Wand Trap - Ignite

This entity defines a "wand trap" that triggers an ignition effect.

## Key Components

### Entity
- **tags**: `wand_trap` - Identifies this entity as a wand trap.

### VariableStorageComponent
- **name**: `entity_file`
- **value_string**: `data/entities/misc/fire_5_frames.xml` - Specifies the entity to spawn when the trap is triggered, in this case, a fire effect.
