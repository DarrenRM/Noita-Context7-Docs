# Sampo Sprite

This documentation describes the sprite configuration for the "Sampo" entity in Noita.

## Sprite

The `<Sprite>` element defines the visual representation of the entity.

### Key Attributes:

*   **filename**: Specifies the path to the sprite image file.
    *   `data/entities/animals/boss_centipede/sampo.png`
*   **offset\_x**: The horizontal offset of the sprite from its origin.
    *   `12`
*   **offset\_y**: The vertical offset of the sprite from its origin.
    *   `12`
