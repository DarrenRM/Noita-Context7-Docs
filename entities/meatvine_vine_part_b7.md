# Meatvine Vine Part B7

This entity defines a specific visual part of the meatvine's verlet chain.

## SpriteComponent

This component handles the visual representation of the entity.

### Key Attributes:

*   `image_file`: Specifies the image file used for the sprite. In this case, it points to `data/entities/verlet_chains/meatvine/vine_parts/vine_b_7.xml`.
*   `offset_x`: Horizontal offset for the sprite.
*   `offset_y`: Vertical offset for the sprite.
*   `update_transform`: Controls whether the sprite's transform is updated.
*   `update_transform_rotation`: Controls whether the sprite's rotation is updated.
