# Dark Blue Vine Part A1

This entity defines a single segment of a dark blue vine, specifically the "A1" variant. It's a visual component that will be rendered in the game.

## Key Attributes

### SpriteComponent
This component handles the visual representation of the vine segment.

*   **image_file**: `data/entities/verlet_chains/vines/vine_parts_blue/vine_dark_a_1.xml` - Specifies the image file used for this sprite.
*   **offset_x**: `0` - No horizontal offset for the sprite.
*   **offset_y**: `1.5` - A slight vertical offset for the sprite.
*   **update_transform**: `0` - The sprite's transform (position, rotation, scale) is not updated by the game engine's default transform system.
*   **update_transform_rotation**: `0` - The sprite's rotation is not updated by the game engine's default transform system.
