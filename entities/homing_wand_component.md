# Homing Wand Component

This documentation describes the `HomingComponent` used in Noita, specifically for entities that exhibit homing behavior.

## Purpose

The `HomingComponent` enables an entity to actively seek out and target specific other entities. This is commonly used for projectiles or effects that need to track a particular target.
