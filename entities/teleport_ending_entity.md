# Teleport Ending Entity

This entity represents the destination point of a teleportation sequence in Noita. It's activated when specific magical liquids are present beneath it and triggers a teleportation event for any entities within its hitbox.

## Key Components

### TeleportComponent
This is the core component that defines the teleportation behavior.
