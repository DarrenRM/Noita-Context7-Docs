# Meatvine Segment B4 Verlet Chain

This entity defines a segment of the Meatvine's verlet chain, specifically `vine_b_4`. It primarily consists of a sprite component.

## Sprite Component

This component defines the visual representation of the Meatvine segment.

### Key Attributes:

*   **`image_file`**: `data/entities/verlet_chains/meatvine/vine_parts/vine_b_4.xml` - Specifies the image file used for this segment.
*   **`offset_x`**: `0` - The horizontal offset of the sprite.
*   **`offset_y`**: `1.5` - The vertical offset of the sprite.
*   **`update_transform`**: `0` - Indicates that the sprite's transform (position, scale, rotation) is not updated dynamically.
*   **`update_transform_rotation`**: `0` - Indicates that the sprite's rotation is not updated dynamically.
