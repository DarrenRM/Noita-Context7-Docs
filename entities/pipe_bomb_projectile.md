# Pipe Bomb Projectile

This document details the configuration of the "pipe_bomb" projectile entity in Noita, useful for AI-assisted modding.

## Core Entity

The `pipe_bomb` is a projectile entity with several key components that define its behavior, appearance, and effects.

```xml
<Entity tags="pipe_bomb,hittable,projectile_player" name="$projectile_default">
	<Base file="data/entities/base_projectile.xml" >
		<VelocityComponent
			mass="0.06"
			>
		</VelocityComponent>
	</Base>
    <!-- ... other components ... -->
</Entity>
```

## ProjectileComponent

This is the primary component defining the projectile's movement, interaction, and death effects.

### Key Attributes:

| Attribute                 | Description                                                                                             |
| :------------------------ | :------------------------------------------------------------------------------------------------------ |
| `lob_min`, `lob_max`      | Controls the minimum and maximum lob angle (vertical deviation) of the projectile.                      |
| `speed_min`, `speed_max`  | Sets the range for the projectile's initial speed.                                                      |
| `friction`                | Determines how quickly the projectile loses velocity due to air resistance.                             |
| `direction_random_rad`    | Adds a small random deviation to the projectile's initial direction.                                    |
| `on_death_explode`        | If `1`, the projectile explodes upon death (e.g., collision, lifetime out).                             |
| `on_collision_die`        | If `1`, the projectile dies upon collision with an entity.                                              |
| `lifetime`                | The duration the projectile exists before dying. `-1` means it doesn't expire naturally.                |
| `damage`                  | Base damage dealt by the projectile itself (before explosion).                                          |
| `bounces_left`            | The number of times the projectile can bounce off surfaces before dying.                                |
| `camera_shake_when_shot`  | The intensity of camera shake when this projectile is fired.                                            |
| `physics_impulse_coeff`   | Affects the force applied to entities when the projectile collides or explodes.                         |

### `config_explosion` Sub-element:

This nested element defines the properties of the explosion that occurs when `on_death_explode` is true.

| Attribute                 | Description                                                                                             |
| :------------------------ | :------------------------------------------------------------------------------------------------------ |
| `camera_shake`            | Intensity of camera shake caused by the explosion.                                                      |
| `explosion_radius`        | The radius of the explosion's effect.                                                                   |
| `explosion_sprite`        | The particle effect sprite used for the explosion visual.                                               |
| `load_this_entity`        | An entity to load at the explosion's center (e.g., for lingering effects).                              |
| `ray_energy`              | The energy of the explosion's destructive rays.                                                         |
| `damage`                  | The damage dealt by the explosion.                                                                      |
| `physics_explosion_power` | Controls the force of the physics-based explosion push.                                                 |
| `sparks_count_min`, `sparks_count_max` | The range for the number of sparks generated by the explosion.                                |
| `spark_material`          | The material type of the sparks generated.                                                              |
| `audio_event_name`        | The sound event to play when the explosion occurs.                                                      |

## HitboxComponent

Defines the physical boundaries of the projectile for collision detection.

| Attribute     | Description                                                               |
| :------------ | :------------------------------------------------------------------------ |
| `aabb_min_x`  | Minimum X-coordinate of the Axis-Aligned Bounding Box.                    |
| `aabb_max_x`  | Maximum X-coordinate of the Axis-Aligned Bounding Box.                    |
| `aabb_min_y`  | Minimum Y-coordinate of the Axis-Aligned Bounding Box.                    |
| `aabb_max_y`  | Maximum Y-coordinate of the Axis-Aligned Bounding Box.                    |

## DamageModelComponent

Configures how the projectile interacts with damage and materials.

| Attribute             | Description                                                                                             |
| :-------------------- | :------------------------------------------------------------------------------------------------------ |
| `hp`                  | The hit points of the projectile. A low value means it's easily destroyed.                              |
| `fire_damage_amount`  | The amount of fire damage the projectile deals.                                                         |
| `fire_probability_of_ignition` | The chance the projectile will ignite entities it hits with fire damage.                          |

## SpriteComponent

Determines the visual representation of the projectile.

| Attribute     | Description                                                               |
| :------------ | :------------------------------------------------------------------------ |
| `image_file`  | The path to the sprite image file used for the projectile's appearance.   |

## LightComponent

Adds a light source to the projectile, affecting the environment's illumination.

| Attribute | Description                                                               |
| :-------- | :------------------------------------------------------------------------ |
| `radius`  | The radius of the light emitted by the projectile.                        |
| `r`, `g`, `b` | The RGB color values of the emitted light.                                |

## AudioComponent

Defines the sound effects associated with the projectile.

| Attribute      | Description                                                               |
| :------------- | :------------------------------------------------------------------------ |
| `file`         | The path to the audio bank file.                                          |
| `event_root`   | The root event name for player projectile sounds.                         |

## VariableStorageComponent

Stores custom variables for the entity.

| Attribute    | Description                                                               |
| :----------- | :------------------------------------------------------------------------ |
| `value_string` | Stores the path to the projectile's own XML file, useful for referencing. |
