# Radioactive Waste Projectile

This entity defines a projectile that converts nearby materials into radioactive substances. It's designed to spread contamination in its vicinity.

## Key Components

### MagicConvertMaterialComponent

This component is responsible for transforming existing materials into radioactive ones. The entity has multiple instances of this component, each targeting different material tags.

### LifetimeComponent

This component dictates how long the projectile exists before disappearing.

| Attribute | Description                               |
| :-------- | :---------------------------------------- |
| `lifetime`| The duration in frames the projectile lasts. |

## Usage in Modding

This entity can be used as a base for creating custom projectiles that spread specific material effects. By modifying the `from_material_tag` and `to_material` attributes within the `MagicConvertMaterialComponent`, you can achieve various material transformation effects. The `radius` attribute controls the area of effect for the material conversion. The `speed` attribute in `ProjectileComponent` is set to `0`, indicating this projectile does not move on its own and likely relies on being spawned by another entity or effect.
