# Rocket Tier 2 Projectile

This document details the configuration for the "Rocket Tier 2" projectile entity in Noita, designed for AI-assisted modding.

## Core Entity Configuration

The projectile is defined as an entity with the following primary tags: `projectile_player` and `rocket`.

```xml
<Entity name="$projectile_default" tags="projectile_player,rocket" >
```

## Base Projectile Properties

Inherits from `data/entities/base_projectile.xml`.

### Velocity Component

Defines the projectile's movement characteristics.

| Attribute       | Value   | Description                                     |
| :-------------- | :------ | :---------------------------------------------- |
| `gravity_y`     | `50`    | Downward acceleration due to gravity.           |
| `air_friction`  | `-5.0`  | Resistance to movement in the air.              |
| `mass`          | `0.05`  | The mass of the projectile.                     |

## Projectile Component

This is the primary component defining the projectile's behavior and effects.

### Key Attributes

| Attribute                     | Value   | Description                                                                 |
| :---------------------------- | :------ | :-------------------------------------------------------------------------- |
| `lob_min`                     | `0.8`   | Minimum lobbing angle (deviation from straight trajectory).                 |
| `lob_max`                     | `1.0`   | Maximum lobbing angle.                                                      |
| `speed_min`                   | `70`    | Minimum projectile speed.                                                   |
| `speed_max`                   | `100`   | Maximum projectile speed.                                                   |
| `on_death_explode`            | `1`     | Causes an explosion upon death (e.g., hitting a target or expiring).        |
| `on_lifetime_out_explode`     | `1`     | Causes an explosion when the projectile's lifetime expires.                 |
| `on_collision_die`            | `1`     | The projectile dies upon colliding with something.                          |
| `lifetime`                    | `360`   | Base lifetime of the projectile in frames.                                  |
| `lifetime_randomness`         | `7`     | Random variation added to the projectile's lifetime.                        |
| `damage`                      | `4`     | Base damage dealt by the projectile.                                        |
| `camera_shake_when_shot`      | `15.0`  | Intensity of camera shake when the projectile is fired.                     |
| `knockback_force`             | `3.0`   | Force applied to entities that are hit by the projectile.                   |
| `physics_impulse_coeff`       | `4000`  | Coefficient for physics impulse, affecting how it interacts with physics.   |

### Explosion Configuration (`config_explosion`)

Details of the explosion that occurs when `on_death_explode` or `on_lifetime_out_explode` is triggered.

| Attribute                 | Value   | Description                                                                 |
| :------------------------ | :------ | :-------------------------------------------------------------------------- |
| `camera_shake`            | `70`    | Intensity of camera shake during the explosion.                             |
| `explosion_radius`        | `32`    | The radius of the explosion effect.                                         |
| `explosion_sprite`        | `data/particles/explosion_032_slime.xml` | Sprite used for the explosion visual.                                       |
| `load_this_entity`        | `data/entities/particles/particle_explosion/main_swirly_green.xml` | Entity to load for the main explosion effect.                               |
| `create_cell_probability` | `20`    | Probability (in %) of creating cell-like effects from the explosion.        |
| `hole_enabled`            | `1`     | Enables the creation of holes in terrain by the explosion.                  |
| `ray_energy`              | `5000000` | Energy value for raycasting effects of the explosion.                       |
| `damage`                  | `4.7`   | Damage dealt by the explosion itself.                                       |
| `physics_explosion_power.min` | `1.0` | Minimum power of the physics-based explosion force.                         |
| `physics_explosion_power.max` | `1.4` | Maximum power of the physics-based explosion force.                         |
| `sparks_count_min`        | `20`    | Minimum number of sparks generated by the explosion.                        |
| `sparks_count_max`        | `40`    | Maximum number of sparks generated by the explosion.                        |
| `spark_material`          | `spark_green` | The material type of the sparks.                                            |
| `stains_enabled`          | `1`     | Enables the creation of impact stains.                                      |
| `stains_radius`           | `15`    | Radius of the impact stains.                                                |
| `audio_event_name`        | `explosions/magic_rocket_mid` | The audio event triggered by the explosion.                                 |

## Particle Emitter Components

Multiple emitters are used to create various visual effects, primarily sparks and smoke.

### Smoke Emitter

| Attribute               | Value   | Description                               |
| :---------------------- | :------ | :---------------------------------------- |
| `emitted_material_name` | `smoke` | Material of the emitted particles.        |
| `count_min`             | `5`     | Minimum number of particles emitted.      |
| `count_max`             | `5`     | Maximum number of particles emitted.      |
| `lifetime_min`          | `0.1`   | Minimum lifetime of emitted particles.    |
| `lifetime_max`          | `0.3`   | Maximum lifetime of emitted particles.    |
| `create_real_particles` | `1`     | Whether to create actual game particles.  |
| `is_emitting`           | `1`     | Enables the emitter.                      |

### Spark Emitters (Multiple)

These emitters generate green sparks with varying gravity, lifetime, and airflow to create a dynamic trail effect.

| Attribute               | Value   | Description                               |
| :---------------------- | :------ | :---------------------------------------- |
| `emitted_material_name` | `spark_green` | Material of the emitted particles.        |
| `count_min`             | `1`     | Minimum number of particles emitted.      |
| `count_max`             | `8`     | Maximum number of particles emitted.      |
| `is_trail`              | `1`     | Marks the particles as a trail effect.    |
| `render_on_grid`        | `1`     | Renders particles on the game grid.       |
| `fade_based_on_lifetime`| `1`     | Particles fade out as their lifetime ends.|
| `emit_cosmetic_particles`| `1`    | Emits cosmetic particles (visual only).   |
| `is_emitting`           | `1`     | Enables the emitter.                      |

### Sparse Spark Emitter

A less frequent emitter for distinct spark effects.

| Attribute               | Value   | Description                               |
| :---------------------- | :------ | :---------------------------------------- |
| `emitted_material_name` | `spark_green` | Material of the emitted particles.        |
| `count_min`             | `1`     | Minimum number of particles emitted.      |
| `count_max`             | `1`     | Maximum number of particles emitted.      |
| `lifetime_min`          | `1.4`   | Minimum lifetime of emitted particles.    |
| `lifetime_max`          | `1.5`   | Maximum lifetime of emitted particles.    |
| `emission_interval_min_frames` | `8` | Minimum frames between emissions.         |
| `emission_interval_max_frames` | `12`| Maximum frames between emissions.         |
