# Orange Firework Projectile

This document details the configuration of the "Orange Firework" projectile entity in Noita, designed for AI-assisted modding.

## Entity Definition

The core of this projectile is based on `data/entities/base_projectile.xml`.

```xml
<Entity name="$projectile_default">
    <Base file="data/entities/base_projectile.xml">
        <VelocityComponent
            gravity_y="50"
            air_friction="-5.0"
            mass="0.05"
        >
        </VelocityComponent>
    </Base>
    <!-- ... other components ... -->
</Entity>
```

### Key Attributes:

*   **`gravity_y`**: Controls the downward acceleration of the projectile.
*   **`air_friction`**: Affects how quickly the projectile loses speed due to air resistance.
*   **`mass`**: Determines the projectile's inertia.

## Projectile Component

This component defines the projectile's behavior, trajectory, and effects.

```xml
<ProjectileComponent
    _enabled="1"
    lob_min="0.8"
    lob_max="1.0"
    speed_min="70"
    speed_max="80"
    direction_random_rad="0.0"
    on_death_explode="1"
    on_death_gfx_leave_sprite="0"
    on_lifetime_out_explode="1"
    explosion_dont_damage_shooter="0"
    on_collision_die="1"
    lifetime="30"
    damage="1"
    velocity_sets_scale="1"
    lifetime_randomness="7"
    ragdoll_force_multiplier="0.04"
    hit_particle_force_multiplier="5.5"
    camera_shake_when_shot="15.0"
    muzzle_flash_file="data/entities/particles/muzzle_flashes/muzzle_flash_launcher.xml"
    shoot_light_flash_r="255"
    shoot_light_flash_g="210"
    shoot_light_flash_b="40"
    shoot_light_flash_radius="90"
    knockback_force="3.0"
>
    <!-- ... config_explosion ... -->
</ProjectileComponent>
```

### Key Attributes:

*   **`lob_min`/`lob_max`**: Controls the minimum and maximum arc of the projectile.
*   **`speed_min`/`speed_max`**: Defines the projectile's initial speed range.
*   **`on_death_explode`**: If `1`, the projectile explodes upon death (e.g., lifetime out or collision).
*   **`on_lifetime_out_explode`**: If `1`, the projectile explodes when its lifetime expires.
*   **`on_collision_die`**: If `1`, the projectile is destroyed upon collision.
*   **`lifetime`**: The duration in frames before the projectile expires.
*   **`damage`**: Base damage dealt by the projectile.
*   **`lifetime_randomness`**: Adds variation to the projectile's lifetime.
*   **`camera_shake_when_shot`**: The intensity of camera shake when the projectile is fired.
*   **`shoot_light_flash_r`/`g`/`b`/`radius`**: Defines the color and radius of the light flash when the projectile is shot.
*   **`knockback_force`**: The force applied to entities upon collision.

#### `config_explosion` Sub-element:

This defines the properties of the explosion that occurs when the projectile dies or collides.

```xml
<config_explosion
    never_cache="1"
    camera_shake="20"
    explosion_radius="15"
    load_this_entity="data/entities/particles/fireworks/firework_orange.xml"
    create_cell_probability="10"
    cell_explosion_power_ragdoll_coeff="0.5"
    hole_enabled="1"
    ray_energy="2000000"
    damage="1.2"
    particle_effect="1"
    damage_mortals="1"
    physics_explosion_power.min="1.0"
    physics_explosion_power.max="1.4"
    physics_throw_enabled="1"
    shake_vegetation="1"
    sparks_count_max="20"
    sparks_count_min="7"
    sparks_enabled="1"
    stains_enabled="1"
    stains_radius="15"
    audio_event_name="explosions/glitter"
>
</config_explosion>
```

**Key Attributes:**

*   **`camera_shake`**: Intensity of camera shake from the explosion.
*   **`explosion_radius`**: The radius of the explosion's effect.
*   **`load_this_entity`**: Specifies the entity to load for the explosion's visual effects.
*   **`create_cell_probability`**: Chance to create cell-like explosions.
*   **`hole_enabled`**: If `1`, the explosion creates holes in terrain.
*   **`ray_energy`**: The energy of the explosion's damaging rays.
*   **`damage`**: Damage dealt by the explosion.
*   **`physics_explosion_power.min`/`max`**: Minimum and maximum force of the physics-based explosion.
*   **`sparks_count_min`/`max`**: Number of sparks generated by the explosion.
*   **`audio_event_name`**: The sound event triggered by the explosion.

## Light Component

This component adds a light source to the projectile.

```xml
<LightComponent
    _enabled="1"
    r="255"
    g="100"
    b="10"
    radius="150"
>
</LightComponent>
```

### Key Attributes:

*   **`r`/`g`/`b`**: RGB color values for the light.
*   **`radius`**: The radius of the light emitted.

## Particle Emitter Components

Multiple `ParticleEmitterComponent` instances are used to generate various visual effects, primarily sparks and smoke.

### Smoke Emitter:

```xml
<ParticleEmitterComponent
    emitted_material_name="smoke"
    offset.x="0"
    offset.y="0"
    x_pos_offset_min="-1"
    y_pos_offset_min="-1"
    x_pos_offset_max="1"
    y_pos_offset_max="1"
    x_vel_min="-10"
    x_vel_max="10"
    y_vel_min="-10"
    y_vel_max="10"
    count_min="5"
    count_max="5"
    lifetime_min="0.1"
    lifetime_max="0.3"
    create_real_particles="1"
    emit_cosmetic_particles="0"
    emission_interval_min_frames="1"
    emission_interval_max_frames="1"
    is_emitting="1"
>
</ParticleEmitterComponent>
```

**Key Attributes:**

*   **`emitted_material_name`**: The type of particle to emit ("smoke").
*   **`count_min`/`max`**: Number of particles emitted per interval.
*   **`lifetime_min`/`max`**: Duration of emitted particles.

    trail_gap="0.5"

    fade_based_on_lifetime="1"
    create_real_particles="0"
    emit_cosmetic_particles="1"
    emission_interval_min_frames="1"
    emission_interval_max_frames="2"

>
</ParticleEmitterComponent>

**Key Attributes:**

*   **`emitted_material_name`**: The type of particle to emit ("spark").
*   **`gravity.y`**: Controls the vertical direction and strength of gravity on the sparks.
*   **`is_trail`**: If `1`, particles leave a trail.
*   **`emit_cosmetic_particles`**: If `1`, emits particles that are purely visual and don't interact with physics.

    airflow_force="0.5"
    airflow_time="0.5"

**Key Attributes:**

*   **`airflow_force`/`time`/`scale`**: Parameters for air resistance effects on particles.

## Audio Component

Defines the sound associated with the projectile.

```xml
<AudioComponent
    file="data/audio/Desktop/projectiles.bank"
    event_root="player_projectiles/bullet_rocket">
</AudioComponent>
```

### Key Attributes:

*   **`file`**: The audio bank file containing the sound.
*   **`event_root`**: The specific sound event to trigger.
