# Meatvine Part A2 Verlet Chain

This entity defines a segment of the Meatvine's verlet chain, specifically the 'A2' variant. It primarily consists of a sprite component for visual representation.

## SpriteComponent

This component handles the visual appearance of the entity.

### Key Attributes:

*   **image_file**: `data/entities/verlet_chains/meatvine/vine_parts/vine_a_2.xml` - Specifies the image file used for the sprite.
*   **offset_x**: `0` - The horizontal offset of the sprite.
*   **offset_y**: `1.5` - The vertical offset of the sprite.
*   **update_transform**: `0` - Disables automatic transform updates for the sprite.
*   **update_transform_rotation**: `0` - Disables automatic rotation updates for the sprite.
