# Flammer Animal Entity

This document describes the `flamer.xml` entity, representing a Flammer creature in Noita.

## Base Entity

The Flammer inherits its base properties from `data/entities/animals/flamer.xml`.

## Key Components

### DamageModelComponent

This component defines the Flammer's health and damage-related attributes.

| Attribute | Value | Description |
|---|---|---|
| `hp` | 4 | Current health points. |
| `max_hp` | 4 | Maximum health points. |
