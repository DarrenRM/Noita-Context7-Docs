# Monk Ragdoll Sprites

This documentation describes the sprite assets used for the Monk character's ragdoll in Noita. These files define the visual components of the Monk when it is in a ragdoll state.

## Sprite Files

The following PNG files represent the individual parts of the Monk's ragdoll.

| Sprite Name | File Path                     | Description                               |
|-------------|-------------------------------|-------------------------------------------|
| torso       | `data/ragdolls/monk/torso.png`  | The main body segment of the ragdoll.     |
| head        | `data/ragdolls/monk/head.png`   | The head segment of the ragdoll.          |
| leg_l       | `data/ragdolls/monk/leg_l.png`  | The left leg segment of the ragdoll.      |
| leg_r       | `data/ragdolls/monk/leg_r.png`  | The right leg segment of the ragdoll.     |

## Usage

These sprite files are typically referenced by the game's entity or physics systems to render the Monk's ragdoll. Modders can potentially replace these sprites to alter the appearance of the Monk's ragdoll.
