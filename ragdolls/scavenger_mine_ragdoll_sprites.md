# Scavenger Mine Ragdoll Sprites

This documentation outlines the sprite assets used for the "Scavenger Mine" ragdoll in Noita. These files define the visual components of the ragdoll when it is destroyed or dismembered.

## Sprite Files

The following PNG files represent individual body parts of the Scavenger Mine ragdoll.

### Key Sprites

*   `torso.png`: The main body segment.
*   `thighs.png`: The upper leg segments.
*   `head.png`: The head segment.
*   `left_foot.png`: The left foot segment.
*   `right_foot.png`: The right foot segment.
*   `hand.png`: The hand segment (likely for both left and right, or a generic hand).

## Usage

These sprite files are referenced by the game's ragdoll system to render the visual representation of the Scavenger Mine when it breaks apart. Modders can potentially replace or modify these sprites to alter the appearance of the ragdoll.
