# Axeman Ragdoll Sprite Filenames

This document lists the sprite filenames used for the Axeman's ragdoll in Noita. These files define the visual components of the Axeman when it is in a ragdoll state.

## Sprite Components

The Axeman ragdoll is composed of several distinct sprite parts, each corresponding to a body segment or item.

### Key Sprites

*   `head.png`: The sprite for the Axeman's head.
*   `torso.png`: The sprite for the Axeman's torso.
*   `left_arm.png`: The sprite for the Axeman's left arm.
*   `left_hand.png`: The sprite for the Axeman's left hand.
*   `axe.png`: The sprite for the Axeman's axe.
*   `right_arm.png`: The sprite for the Axeman's right arm.
*   `right_hand.png`: The sprite for the Axeman's right hand.
*   `left_leg.png`: The sprite for the Axeman's left leg.
*   `left_foot.png`: The sprite for the Axeman's left foot.
*   `right_leg.png`: The sprite for the Axeman's right leg.
*   `right_foot.png`: The sprite for the Axeman's right foot.

These sprites are located within the `data/ragdolls/axeman/` directory.
