# Scavenger Heal Ragdoll Sprites

This documentation describes the sprite assets used for the "Scavenger Heal" ragdoll in Noita. These files define the visual components of the ragdoll when it is in a non-active state.

## Sprite Files

The following PNG files constitute the visual elements of the Scavenger Heal ragdoll. Each file represents a distinct body part.

### Key Sprites

*   `torso.png`: The main body segment.
*   `thighs.png`: Represents the upper leg portions.
*   `head.png`: The head sprite.
*   `left_foot.png`: The sprite for the left foot.
*   `right_foot.png`: The sprite for the right foot.
*   `hand.png`: The sprite for the hand(s).

## Usage

These sprites are typically referenced by the game's entity system to render the ragdoll when an entity is defeated or in a specific ragdoll state. The arrangement and animation of these sprites are handled by the entity's associated scripts and configuration.
