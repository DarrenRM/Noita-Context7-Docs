# Frog Ragdoll Sprites

This documentation outlines the sprite assets used for the frog ragdoll in Noita. These files define the visual components of the frog's ragdoll physics.

## Sprite Files

The following PNG files represent the individual parts of the frog's ragdoll.

| Sprite Name   | Description        | File Path                     |
|---------------|--------------------|-------------------------------|
| torso         | The main body      | `data/ragdolls/frog/torso.png`  |
| tail          | The frog's tail    | `data/ragdolls/frog/tail.png`   |
| left_foot     | The left foot      | `data/ragdolls/frog/left_foot.png` |
| right_foot    | The right foot     | `data/ragdolls/frog/right_foot.png`|

## Usage

These sprite files are referenced by the game's ragdoll system to render the visual representation of a frog when it is affected by physics. Modders can potentially replace or modify these sprites to alter the appearance of the frog ragdoll.
