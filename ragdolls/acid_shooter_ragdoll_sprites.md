# Acid Shooter Ragdoll Sprites

This documentation outlines the sprite assets used for the "Acid Shooter" entity's ragdoll. These files define the visual components of the entity when it is in a ragdoll state.

## Sprite Files

The following PNG files represent the individual sprite components for the Acid Shooter's ragdoll.

*   `torso.png`: The main body sprite.
*   `tentacle_1_a.png`, `tentacle_1_b.png`: First pair of tentacle sprites.
*   `tentacle_2_a.png`, `tentacle_2_b.png`: Second pair of tentacle sprites.
*   `tentacle_3_a.png`, `tentacle_3_b.png`: Third pair of tentacle sprites.
*   `tentacle_4_a.png`, `tentacle_4_b.png`: Fourth pair of tentacle sprites.

These sprites are typically used in conjunction with a ragdoll definition file (e.g., a `.xml` file) that specifies how these sprites are attached, animated, and interact within the game engine.
