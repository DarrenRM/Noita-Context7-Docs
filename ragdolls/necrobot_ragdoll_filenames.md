# Necrobot Ragdoll Filenames

This document outlines the image filenames used for the Necrobot's ragdoll components in Noita. These files define the visual assets for the Necrobot's dismembered body parts when it is defeated.

## Core Components

The Necrobot ragdoll is composed of several distinct image files, each representing a specific body part.

### Image Files

*   `torso.png`: The main body segment of the Necrobot.
*   `head.png`: The head of the Necrobot.
*   `hand_right.png`: The right hand of the Necrobot.
*   `hand_left.png`: The left hand of the Necrobot.
*   `foot_1.png`: One of the Necrobot's feet.
*   `foot_2.png`: The other of the Necrobot's feet.
