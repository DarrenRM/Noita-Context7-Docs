# Miner Ragdoll Sprites

This documentation describes the sprite assets used to construct the "miner" ragdoll in Noita. These files define the visual components of the miner character's body when it is in a ragdoll state.

## Sprite Components

The following image files are used to assemble the miner's ragdoll. Each file represents a distinct body part.

### Key Sprites

*   `torso.png`: The main body segment.
*   `head.png`: The miner's head.
*   `left_arm.png`: The left arm.
*   `right_arm.png`: The right arm.
*   `left_hand.png`: The left hand.
*   `right_hand.png`: The right hand.
*   `thighs.png`: The upper leg segments.
*   `left_foot.png`: The left foot.
*   `right_foot.png`: The right foot.
*   `fuse.png`: A visual element, likely related to the miner's explosive nature.

## File Structure

All sprite files for the miner ragdoll are located within the `data/ragdolls/miner/` directory.

```
data/ragdolls/miner/
├── torso.png
├── thighs.png
├── right_foot.png
├── right_arm.png
├── right_hand.png
├── fuse.png
├── left_foot.png
├── left_arm.png
├── left_hand.png
└── head.png
```
