# Wolf Ragdoll Sprites

This documentation describes the sprite filenames used for the wolf ragdoll in Noita. These files define the visual components of the wolf's ragdoll physics.

## Sprite Components

The wolf ragdoll is composed of several distinct sprite parts, each corresponding to a specific body segment.

### Key Sprite Files

*   `torso.png`: The main body sprite.
*   `head.png`: The sprite for the wolf's head.
*   `tail.png`: The sprite for the wolf's tail.
*   `right_thigh.png`: The sprite for the right upper leg.
*   `right_leg.png`: The sprite for the right lower leg.
*   `right_arm.png`: The sprite for the right upper arm.
*   `right_hand.png`: The sprite for the right hand.
*   `left_thigh.png`: The sprite for the left upper leg.
*   `left_leg.png`: The sprite for the left lower leg.
*   `left_arm.png`: The sprite for the left upper arm.
*   `left_hand.png`: The sprite for the left hand.

These PNG files are typically located within the `data/ragdolls/wolf/` directory. When used in the game, these sprites are assembled and animated to create the visual representation of the wolf's ragdoll.
