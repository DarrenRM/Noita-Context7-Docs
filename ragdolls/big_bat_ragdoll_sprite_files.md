# Big Bat Ragdoll Sprite Files

This documentation outlines the sprite files used for the "Big Bat" ragdoll in Noita. These files define the visual components of the ragdoll when it is in a non-animated state.

## Sprite Components

The following image files are used to construct the Big Bat ragdoll:

| Sprite Name | File Path                     | Description                               |
| :---------- | :---------------------------- | :---------------------------------------- |
| torso       | `data/ragdolls/bigbat/torso.png` | The main body of the Big Bat.             |
| wing1       | `data/ragdolls/bigbat/wing1.png` | The first wing of the Big Bat.            |
| wing2       | `data/ragdolls/bigbat/wing2.png` | The second wing of the Big Bat.           |

## Key Attributes

*   **File Format:** PNG images.
*   **Purpose:** Defines the visual appearance of the Big Bat ragdoll.
*   **Location:** Stored within the `data/ragdolls/bigbat/` directory.
*   **Modding Potential:** Modders can replace these sprites to alter the visual appearance of the Big Bat ragdoll.
