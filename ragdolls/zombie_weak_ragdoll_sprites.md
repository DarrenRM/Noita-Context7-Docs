# Zombie Weak Ragdoll Sprites

This documentation describes the sprite assets used for the "Zombie Weak" ragdoll in Noita. These files define the visual components of the ragdoll when it is broken apart.

## Sprite Components

The following image files define the individual parts of the zombie weak ragdoll. Each file represents a distinct body segment.

### Key Components

*   `head.png`: The head sprite.
*   `upper_torso.png`: The upper body sprite.
*   `lower_torso.png`: The lower body sprite.
*   `left_arm.png`: The left arm sprite.
*   `right_arm.png`: The right arm sprite.
*   `left_thigh.png`: The left thigh sprite.
*   `right_thigh.png`: The right thigh sprite.
*   `left_foot.png`: The left foot sprite.
*   `right_foot.png`: The right foot sprite.
*   `left_hand.png`: The left hand sprite.
*   `right_hand.png`: The right hand sprite.
*   `tail.png`: The tail sprite.

## File Structure

The sprites are located within the `data/ragdolls/zombie_weak/` directory.

```
data/ragdolls/zombie_weak/
├── head.png
├── lower_torso.png
├── left_arm.png
├── left_foot.png
├── left_hand.png
├── left_thigh.png
├── right_arm.png
├── right_foot.png
├── right_hand.png
├── right_thigh.png
├── tail.png
├── upper_torso.png
```
