# Zombie Ragdoll Sprite Filenames

This document lists the sprite filenames used for the zombie ragdoll in Noita. These files define the visual components of the zombie's dismembered body parts.

## Sprite Components

The following sprites are used to construct the zombie ragdoll:

### Key Sprites

*   `upper_torso.png`: The upper part of the zombie's torso.
*   `lower_torso.png`: The lower part of the zombie's torso.
*   `head.png`: The zombie's head.
*   `right_arm.png`: The zombie's right arm.
*   `left_arm.png`: The zombie's left arm.
*   `right_thigh.png`: The zombie's right thigh.
*   `left_thigh.png`: The zombie's left thigh.
*   `right_foot.png`: The zombie's right foot.
*   `left_foot.png`: The zombie's left foot.
*   `tail.png`: The zombie's tail.

## Usage

These `.png` files are referenced within the game's data files to define how a zombie's ragdoll is assembled when it is killed or dismembered. Each sprite represents a distinct visual piece that can be manipulated and rendered independently.
