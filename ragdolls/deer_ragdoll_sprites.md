# Deer Ragdoll Sprites

This documentation describes the sprite assets used for the deer ragdoll in Noita. These files define the visual components of the deer's body when it is in a ragdoll state.

## Sprite Components

The following image files define the individual parts of the deer's ragdoll. Each file corresponds to a specific body part and is used to animate its movement and physics.

### Key Sprite Files

*   `data/ragdolls/deer/torso.png`: The main body segment of the deer.
*   `data/ragdolls/deer/head.png`: The deer's head.
*   `data/ragdolls/deer/right_thigh.png`: The upper part of the deer's right leg.
*   `data/ragdolls/deer/right_leg.png`: The lower part of the deer's right leg.
*   `data/ragdolls/deer/right_arm.png`: The deer's right arm.
*   `data/ragdolls/deer/right_hand.png`: The deer's right hand.
*   `data/ragdolls/deer/left_thigh.png`: The upper part of the deer's left leg.
*   `data/ragdolls/deer/left_leg.png`: The lower part of the deer's left leg.
*   `data/ragdolls/deer/left_arm.png`: The deer's left arm.
*   `data/ragdolls/deer/left_hand.png`: The deer's left hand.

## Usage

These sprite files are typically referenced within Noita's entity or physics configuration files to assemble the deer ragdoll. The game engine uses these images to render the ragdoll's appearance and simulate its physical interactions.
