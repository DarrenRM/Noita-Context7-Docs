# Drone Ragdoll Filenames

This documentation outlines the filenames associated with the drone ragdoll in Noita, as found in the `data/ragdolls/drone/` directory.

## Key Files

The primary file defining the drone's ragdoll appearance is:

*   `drone.png`: This image file contains the visual assets for the drone's ragdoll.

## Usage

These files are referenced by the game engine to render the ragdoll physics and animations for the drone entity. Modders can potentially modify `drone.png` to alter the drone's appearance in its ragdoll state.
