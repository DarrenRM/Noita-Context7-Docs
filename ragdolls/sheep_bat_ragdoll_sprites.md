# Sheep Bat Ragdoll Sprites

This documentation outlines the sprite filenames used for the Sheep Bat ragdoll in Noita. These files define the visual components of the ragdoll when it is in a non-animated state.

## Sprite Components

The following files define the individual parts of the Sheep Bat ragdoll:

### Head
- `data/ragdolls/sheep_bat/head.png`

### Torso
- `data/ragdolls/sheep_bat/right_torso.png`
- `data/ragdolls/sheep_bat/left_torso.png`

### Legs
- `data/ragdolls/sheep_bat/left_frontleg.png`
- `data/ragdolls/sheep_bat/right_frontleg.png`
- `data/ragdolls/sheep_bat/left_hindleg.png`
- `data/ragdolls/sheep_bat/right_hindleg.png`

### Wings
- `data/ragdolls/sheep_bat/wing.png`

## Usage

These `.png` files are referenced by the game's ragdoll system to assemble the visual representation of the Sheep Bat when it is not actively animated. Modders can replace or modify these sprites to alter the appearance of the Sheep Bat ragdoll.
