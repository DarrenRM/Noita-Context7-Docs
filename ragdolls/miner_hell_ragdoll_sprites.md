# Miner Hell Ragdoll Sprites

This documentation describes the sprite assets used for the "Miner Hell" ragdoll in Noita. These files define the visual components of the ragdoll when it is destroyed or dismembered.

## Sprite Components

The following image files define the individual parts of the Miner Hell ragdoll.

### Key Sprites

*   `head.png`: The head sprite.
*   `torso.png`: The main body sprite.
*   `fuse.png`: Likely represents a fuse or explosive component.

### Limb Sprites

*   `left_arm.png`: The left arm sprite.
*   `right_arm.png`: The right arm sprite.
*   `left_hand.png`: The left hand sprite.
