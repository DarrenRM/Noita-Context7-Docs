# Rat Ragdoll Sprite Definitions

This documentation outlines the sprite definitions for the "rat" ragdoll in Noita, as found in `data/ragdolls/rat/filenames.txt`. These files specify the individual image components that make up the rat's ragdoll physics.

## Key Sprite Components

The rat ragdoll is composed of several distinct sprite parts, each contributing to its visual representation and physical behavior.

### Sprite Filenames

The following files define the visual assets for the rat ragdoll:

*   `head.png`: The sprite for the rat's head.
*   `torso_right.png`: The sprite for the right side of the rat's torso.
*   `torso_left.png`: The sprite for the left side of the rat's torso.
*   `tail_start.png`: The sprite for the beginning segment of the rat's tail.
*   `tail_end.png`: The sprite for the final segment of the rat's tail.

These sprites are typically loaded and used by the game engine to animate and simulate the physics of the rat entity when it is in a ragdoll state.
