# Fish Ragdoll Sprites

This documentation describes the sprite assets used for the `fish_01` ragdoll in Noita. These files define the visual components of the fish's ragdoll.

## Sprite Components

The `fish_01` ragdoll is composed of three distinct sprite images, each representing a part of the fish's body.

### Key Sprites

*   **Head:** `data/ragdolls/fish_01/head.png`
*   **Torso:** `data/ragdolls/fish_01/torso.png`
*   **Tail:** `data/ragdolls/fish_01/tail.png`

These PNG files are used by the game engine to render the ragdoll's visual representation when it is active.
