# Tank Super Ragdoll Sprites

This documentation outlines the sprite assets used for the "Tank Super" ragdoll in Noita. These files define the visual components of the ragdoll's appearance when it is in a ragdoll state.

## Sprite Files

The following `.png` files represent the individual visual elements of the Tank Super ragdoll.

### Key Sprites

*   `gun.png`: The sprite for the tank's gun component.
*   `strut.png`: The sprite for a strut or support structure.
*   `base.png`: The sprite for the main body or base of the tank.
*   `cord1.png`: The sprite for the first cord or cable element.
*   `cord2.png`: The sprite for the second cord or cable element.

These sprites are typically used in conjunction with a `.xml` definition file that dictates their positioning, rotation, and behavior within the ragdoll system.
