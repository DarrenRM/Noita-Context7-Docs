# Statue Ragdoll Sprite Filenames

This documentation outlines the sprite filenames used for the statue ragdoll in Noita. These files define the visual components of the statue when it breaks apart.

## Sprite Components

The statue ragdoll is composed of several distinct sprite parts, each corresponding to a specific body segment.

### Key Sprite Files

*   `torso.png`: The main body segment of the statue.
*   `head.png`: The head segment of the statue.
*   `right_thigh.png`: The right thigh segment.
*   `right_arm.png`: The right arm segment.
*   `right_hand.png`: The right hand segment.
*   `left_thigh.png`: The left thigh segment.
*   `left_arm.png`: The left arm segment.
*   `left_hand.png`: The left hand segment.

These files are located within the `data/ragdolls/statue/` directory. When a statue entity is destroyed, these sprites are used to create the ragdoll effect.
