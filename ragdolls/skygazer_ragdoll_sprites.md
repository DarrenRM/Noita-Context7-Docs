# Skygazer Ragdoll Sprites

This documentation describes the sprite assets used for the "skygazer" ragdoll in Noita. These files define the visual appearance of the ragdoll's torso.

## Torso Sprite

### `torso.png`

This PNG file represents the visual sprite for the skygazer's torso.

*   **File Path:** `data/ragdolls/skygazer/torso.png`
*   **Purpose:** Defines the graphical representation of the skygazer's main body segment when it becomes a ragdoll.
