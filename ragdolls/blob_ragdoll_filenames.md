# Blob Ragdoll Filenames

This document lists the filenames for the individual sprite components that make up the "blob" ragdoll in Noita. These files are used to define the visual appearance and physics behavior of the blob entity when it is in a ragdoll state.

## Ragdoll Components

The blob ragdoll is composed of several distinct sprite parts, each corresponding to a specific body segment.

### Torso

*   **Filename:** `data/ragdolls/blob/torso.png`
*   **Description:** The main body segment of the blob.

### Legs

The blob ragdoll utilizes multiple leg sprites to create a more dynamic and varied ragdoll effect.

*   **Filename:** `data/ragdolls/blob/leg1.png`
*   **Filename:** `data/ragdolls/blob/leg2.png`
*   **Filename:** `data/ragdolls/blob/leg3.png`
*   **Filename:** `data/ragdolls/blob/leg4.png`
*   **Description:** These files represent the individual leg sprites. Their arrangement and physics properties will determine how the blob moves and reacts when ragdolled.
