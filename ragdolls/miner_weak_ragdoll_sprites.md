# Miner (Weak) Ragdoll Sprites

This documentation describes the sprite filenames used for the "miner_weak" ragdoll in Noita. These files define the visual components of the ragdoll when it is broken apart.

## Sprite Filenames

The following files are used to construct the "miner_weak" ragdoll. Each file represents a distinct body part.

### Key Components

*   `head.png`: The head sprite.
*   `torso.png`: The main body sprite.
*   `left_arm.png`: The left arm sprite.
*   `right_arm.png`: The right arm sprite.
*   `left_foot.png`: The left foot sprite.
*   `right_foot.png`: The right foot sprite.
*   `left_hand.png`: The left hand sprite.
*   `right_hand.png`: The right hand sprite.
*   `thighs.png`: The thigh sprites.
*   `fuse.png`: Likely a sprite related to the fuse or explosive component of this enemy.

### File Path

All sprite files for this ragdoll are located within the `data/ragdolls/miner_weak/` directory.
