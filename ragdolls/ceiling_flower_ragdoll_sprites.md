# Ceiling Flower Ragdoll Sprites

This documentation outlines the sprite assets used for the "Ceiling Flower" ragdoll in Noita. These files define the visual components of the ragdoll when it is in a non-animated state.

## Sprite Files

The following `.png` files are used to construct the Ceiling Flower ragdoll. Each file represents a distinct visual part of the ragdoll.

| Filename        | Description        |
|-----------------|--------------------|
| `root.png`      | The central body/root of the flower. |
| `left_leaf.png` | The left leaf component. |
| `right_leaf.png`| The right leaf component. |
| `petal_left.png`| The left petal component. |
| `petal_middle.png`| The middle petal component. |
| `petal_right.png`| The right petal component. |

## Usage

These sprite files are referenced within the Noita data files to define the appearance of the Ceiling Flower ragdoll. They are typically used in conjunction with ragdoll definitions that specify how these sprites are attached and animated.

**Note:** This documentation focuses solely on the sprite assets. For information on ragdoll behavior, physics, or animation, please refer to the relevant script and entity definition files.
