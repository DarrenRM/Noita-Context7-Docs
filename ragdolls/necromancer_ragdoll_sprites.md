# Necromancer Ragdoll Sprites

This documentation outlines the sprite assets used for the Necromancer's ragdoll in Noita. These files define the visual components of the Necromancer's body when it is in a ragdoll state.

## Sprite Components

The Necromancer ragdoll is composed of several distinct sprite files, each representing a different body part or clothing item.

### Key Sprite Files

*   `head_1.png`: Primary sprite for the Necromancer's head.
*   `head_2.png`: Alternative sprite for the Necromancer's head.
*   `torso.png`: Sprite for the Necromancer's torso.
*   `cape_1.png` to `cape_4.png`: A series of sprites representing different states or layers of the Necromancer's cape.
*   `arm_right.png`: Sprite for the Necromancer's right arm.
*   `arm_left.png`: Sprite for the Necromancer's left arm.

## Usage

These `.png` files are referenced by the game's engine to render the Necromancer's ragdoll. They are typically used in conjunction with physics and animation systems to create realistic or stylized death animations.

## File Structure

The sprites are organized within the `data/ragdolls/necromancer/` directory. This structure helps in managing and identifying assets related to specific entities or game elements.
