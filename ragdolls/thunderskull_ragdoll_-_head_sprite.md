# Thunderskull Ragdoll - Head Sprite

This documentation describes the sprite file used for the head of the Thunderskull ragdoll in Noita.

## Sprite File

### `head.png`

This is the primary image file defining the visual appearance of the Thunderskull's head when it becomes a ragdoll.

**Key Attributes:**

*   **Format:** PNG image.
*   **Purpose:** Defines the visual sprite for the Thunderskull's head ragdoll.
*   **Location:** `data/ragdolls/thunderskull/head.png`

**Usage:**

This sprite is referenced by the game's ragdoll system to render the detached head of a Thunderskull enemy. The exact positioning and animation of this sprite within the ragdoll physics are determined by other configuration files not present in this source.
