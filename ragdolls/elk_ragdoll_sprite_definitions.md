# Elk Ragdoll Sprite Definitions

This documentation outlines the sprite definitions for the Elk ragdoll in Noita, as found in `data/ragdolls/elk/filenames.txt`. These files define the individual image assets used to construct the Elk's ragdoll animation.

## Key Sprite Components

The following are the primary sprite components that make up the Elk ragdoll:

### Body Parts

*   **torso.png**: The main body segment of the elk.
*   **head.png**: The head of the elk.
*   **right_thigh.png**: The upper part of the elk's right hind leg.
*   **right_leg.png**: The lower part of the elk's right hind leg.
*   **right_arm.png**: The upper part of the elk's right foreleg.
*   **right_hand.png**: The lower part of the elk's right foreleg (hoof).
*   **left_thigh.png**: The upper part of the elk's left hind leg.
*   **left_leg.png**: The lower part of the elk's left hind leg.
*   **left_arm.png**: The upper part of the elk's left foreleg.
*   **left_hand.png**: The lower part of the elk's left foreleg (hoof).

## File Structure

The `filenames.txt` file simply lists the relative paths to each sprite image file.

```
data/ragdolls/elk/torso.png
data/ragdolls/elk/right_thigh.png
data/ragdolls/elk/right_leg.png
```
